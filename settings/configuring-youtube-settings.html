Configuration des paramètres YouTube

{{ page.description }}

Pour configurer les paramètres YouTube, procédez comme suit :

  1. Ouvrez le module Gallery.
  2. Cliquez sur le lien Paramètres sur la page d'accueil.
  3. Dans le volet de navigation de gauche, cliquez sur le lien YouTube.
  4. Cliquez sur Connect.
  5. Sélectionnez le compte auquel vous souhaitez vous connecter.
  6. Connectez-vous à votre compte Google.
  7. Si vous avez accès à plusieurs comptes Google, sélectionnez le compte associé au compte YouTube auquel vous souhaitez accéder.
  8. Brightcove demandera l'autorisation de gérer votre compte YouTube. Cliquez sur Allow (Autoriser).
  9. Vérifiez que la connexion a réussi. La page Paramètres affiche la chaîne YouTube à laquelle vous êtes connecté.
