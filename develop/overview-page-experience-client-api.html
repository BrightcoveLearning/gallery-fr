Vue d'ensemble : API client d'expérience en page

{{ page.description }}

Introduction

L'API Client In-Page Experiences est une bibliothèque JavaScript qui vous aide à gérer le comportement de l'expérience en page au moment de l'exécution, de la même manière que l'API Brightcove Player est utilisée pour contrôler le comportement du lecteur. L'API est disponible sur n'importe quelle page exécutant une ou plusieurs expériences intégrées Brightcove. Il est conçu pour aider des tiers à intégrer les expériences Brightcove. Fournit un accès aux informations actuelles sur le lecteur, les vidéos et l'état, ainsi qu'aux écouteurs pour les événements liés à la lecture.

Consultez la référence complète de l'API pour plus de détails sur les méthodes et événements disponibles.

Débuter.

Remarque : pour une explication plus complète de l'utilisation de l'API, consultez Exemple d'API client d'expérience sur la page. Ici, nous allons nous concentrer sur le code d'utilisation de l'API.
<div data-experience="5bb2134180b4990011750f06"></div>
<script src="https://players.brightcove.net/1752604059001/experience_5bb2134180b4990011750f06/live.js"></script>

Le moyen le plus simple d'obtenir une référence à l'expérience est de lui donner un id (dans la div balise). Le id peut être tout ce que vous aimez tant qu'il est unique dans la page, mais puisque l'identifiant d'expérience lui-même est là dans l'URL du script (voir la partie surlignée du code ci-dessus), vous pouvez aussi bien l'utiliser :

<div data-experience="5bb2134180b4990011750f06" id="experience_5bb2134180b4990011750f06"></div>
<script src="https://players.brightcove.net/1752604059001/experience_5bb2134180b4990011750f06/live.js"></script>

Maintenant, vous êtes prêt à obtenir une référence à l'expérience en JavaScript. Rappelez-vous que l'expérience est implémentée dans un iframe, de sorte que vous ne pouvez pas facilement communiquer avec elle à partir d'un script dans la page parent. Au lieu de cela, vous devez placer votre script dans un bloc HTML personnalisé dans l'expérience elle-même.

var experience = window.top.bcov.gal.getEmbed('experience_5bb2134180b4990011750f06');

Une fois que vous avez une référence à l'expérience, vous pouvez appeler des méthodes à partir de l'API. Notez que toutes les méthodes sont sur l' clientApi objet enfant :

var experience = window.top.bcov.gal.getEmbed('experience_5bb2134180b4990011750f06'),
experienceApi,
video,
current_video = document.getElementById('current_video'),
video_paused = document.getElementById('video_paused');

experienceApi = experience.clientApi;

// get initial video
video = experienceApi.getCurrentVideo();
current_video.textContent = video.name;

// event listeners
experienceApi.on('videoChanged', function() {
  video = experienceApi.getCurrentVideo();
  current_video.textContent = video.name;
});

experienceApi.on('videoStarted', function() {
  video_paused.textContent = 'false';
});

experienceApi.on('videoPaused', function() {
  video_paused.textContent = 'true';
});
