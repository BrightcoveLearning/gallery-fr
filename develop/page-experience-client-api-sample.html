Exemple d'API client d'expérience en page

{{ page.description }}

Introduction

L'API client In-Page Experience fournit une bibliothèque JavaScript similaire à l'API Brightcove Player. L'API simplifie l'interaction et le contrôle de votre expérience en page sur la page.

Cet exemple de base vous montre comment :

  1. Obtenez une référence à l'expérience, puis à l'objet ClientAPI, qui a les méthodes.
  2. Appelez des méthodes API pour configurer des écouteurs pour divers événements et obtenir des informations sur la vidéo actuellement chargée dans le lecteur.
  3. Injectez des informations dans les éléments HTML que vous ajoutez à l'interface utilisateur de l'expérience.

Exemple d'expérience dans la page

 

Étapes à suivre pour créer l'exemple

  1. Créez une expérience dans la page avec une liste de lecture (peu importe la façon dont la liste de lecture est affichée).
  2. Publiez l'expérience.
  3. Copiez et collez le code d'intégration expérience dans une page HTML.
  4. Ajoutez un id attribut à la <div> balise, avec la valeur : customized_in_page_experience.
  5. Revenez à Studio et ajoutez un composant HTML personnalisé à l'expérience avec le code suivant : 
    <div style="padding: .5em;border: 1px #64AAB2 solid; border-radius:.5em;">
<p>Current Video: <span id="current_video" style="color:RGB(155, 37, 86);font-weight:bold;"></span></p>
<p>Video Paused: <span id="video_paused" style="color:RGB(155, 37, 86);font-weight:bold;"></span></p>
</div>
<script>
  var BCLS = ( function (window, document) {
    var experience = window.top.bcov.gal.getEmbed('customized_in_page_experience'),
    experienceApi,
    video,
    current_video = document.getElementById('current_video'),
    video_paused = document.getElementById('video_paused');

    experienceApi = experience.clientApi;

    // get initial video
    video = experienceApi.getCurrentVideo();
    current_video.textContent = video.name;

    // event listeners
    experienceApi.on('videoChanged', function() {
      video = experienceApi.getCurrentVideo();
      current_video.textContent = video.name;
    });

    experienceApi.on('videoStarted', function() {
      video_paused.textContent = 'false';
    });

    experienceApi.on('videoPaused', function() {
      video_paused.textContent = 'true';
    });

  })(window, document);
</script>
  6. Définissez le composant pour afficher Avant la lecture, Lecture et Après la lecture.
  7. Enregistrez vos modifications et republiez l'expérience.
  8. Parcourez votre page, et vous devriez voir la boîte contenant les messages « Vidéo actuelle » et « Player en pause ». Notez que la publication étant asynchrone, vous devrez peut-être attendre une minute, vider le cache du navigateur et actualiser la page pour voir les modifications.
