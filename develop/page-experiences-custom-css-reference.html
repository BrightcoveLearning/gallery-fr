Composants stylisés

Ces classes CSS s'appliquent aux éléments qui sont stylisés via l'interface utilisateur.  Ces styles peuvent être remplacés par CSS personnalisé à l'aide de ces classes.  Vous pouvez également faire appliquer les styles d'interface utilisateur au code HTML personnalisé à l'aide de ces classes.  Ces classes sont stables et ne devraient pas changer d'une version de modèle à l'autre.

 

Classe Description
ee-composants-style-global Style global pour l'expérience
ee-composants-style-ancre Style normal du lien
ee-composants-style-navigation Style des liens de navigation (bouton retour, bouton de partage, etc.)
EE-Components-Style-VideoInfo Style de boîte d'informations vidéo (non-superposition)
E-Components-Style-vidéo Titre Style de titre vidéo (non-superposition)
EE-Components-Style-Vidéo Description Description vidéo style (non-superposition)
ee-composants-style-superposition Coiffure de boîte d'informations vidéo
EE-Components-Style-OverlayTitle Superposer le style du titre vidéo
EE-Components-Style-OverlayDescription Superposition vidéo description style
EE-Components-Style-OverlayBanner Bannière superposée/style de l'indicateur (indicateur de surveillance)
Bouton de lecture EE-Components-Style Style de base du bouton de lecture (voir le composant Bouton de lecture pour un style plus complexe)

Composants de base

Ce sont des composants de base qui peuvent être coiffés en utilisant CSS personnalisé.  Les classes de composants sont stables et ne doivent pas changer entre les versions du modèle.  Les classes internes sont relativement stables et ne changeront probablement pas entre les versions du modèle.
