Création et gestion d'expériences de galerie

{{ page.description }}

Si vous souhaitez commencer rapidement à créer et à publier une expérience Gallery, consultez l'un des documents suivants :

  • Étape par étape : Création et publication d'une expérience de portail
  • Étape par étape : Création et publication d'une expérience dans la page

Lorsque vous ouvrez le module Galerie, la page Expériences répertorie toutes les expériences qui ont été créées.

Si aucune expérience n'a été créée, vous serez invité à créer une expérience. La page Expériences offre la possibilité de :

  • Supprimer des expériences
  • Prendre des expériences hors ligne
  • Expériences en double
  • Modifier les expériences
  • Créer des expériences
  • Filtrer expériences
  • Recherche d'expériences

Cliquez sur un en-tête de colonne pour trier les données. Le sélecteur de colonne () permet de sélectionner les colonnes à afficher sur la page.

Les colonnes suivantes peuvent être affichées.

  • Titre - Titre de l'expérience
  • Modèle - Modèle utilisé par l'expérience. Le type d'expérience est affiché ci-dessous :
    • Portal - Site prenant en charge une bibliothèque de contenu de n'importe quelle taille avec appel à l'action et support publicitaire en option
    • In-Page - Publiez une ou plusieurs vidéos et ajoutez de l'interactivité, puis ajoutez cette mise en page à votre site
    • Landing Page - Site d'une seule page avec formulaire de prospect et appel à l'action et support publicitaire en option
    • Événement - Site optimisé pour promouvoir et diffuser des événements en direct
  • Statut du modèle - Statut du modèle
    • À jour : l'expérience consiste à utiliser la dernière version du modèle de projet
    • Mise à jour des besoins - Un modèle de mise à jour pour l'expérience est disponible
    • Mise à jour de la reprise - La mise à jour du modèle est en cours mais n'a pas été publiée
  • Mise à jour - Date et heure de la dernière mise à jour de l'expérience
  • Statut de l'expérience - Statut de l'expérience
    • En ligne - L'expérience est en ligne et est accessible
    • Hors ligne - L'expérience est hors ligne et n'est pas accessible
    • Changements non publiés - L'expérience est en ligne et est accessible, mais des modifications ont été apportées à l'expérience et n'ont pas été publiées

Cliquez sur une ligne pour afficher les détails de l'expérience sur le côté droit de la page.

Création d'expériences

Pour créer une nouvelle expérience, cliquez sur Nouvelle expérience. Lorsque vous créez une nouvelle expérience, vous commencerez par choisir un modèle. Le modèle contrôle l'aspect général et le comportement. Après avoir sélectionné un modèle, vous pouvez personnaliser l'expérience. Pour plus d'informations, consultez l'une des rubriques suivantes :

  • Vue d'ensemble : Création, modification et publication d'expériences de portail
  • Vue d'ensemble : Création, modification et publication d'expériences dans la page

Duplication d'expériences

La duplication d'une expérience peut être utile lorsque vous avez besoin de créer une expérience dans plusieurs langues ou si vous créez des microsites à partir de l'apparence d'une expérience existante. Pour dupliquer une expérience, cliquez sur une ligne, puis cliquez sur Dupliquer.

Entrez un Titre de l'expérience et La description puis cliquez sur sauver.

Édition d'expériences

Pour modifier une expérience, cliquez sur le titre de l'expérience. La modification d'une expérience ouvrira l'éditeur de site de la Galerie. Pour plus d'informations sur les expériences de modification, reportez-vous à l'une des rubriques suivantes :

  • Utilisation de l'éditeur de site pour personnaliser une expérience de portail
  • Personnalisation d'une expérience dans la page

Prendre des expériences hors ligne

Prendre une expérience hors ligne la rend plus accessible. Pour mettre une expérience hors ligne, cliquez sur une ligne, puis cliquez sur Prendre hors ligne. Pour confirmer que l'expérience doit être mise hors ligne, cliquez sur Déconnexion. Pour rendre une expérience à nouveau disponible, l'expérience doit être republiée.

Suppression d'expériences

Pour supprimer une expérience, cliquez sur une ligne, puis cliquez sur Supprimer. Confirmez la suppression en cliquant sur Delete.

Note : Si votre expérience est configurée pour utiliser un domaine personnalisé, vous devez supprimer le domaine personnalisé affecté à l'expérience, puis republier l'expérience AVANT de supprimer l'expérience.

Filtrage des expériences

Un filtre peut être appliqué à la page Expériences pour limiter les types d'expériences affichées. Pour appliquer un filtre, sélectionnez le type d'expériences à afficher :

  • Tous - Afficher toutes les expériences
  • Publié - Afficher toutes les expériences publiées (en ligne)
  • Non publié : affiche toutes les expériences mises hors ligne
  • Mise à jour du modèle nécessaire - Afficher toutes les expériences où des mises à jour de modèles sont disponibles
  • Landing Page - Afficher toutes les expériences de page de destination
  • Événement - Afficher toutes les expériences d'événements
  • Portal - Expériences du portail Display
  • In-Page - Afficher toutes les expériences en page

Recherche d'expériences

La zone de recherche permet de rechercher dans la liste des expériences. Lors de la recherche, les propriétés suivantes seront recherchées :

  • Titre
  • Description
  • ID
  • Modèle
  • URL
