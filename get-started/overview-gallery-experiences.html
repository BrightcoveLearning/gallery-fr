Brightcove Gallery facilite la création d'expériences vidéo puissantes et attrayantes sans nécessiter de développement personnalisé. Gallery vous permet de créer les types d'expériences suivants :

  • Portail
  • Dans la page
  • Page de destination
  • Événement
Note : Si vous souhaitez commencer rapidement à créer et à publier une expérience Gallery, consultez l'une des rubriques étape par étape pour créer des expériences Portal ou In-Page.

Lorsque vous sélectionnez le type d'expérience à créer, Gallery fournit un ensemble de modèles pour Portal (qui incluent les expériences de page de destination et d'événements) et les expériences en page qui contrôlent l'aspect général de l'expérience.

Portail

Les expériences de portail sont des sites vidéo autonomes (c'est-à-dire des portails vidéo) accessibles à partir d'une URL spécifiée hébergée par Brightcove. Dans l'exemple suivant, les vidéos de formation Video Cloud et Gallery ont été organisées en une expérience de portail. La sélection d'une catégorie sur la gauche affiche les vidéos associées dans cette catégorie dans le corps de la page.

In-Page

Les expériences dans la page sont intégrées à une page Web existante. Les expériences en page vous permettent de créer une mise en page vidéo et de l'afficher à l'intérieur de votre site Web existant. Les expériences en page prennent en charge l'interactivité du joueur et de l'extérieur, telles que les liens et les cartes qui peuvent être affichés lors de la lecture de la vidéo. Dans l'exemple suivant, une expérience dans la page (les vignettes vidéo) a été incorporée dans une page Web existante. Cliquez sur l'une des vignettes vidéo pour ouvrir un lecteur vidéo dans une visionneuse.

Page de destination

Les expériences de page de destination sont des pages vidéo uniques hébergées par Brightcove et sont généralement utilisées dans le cadre d'une campagne par e-mail. Lorsqu'un utilisateur clique sur un lien dans un message électronique, il est redirigé vers une expérience Page de destination où il peut visionner une seule vidéo. L'exemple ci-dessous contient également un formulaire de prospect qui peut être utilisé pour capturer des informations de visionneuse.

Événement

Les expériences événementielles vous permettent d'engager votre public avant, pendant et après votre événement en direct en créant une destination pré-événement de marque, une page d'événement en direct et une archive vidéo après l'événement. Les expériences événementielles peuvent être créées en tant que sites de portail autonomes ou intégrées à une page existante en tant qu'expérience en page.
