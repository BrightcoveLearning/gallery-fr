    Étape par étape : Création et publication d'une expérience de portail

    Le but de cette rubrique est de vous montrer comment créer une expérience de portail attrayante à l'aide de la Galerie Brightcove.
     

    Après avoir terminé ce sujet, vous devriez être en mesure de :

    • Configuration des paramètres de Gallery
    • Utiliser la Galerie pour créer une nouvelle expérience de portail
    • Ajouter des vidéos à une expérience de portail
    • Personnaliser une expérience de portail
    • Prévisualiser et publier une expérience de portail
    Remarque : Pour obtenir un guide étape par étape sur la création d'une expérience en page, voir étape par étape : Création et publication d'une expérience dans la page.

    Audience

    N'importe quel éditeur de Video Cloud qui a besoin de créer des expériences de portail attrayantes sans avoir besoin de développement personnalisé.

    Conditions préalables

    • Un compte Video Cloud avec accès au module Galerie
    • Connaissance pratique de Video Cloud
    • Les exemples de vidéos utilisés dans cette rubrique sont disponibles au téléchargement si nécessaire
    • Utile pour visionner/assister à la formation Construire le portail à l'aide de la Galerie

    Préliminaires

    Pour utiliser les exemples de vidéos fournis, effectuez les opérations suivantes.

    1. Téléchargez les exemples de fichiers.
    2. Chargez les vidéos sur votre compte Video Cloud.
    3. Ajoutez une balise de 'bird in flight' à la vidéo Chickadee.
    4. Créez trois playlists et ajoutez les vidéos aux playlists comme suit.

    5. Ajoutez des URL de liens connexes à quelques-unes des vidéos. Dans une section ultérieure, nous allons configurer le portail pour afficher les liens connexes pour les vidéos.
    6. À l'aide du module Joueurs , créez un nouveau lecteur appelé Galerie Player.
    7. Configurez le lecteur pour ne pas afficher le titre et la description. Publiez le lecteur.

    Création d'une expérience de portail

    Pour créer une expérience de portail à l'aide de Brightcove Gallery, vous devez effectuer les tâches suivantes :

    • Configurer Gallery avec un lecteur par défaut
    • Créer un nom de domaine personnalisé
    • Créer une nouvelle expérience de portail
    • Modifier les détails du site
    • Ajouter des objets à un modèle de page
    • Aperçu de l'expérience
    • Créer des collections et ajouter des vidéos aux collections
    • Modifier le style du site
    • Activer la recherche de site
    • Personnaliser les paramètres sociaux
    • Personnalisation des paramètres vidéo et de lecture
    • Attribuer un domaine personnalisé
    • Publier l'expérience

    Configuration de Gallery avec un lecteur par défaut

    Brightcove Gallery peut être configuré avec un lecteur vidéo par défaut qui sera utilisé sur vos sites Gallery. Si aucun joueur par défaut n'est spécifié, le lecteur par défaut de Brightcove sera celui par défaut. Pour configurer Gallery avec un lecteur par défaut, procédez comme suit.

    1. Connectez-vous à Video Cloud Studio.
    2. Le nom du compte Video Cloud actuel s'affiche dans le coin supérieur droit de la page. Si vous avez plusieurs comptes, cliquez sur la liste déroulante du sélecteur de compte et sélectionnez le compte dans lequel vous souhaitez créer l'expérience. Notez que ce compte doit également être le compte qui contient les vidéos que vous souhaitez utiliser également.
    3. Cliquez sur Galerie dans le menu de navigation.
    4. Cliquez sur le lien Paramètres en haut de la page.
    5. Sélectionnez le lecteur qui sera utilisé sur vos sites Gallery. Si vous suivez les étapes, sélectionnez le lecteur Galerie que vous avez créé précédemment.
      Remarque : En tant que meilleure pratique, nous vous recommandons de créer et de styliser des joueurs dédiés pour vos expériences Galerie.
    6. Cliquez sur Save.

    Création d'un nom de domaine personnalisé

    Lorsque vous créez une nouvelle expérience de portail, Gallery lui attribue une URL aléatoire (c.-à-d. site-12345.bcvp0rtal.com). Portal Experiences peut se voir attribuer un domaine fourni par Brightcove (ce que nous ferons dans cette rubrique) ou vous pouvez utiliser votre propre domaine personnalisé. Pour créer un nouveau domaine fourni par Brightcove, procédez comme suit.

    1. Dans la page Paramètres , cliquez sur Domaines personnalisés et SSL dans la navigation de gauche.
    2. Cliquez sur Create Custom Domain.
    3. Sélectionnez un domaine brightcovegallery.com ou gallery.video .
    4. Entrez un domaine personnalisé. Utilisez votre propre nom ou nom d'entreprise. Si le domaine n'est pas disponible, un message s'affiche. Entrez un autre nom.
    5. Lorsque vous avez entré un domaine valide, cliquez sur Enregistrer.
    Remarque : Le domaine personnalisé sera attribué au portail dans une section ultérieure.

    Création d'une nouvelle expérience de portail

    Pour créer une nouvelle expérience de portail Galerie, procédez comme suit.

    1. Cliquez sur le lien Expériences en haut de la page.
    2. Cliquez sur Nouvelle expérience.
    3. Cliquez sur l'onglet Portail en haut de la page. Une liste de modèles de portail s'affiche. Recherchez le modèle de sélection de sélection et cliquez sur Choisir.
      Remarque : Une fois que vous avez sélectionné un modèle, il ne peut pas être modifié. Pour plus d'informations sur les modèles de Galerie, reportez-vous à la rubrique Présentation des modèles de portail Gallery .
    4. Entrez Animal World pour le titre de l'expérience. Pour l' expérience Pre-peupler ? , sélectionnez Non merci.
    5. Cliquez sur Créer une expérience.

    L'Éditeur de site Gallery s'ouvrira. L'éditeur de site Gallery permet de personnaliser l'expérience et permet de gérer les mises en page, d'ajouter des vidéos, de choisir des styles et de configurer d'autres paramètres pour le portail. Tous les paramètres de personnalisation sont accessibles à l'aide du menu de navigation de gauche.

    Modification des détails du site

    Les paramètres de titre, de description et de langue de l'expérience peuvent être modifiés à l'aide de l'éditeur de site Gallery.

    1. Cliquez sur APERÇU dans la navigation de gauche.
    2. Pour la description de l'expérience, ajoutez Animal World Video Portal.
    3. Cliquez sur Save.

    Ajout d'objets à un modèle de page

    Chaque modèle de portail fournit un ensemble de modèles de page qui peuvent être personnalisés. Les modèles de page fournis dépendent du modèle sélectionné lors de la création de l'expérience. Le modèle de sélection fournit des modèles de page Accueil, Lecture et Catégorie. Ces pages peuvent être affichées dans la marge gauche dans le menu PAGES .

    Pour plus d'informations sur l'ajout d'objets aux modèles de page, voir Personnalisation du contenu des pages Expérience de portail.

    Pour ajouter des objets à un modèle de page, procédez ainsi.

    1. Cliquez sur PAGES dans la navigation de gauche.
    2. Cliquez sur la page Home dans la marge de gauche. Les icônes plus () indiquent les zones de la page où des objets peuvent être ajoutés. Les objets texte statiques sur la page peuvent être modifiés en cliquant dessus.
    3. En haut de la page, cliquez sur + Ajouter une image ou un texte.
    4. Cliquez sur l'onglet Image .
    5. Cliquez sur Parcourir , puis parcourez et sélectionnez Animal World Logo.png. Ce fichier est fourni dans le cadre des exemples de ressources. Cliquez sur Save.
    6. Vérifiez que l'image s'affiche dans l'en-tête de page.
    7. Faites défiler la page jusqu'au bas de la page. Cliquez sur [VOTRE MARQUE]. Saisissez Animal World comme titre et cliquez sur Save.
    8. Cliquez sur les pages Lecture et Catégorie dans la marge gauche. Notez que l'image d'en-tête et le texte du titre apparaissent également sur ces pages. Revenez à la page Home.

    Prévisualisation de l'expérience

    L'aperçu d'une expérience vous permet de voir comment le portail va s'afficher sur les appareils mobiles, tablettes et ordinateurs de bureau. Vous voudrez prévisualiser le portail au fur et à mesure que vous passez par les étapes pour le personnaliser. Le portail peut être prévisualisé dans un nouvel onglet de navigateur ou un lien partageable peut être généré pour que d'autres utilisateurs puissent prévisualiser le portail. Pour plus d'informations sur l'aperçu et la publication d'une expérience de portail, consultez Aperçu et publication d'une expérience de portail.

    Pour afficher un aperçu du portail, procédez comme suit.

    1. Lors de la modification du site à l'aide de l'éditeur de site Gallery, cliquez sur Aperçu , puis sur Aperçu réactif.
    2. L'aperçu s'ouvre dans un nouvel onglet du navigateur.
    3. Fermez l'onglet Aperçu du site.

    Création de collections et ajout de vidéos aux collections

    À l'aide de l'Éditeur de site Gallery, vous pouvez sélectionner les vidéos à afficher sur le portail. Lors de la création d'une expérience de portail, les vidéos sont organisées en collections. Les collections sont créées à partir des métadonnées vidéo Video Cloud et peuvent inclure :

    • Sélections Video Cloud
    • Mots-clés
    • Métadonnées personnalisées
    • Balises

    Les vidéos d'un compte YouTube peuvent également être ajoutées aux collections. Pour plus d'informations sur la création de collections et l'ajout de vidéos à votre site, voir Ajout de vidéos à une expérience de portail.

    Pour ajouter des vidéos à une expérience de portail, procédez comme suit.

    1. Cliquez sur VIDÉOS dans la navigation de gauche.
    2. Nommez la première collection Oiseaux , puis appuyez sur Entrée.
    3. Cliquez sur le nom de la collection pour ouvrir l'éditeur de collection.
    4. Sélectionnez Video Cloud pour la source de la collection et sélectionnez Liste de lecture pour les critères.
    5. Cliquez sur la liste de lecture Birds . Toutes les vidéos de la playlist Birds apparaîtront dans la liste des vidéos. Cliquez sur Save.
      Remarque : Lors de l'utilisation d'une attribution de vidéo de playlist, l'ordre des vidéos est basé sur les paramètres de la liste de lecture définis dans le module Media.
    6. Créez une autre collection nommée Animaux en utilisant la playlist Animaux comme critère.
    7. Créez une autre collection nommée Créatures marines en utilisant la playlist Créatures marines comme critère.
    8. Créez une autre collection nommée Oiseaux en vol. Sélectionnez Personnaliser comme critère.
    9. Pour que les vidéos soient inclues, sélectionnez Tags et ajoutez une balise 'bird in flight'. Appuyez sur Entrée.
    10. Commandez des vidéos par les plus récentes.
    11. Confirmez que la vidéo Chickadee s'affiche dans la liste Collection de vidéos. Cliquez sur Save.
    12. Confirmez que 4 collections ont été créées.
      Remarque : Vous pouvez modifier l'ordre des collections en cliquant sur la collection et en faisant glisser la collection vers une nouvelle position.

    Modification du style du site

    Les paramètres Style peuvent être utilisés pour personnaliser les couleurs du site et les polices utilisées sur l'expérience. Pour plus d'informations sur la modification du style de site, voir Personnalisation du style d'une expérience de portail.

    Pour modifier le style de site, procédez comme suit.

    1. Cliquez sur STYLE dans la navigation de gauche.
    2. Un thème est un ensemble de couleurs et de polices qui peuvent être appliquées à un modèle. Sélectionnez un thème. Ce portail utilisera le thème Triton .
    3. Sélectionnez une police principale et secondaire à utiliser. Ce portail utilisera la police Open Sans .
    4. Cliquez sur le lien < Retour à l'expérience en haut du menu Style.

    D'autres paramètres associés au style sont également disponibles dans le cadre des paramètres APPARENCE ET COMPORTEMENT .

    1. Cliquez sur Paramètres du > modèle APPARENCE ET COMPORTEMENT dans la navigation de gauche.
    2. Sélectionnez Modifier la vidéo en vedette par la première vidéo de la catégorie sélectionnée.
      Remarque : Les paramètres de modèle disponibles sont basés sur le modèle sélectionné lors de la création de l'expérience. Tous les modèles n'ont pas de paramètres de modèle.
    3. Cliquez sur Save.

    Pour plus d'informations sur les paramètres d'apparence et de comportement, voir Personnalisation de l'apparence et du comportement d'une expérience de portail.

    Activer la recherche de site

    Les paramètres de recherche de site peuvent être utilisés pour activer et personnaliser les fonctionnalités de recherche de site. En activant la recherche de site, les fonctionnalités de recherche seront ajoutées aux pages de sites, permettant ainsi aux internautes de rechercher votre contenu. Pour plus d'informations sur la personnalisation de la recherche de site, voir Personnalisation des fonctionnalités de site d'une expérience de portail.

    Pour activer la recherche sur le site, procédez comme suit.

    1. Cliquez sur Site Features Recherche de > site dans la navigation de gauche.
    2. Cochez l'option Activer la recherche de site .
    3. Cliquez sur Enregistrer pour enregistrer les paramètres.

    Personnalisation des paramètres sociaux

    Les paramètres sociaux offrent aux utilisateurs un moyen de partager facilement votre contenu sur des sites de médias sociaux populaires. Vous pouvez également permettre aux internautes d'accéder rapidement aux sites de médias sociaux. Pour plus d'informations sur la personnalisation des paramètres sociaux, voir Personnalisation des paramètres sociaux pour une expérience de portail.

    Pour configurer les paramètres sociaux, procédez comme suit.

    1. Cliquez sur SITE Features > Social dans la navigation de gauche.
    2. Sélectionnez toutes les Sharing Options. Cela ajoutera des icônes de partage social à côté du lecteur.
    3. Sélectionnez l'option Icône sombre sur la lumière .
    4. Cliquez sur Save.
    5. Cliquez sur l'onglet Rester connecté en haut de la page. En fournissant vos informations de compte de médias sociaux, le site Gallery affichera les icônes dans le pied des pages, offrant ainsi aux internautes l'accès direct à vos sites de médias sociaux.
    6. Entrez vos informations sur les réseaux sociaux ou entrez simplement brightcove pour le nom Facebook , la poignée Twitter et le nom d'utilisateur LinkedIn.
    7. Sélectionnez l'option Icône Lumière sur l'obscurité .
    8. Cliquez sur Enregistrer pour enregistrer les paramètres.

    Personnalisation des paramètres vidéo et de lecture

    Les paramètres Vidéo et Lecture peuvent être utilisés pour sélectionner un lecteur personnalisé pour l'expérience, ajouter un formulaire de prospect et personnaliser d'autres paramètres liés à la vidéo. Pour plus d'informations sur la personnalisation des paramètres sociaux, voir Personnalisation des paramètres vidéo et de lecture d'une expérience de portail.

    Pour personnaliser les paramètres Vidéo et Lecture, procédez comme suit :

    1. Dans l'éditeur de site Gallery, cliquez sur > Lecteur VIDÉO ET LECTURE et Formulaire de prospect.
    2. Assurez-vous que les deux paramètres de comportement du joueur sont cochés. Cliquez sur Enregistrer si nécessaire.
    3. Cliquez sur VIDÉO ET LECTURE > VIDÉO dans la navigation de gauche.
    4. Sélectionnez le lien apparenté pour chaque option vidéo .
    5. Cliquez sur Save.

    Affectation d'un domaine personnalisé

    Lorsque Portal Experiences sont créées, une URL statique et aléatoire leur est assignée. Dans la plupart des cas, vous voudrez accéder au site à une URL plus conviviale. Pour plus d'informations sur l'affectation d'un domaine personnalisé à une expérience de portail, voir Affectation d'un domaine personnalisé à une expérience de portail.

    Pour afficher l'URL affectée à un portail et affecter une URL personnalisée, procédez comme suit :

    1. Dans l'Éditeur de site Gallery, cliquez sur > URL DE CONFIGURATION DU SITE. L'URL assignée au site sera affichée.
    2. Cliquez sur Add Custom Domain.
      Remarque : S'il n'y a pas de bouton Ajouter un domaine personnalisé , aucun domaine personnalisé n'a été créé.
    3. Sélectionnez le domaine que vous avez créé précédemment.
    4. Cliquez sur Save.

    Publication de l'expérience

    La publication d'une expérience de portail rend l'expérience disponible sur une URL publique. Pour plus d'informations sur l'aperçu et la publication d'une expérience de portail, consultez Aperçu et publication d'une expérience de portail.

    Pour publier l'expérience, procédez comme suit.

    1. Dans l'Éditeur de site Gallery, cliquez sur Publier.
      Remarque : Le domaine personnalisé que vous avez attribué au site doit être répertorié dans la boîte de dialogue.
    2. Dans la boîte de dialogue Publier le site, cliquez sur Publier maintenant pour confirmer la composition du site.
