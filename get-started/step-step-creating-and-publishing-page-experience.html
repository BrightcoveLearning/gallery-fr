Contenu de la page

    Étape par étape : Création et publication d'une expérience dans la page

    Le but de ce démarrage rapide est de vous montrer comment créer une expérience en page à l'aide de la Galerie Brightcove.

    Une fois cette brève introduction terminée, vous saurez :

    • Utiliser Gallery pour créer une nouvelle expérience
    • Ajouter des vidéos à une expérience en page
    • Personnaliser et styliser une expérience dans la page
    • Prévisualiser et publier une expérience dans la page
    Remarque : Pour obtenir un guide étape par étape sur la création d'une expérience de portail, voir étape par étape : Création et publication d'une expérience de portail.

    Audience

    N'importe quel éditeur de Video Cloud souhaitant créer des expériences en page pouvant être intégrées dans des pages Web.

    Conditions préalables

    • Un compte Video Cloud avec accès au module Galerie
    • Connaissance pratique de Video Cloud
    • Les exemples de vidéos utilisés dans cette brève introduction sont téléchargeables, le cas échéant.
    • Utile pour afficher la formation Création d'expériences dans la page à l'aide de la Galerie

    Utilisation des ressources de démarrage rapide

    Si vous voulez suivre en utilisant les exemples de vidéos fournis avec cette brève introduction, procédez comme suit.

    1. Téléchargez l'exemple de fichier.
    2. Utilisez le module Upload pour télécharger les vidéos sur votre compte Video Cloud.
    3. Utilisez le module Media pour créer une playlist Animaux et ajouter les vidéos à la liste de lecture.
    4. Utilisez le module Joueurs pour créer un nouveau lecteur Brightcove appelé Gallery Player.
    Remarque : L'image du fichier exemple sera utilisée plus tard dans le démarrage rapide.

    Utilisation de Gallery pour créer une expérience dans la page

    Pour créer une expérience en page à l'aide de la Galerie Brightcove, vous devez effectuer les tâches suivantes :

    • Sélectionner un compte
    • Créer une nouvelle expérience dans la page
    • Personnaliser l'expérience
    • Ajouter des vidéos à l'expérience
    • Ajouter de l'interactivité à une vidéo
    • Personnaliser le style
    • Configurer les paramètres d'expérience
    • Examinez les détails de l'expérience
    • Prévisualiser et publier l'expérience en page

    Sélection d'un compte

    Selon votre abonnement Video Cloud, vous pouvez avoir accès à plusieurs comptes Video Cloud. Avant de commencer, assurez-vous d'utiliser le bon compte.

    1. Connectez-vous à Video Cloud Studio.
    2. Le nom du compte Video Cloud actuel s'affiche dans le coin supérieur droit de la page. Si vous avez plusieurs comptes, cliquez sur le menu déroulant du sélecteur de compte et sélectionnez le compte dans lequel vous souhaitez créer l'expérience en page. Notez que ce compte doit également être le compte qui contient les vidéos que vous souhaitez utiliser également.
    3. Cliquez sur Galerie dans l'en-tête de navigation.

    Création d'une nouvelle expérience dans la page

    Pour créer une nouvelle expérience en page, procédez comme suit.

    1. Cliquez sur le lien Expériences en haut de la page.
    2. Si vous n'avez créé aucune expérience, une page Aucun projet s'affiche. Cliquez sur Créer en regard de In-page.

      Si vous avez des expériences existantes, cliquez sur Nouvelle expérience , puis sur le lien In-page en haut de la page.

    3. Une liste de modèles d'expérience dans la page s'affiche. Recherchez le modèle de liste de lecture verticale et cliquez sur Choisir.
      Remarque : Une fois que vous avez sélectionné un modèle, il ne peut pas être modifié. Pour plus d'informations sur les modèles d'expérience en page, consultez la rubrique Présentation des modèles d'expérience dans la page Galerie .
    4. Entrez un Titre de l'expérience et Description de l'expérience. Pour la taille de mise en page, sélectionnez Responsive.
      Remarque : Ces valeurs peuvent être modifiées ultérieurement. Ceci est abordé plus loin dans ce Quick Start.
    5. Cliquez sur Créer une expérience.

    L'éditeur de mise en page s'ouvre. Le titre de l'expérience apparaîtra en haut de la page. Le titre peut être modifié dans le cadre des détails de l'expérience.

    Personnalisation de l'expérience

    Des objets peuvent être ajoutés à l'expérience pour personnaliser l'apparence. Les expériences en page ont des états différents et chaque état peut être personnalisé. Les états suivants sont disponibles pour personnaliser :

    • AVANT LA LECTURE - Comment l'expérience apparaîtra lors du premier rendu, avant le début de la lecture vidéo
    • LECTURE - Comment l'expérience apparaîtra lors de la lecture d'une vidéo
    • AFTER PLAY - Comment l'expérience apparaîtra après la lecture d'une vidéo

    Pour modifier les différents états, cliquez sur un état, puis ajoutez des composants à l'expérience selon vos besoins. Pour plus d'informations, voir Personnalisation d'une expérience dans la page.

    Pour ajouter des objets à une expérience en page, procédez comme suit.

    1. Cliquez sur LAYOUT dans la navigation de gauche.
    2. Cliquez sur l'état AVANT LA LECTURE . Les icônes Plus () indiquent où les composants peuvent être ajoutés à l'expérience. Cliquez sur l'icône plus en haut de l'expérience. Le sélecteur de composants s'ouvre.
    3. Cliquez sur le composant Image . Cliquez sur le lien Parcourir et sélectionnez une image à utiliser comme en-tête pour l'expérience. Notez que les fichiers exemples de démarrage rapide incluent une image qui peut être utilisée.
    4. Cliquez sur Save. L'image apparaîtra sur l'expérience.
    5. Cliquez sur l'icône plus au-dessus du composant image et ajoutez un composant Texte à l'expérience. Entrez du texte, puis cliquez sur Enregistrer.
    6. Vérifiez que l'image et le texte apparaissent sur l'expérience.

    Pour modifier l'une des propriétés du composant ou pour supprimer un composant, passez la souris sur le composant et cliquez sur les icônes de modification ou de suppression.

    Ajout de vidéos à l'expérience

    Selon le modèle sélectionné, l'expérience peut afficher une ou plusieurs vidéos à la fois. Les vidéos peuvent être ajoutées manuellement à une expérience, une playlist peut être sélectionnée ou un ID de référence peut être fourni. Pour plus d'informations sur l'ajout de vidéos à une expérience en page, reportez-vous à la section Ajout de vidéos à une expérience dans la page.

    Pour ajouter des vidéos à une expérience en page, procédez comme suit.

    1. Cliquez sur l'icône VIDEOS dans la navigation de gauche.
    2. Cliquez sur Ajouter des vidéos.
    3. Sélectionnez une méthode de sélection des vidéos pour sélectionner manuellement les vidéos à ajouter, OU sélectionnez une méthode de sélection de liste de lecture et parcourez et sélectionnez l'option Playlist Animaux que vous avez créé avec les exemples de vidéos.
    4. Cliquez sur Select (Sélectionner). La liste des vidéos s'affichera.

    Ajout d'interactivité à une vidéo

    Les interactions entre les liens et les cartes peuvent être ajoutées aux vidéos. Les interactions s'affichent au fur et à mesure de la lecture de la vidéo. Pour plus d'informations sur l'ajout d'interactivité à une expérience, voir Ajout d'interactions à une expérience en page.

    Pour configurer l'interactivité d'une vidéo, procédez comme suit.

    1. Cliquez sur l'icône VIDEOS dans la navigation de gauche.
    2. Sélectionnez une vidéo à laquelle ajouter de l'interactivité et cliquez sur l'icône de lien () dans la colonne Actions .
    3. Ajouter un texte de lien et une URL de lien. Le lecteur de prévisualisation affichera un aperçu du lien.
    4. Cliquez sur le lien Configurer .
    5. Cliquez sur le lien Basculer à% . Définissez l' heure de début sur 50 % dans la vidéo.
    6. Pour la durée, cochez Jusqu'à la fin de la vidéo.
      Remarque : Le curseur sous le lecteur de prévisualisation peut également être utilisé pour définir l'heure de début et la durée du lien.
    7. Cliquez sur le lien Création . Définissez le lien pour qu'il apparaisse en bas à droite du lecteur.
    8. Cliquez sur Save. Une liste de toutes les interactions configurées pour la vidéo s'affiche.
    9. Cliquez sur le X dans le titre de la boîte de dialogue pour fermer l'éditeur d'interactivité.

    Les vidéos avec interactivité afficheront une icône dans la colonne Interactivité indiquant le ou les types d'interactions qui ont été configurés.

    Personnalisation du style

    Le style contrôle les polices et les couleurs utilisées sur l'expérience. Pour plus d'informations sur la personnalisation du style, voir Personnalisation du style d'une expérience dans la page.

    Pour configurer le style, procédez comme suit.

    1. Cliquez sur STYLE dans la navigation de gauche.
    2. Recherchez le style GLOBAL . Cliquez sur le cercle Couleur du texte et modifiez la couleur du texte en Rouge. Le composant de texte qui a été ajouté à l'expérience précédemment doit passer au rouge.
    3. Cliquez sur le X pour fermer le sélecteur de couleurs.
    4. Recherchez le style VIDEO TITLE . Cliquez sur le cercle Couleur du texte et modifiez la couleur du texte en bleu. Cliquez sur le X pour fermer le sélecteur de couleurs.
    5. Faites défiler la navigation vers la gauche vers le haut et cliquez sur le lien Retour à la mise en page .

    Configuration des paramètres d'expérience

    Les paramètres d'expérience contrôlent divers aspects de l'expérience tels que le joueur à utiliser, les paramètres sociaux et d'autres paramètres. Pour plus d'informations sur la personnalisation des paramètres, voir Configuration des paramètres pour une expérience en page.

    Pour configurer les paramètres d'expérience, procédez comme suit.

    1. Cliquez sur SETTINGS > Player dans la navigation de gauche.
    2. Utilisez la liste déroulante Lecteurs disponibles pour sélectionner le Lecteur Galerie que vous avez créé précédemment. Il s'agit du joueur qui sera utilisé sur l'expérience en page.
    3. Cochez l'option Avance à la vidéo suivante automatiquement .
    4. Cliquez sur Save.
    5. Cliquez sur SETTINGS > Social dans la navigation de gauche.
    6. Sélectionnez la façon dont les options de partage social doivent s'afficher.
    7. Sélectionnez les sites de médias sociaux sur lesquels les vidéos peuvent être partagées. Certains sites autorisent également la publication de messages personnalisés.
    8. Cliquez sur Save.

    Examiner les détails de l'expérience

    Les détails de l'expérience sont l'endroit où le titre de l'expérience et le dimensionnement sont configurés. Pour plus d'informations sur la personnalisation des détails de l'expérience, voir Configuration des détails pour une expérience dans la page.

    Pour configurer les détails de l'expérience, procédez comme suit.

    1. Cliquez sur APERÇU dans la navigation de gauche.
    2. Si nécessaire, modifiez le titre et la description de l'expérience.
    3. Confirmez que la taille de mise en page est définie sur Responsive. Lorsque la taille de mise en page est définie sur responsive, elle se redimensionnera pour l'adapter à son conteneur sur la page.
    4. Si des modifications ont été apportées, cliquez sur Enregistrer.

    Prévisualisation et publication de l'expérience dans la page

    L'aperçu d'une expérience en page vous permet de voir comment l'expérience s'affiche sans avoir à la visualiser dans l'interface Galerie. La publication de l'expérience dans la page génère le code d'intégration nécessaire pour afficher l'expérience à l'intérieur d'une page Web. Pour plus d'informations sur la prévisualisation et la publication d'une expérience en page, consultez Aperçu et publication d'une expérience dans la page.

    1. Pour prévisualiser une expérience en page, modifiez l'expérience, puis cliquez sur Aperçu.
    2. Un aperçu du bureau s'ouvrira dans un nouvel onglet du navigateur.
    3. Les listes déroulantes dans le coin supérieur gauche peuvent être utilisées pour prévisualiser l'expérience sur différents types de périphériques et également pendant différents états d'expérience. Un aperçu mobile est affiché ci-dessous.
    4. Pour publier une expérience dans la page, cliquez sur Publier et intégrer.
    5. Cliquez sur Publier ou Publier les modifications.
    6. Le processus de publication débutera et une fois terminé, l' état de publication devient Cette expérience est à jour.  Notez que la mise à jour de l'expérience peut prendre quelques minutes.

    Une fois l'expérience en page publiée, l' URL partageable peut être partagée avec d'autres personnes afin qu'elles puissent prévisualiser l'expérience. Le code d'intégration peut être utilisé pour incorporer l'expérience en page dans une page Web existante.

    Remarque : Toute modification apportée à une expérience en page nécessitera que l'expérience soit republiée. AUCUNE modification n' est requise à un code intégré existant, SAUF si vous changez la taille de mise en page d'une taille fixe à responsive ou vice versa.
    Remarque : Si la taille d'expérience est définie sur Responsive, le code d'intégration recommandé doit être utilisé. Le code d'intégration No JavaScript n'est pas réactif.
