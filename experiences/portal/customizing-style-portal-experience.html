Personnaliser le style d'une expérience de portail

{{ page.description }}

À l'aide de l'éditeur de site Galerie, vous pouvez personnaliser le style d'une expérience de portail en modifiant le thème, les couleurs et les polices. Pour modifier le style, modifiez l'expérience, puis cliquez sur STYLE dans la navigation de gauche.

Sélection d'un thème

La sélection d'un thème applique un ensemble prédéfini de polices et de couleurs au site. Pour modifier le thèmedu site :

  1. Cliquez sur STYLE dans la navigation de gauche.
  2. Sélectionnez un thème dans la liste déroulante. Au fur et à mesure que vous sélectionnez un thème, le site s'actualise afin que vous puissiez prévisualiser le thème.
  3. Pour revenir à l'Éditeur de site, cliquez sur < Retour à l'expérience.

Personnalisation des couleurs

Les paramètres Couleurs peuvent être utilisés pour affiner les couleurs fournies par un thème. Par exemple, les paramètres de couleurs permettent de personnaliser les couleurs d'arrière-plan et de police. Pour personnaliser les couleurs:

  1. Cliquez sur l'icône STYLE dans la navigation de gauche. La section Couleurs affiche tous les objets de page et leurs couleurs.
  2. Pour sélectionner une nouvelle couleur, cliquez sur le cercle de couleur.
  3. En utilisant le sélecteur de couleur, sélectionnez une couleur en cliquant sur celle-ci dans le carré de couleur ou en saisissant un numéro de couleur hexadécimal au-dessous du carré de couleur. Notez que vous pouvez modifier la plage de couleurs en cliquant dans le spectre de couleurs sous le carré de couleur. Cliquez sur X pour fermer le sélecteur de couleurs et enregistrer la modification.

    Pour réinitialiser une sélection de couleurs, cliquez sur la flèche gauche à gauche du cercle de couleurs.

  4. Pour revenir à l'Éditeur de site, cliquez sur < Retour à l'expérience.

Sélection des polices

Les polices utilisées sur le site peuvent également être modifiées. Pour sélectionner différentes polices :

  1. Cliquez sur STYLE dans la navigation de gauche.
  2. Sélectionnez les nouvelles polices à l'aide des listes déroulantes.

    Pour réinitialiser une sélection de police, cliquez sur le lien de réinitialisation au-dessus de la liste déroulante de police.

    Note : Le nombre de polices disponibles dépend du thème du site sélectionné.
  3. Pour revenir à l'Éditeur de site, cliquez sur < Retour à l'expérience.
Note : Pour plus d'informations sur l'utilisation de polices personnalisées dans une expérience Portal, reportez-vous à la section Utilisation de polices personnalisées dans une expérience Portal.
