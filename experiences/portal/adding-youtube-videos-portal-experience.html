Ajout de vidéos YouTube à une expérience de portail

{{ page.description }}

À l'aide de l'Éditeur de collection Gallery, vous pouvez sélectionner les vidéos à rendre disponibles sur une expérience de portail. Avec Portal Experiences, les vidéos sont organisées en collections. Les collections peuvent contenir des vidéos de votre compte Video Cloud ou d'une chaîne YouTube. Cette rubrique explique comment ajouter des vidéos YouTube à une collection. Pour plus d'informations sur l'ajout de vidéos Video Cloud à une collection, voir Ajout de vidéos Video Cloud à une expérience de portail.

Exigences relatives à l'intégration de YouTube

  • Avant d'ajouter des vidéos YouTube à un site vidéo, vous devez configurer les paramètres YouTube.
  • Vous ne pouvez ajouter dans une collection que les vidéos des sélections YouTube. Les vidéos ne faisant pas partie d'une sélection YouTube ne seront pas accessibles via le gestionnaire de collections Gallery. Par ailleurs, le paramètre Playlist Privacy de la sélection en question doit être défini sur Public.
  • Les vidéos privées ne doivent pas figurer dans des sélections publiques.

Restrictions relatives à l'intégration de YouTube

  • Les vidéos YouTube ne peuvent pas être téléchargées
  • Les vidéos YouTube visionnées dans un lecteur Brightcove avec le plug-in IMA n'afficheront pas les publicités sur les dispositifs mobiles.
  • Lors de la recherche, toutes les collections doivent être composées de vidéos YouTube et les vidéos doivent être la propriété de la chaîne sélectionnée.
  • ' L'utilisation de YouTube par les clients est soumise aux conditions générales de YouTube

Création d'une collection

Les vidéos dans un site vidéo sont organisées en collections. Pour créer une collection, procédez ainsi :

  1. Cliquez sur VIDEOS dans la navigation de gauche. La page Gérer les collections s'ouvre. Le côté droit de la page affiche un aperçu de la mise en page vidéo de la page d'accueil pour le type de modèle sélectionné.
  2. Saisissez le nom de la collection et appuyez sur Entrée. Le nom sera affiché aux spectateurs, alors assurez-vous d'utiliser un nom significatif. Cliquez sur modifier () pour modifier le nom de la collection.

Modification d'une collection

  1. Pour modifier une collection, cliquez sur le nom de la collection.
  2. La page Modifier la collection s'ouvre. La page Modifier la collection permet d'ajouter des vidéos à une collection et de configurer des propriétés de collection supplémentaires. Le menu déroulant Change Collection est une liste de toutes les collections créées. Sélectionnez la collection à modifier. L'aperçu sur la droite indique où la collection apparaît sur la page d'accueil.
  3. Collection Source indique la source du contenu vidéo de la collection :
    • Video Cloud : le contenu vidéo Video Cloud sera utilisé. Pour plus d'informations, voir Ajout de vidéos Video Cloud à une expérience de portail.
    • YouTube : le contenu vidéo Video Cloud sera utilisé.
  4. Sélectionnez la chaîne YouTube à utiliser.
  5. Sélectionnez une sélection. Seules les vidéos incluses dans une sélection seront affichées.
  6. Cliquez sur Save.

Ajout d'informations de collecte

Selon le modèle sélectionné, vous pouvez configurer les informations de la collection. Pour ajouter des détails sur la collection, cliquez sur le bouton INFORMATIONS SUR LA COLLECTE section. La section Informations sur la collection permet d'ajouter une image de collection et une description de collection. Certains réseaux sociaux utiliseront l'image et la description lorsqu'un lien vers l'expérience est affiché sur les sites de médias sociaux.

L' image de la collection est utilisée pour représenter la collection sur un site Galerie. Si aucune image de collection n'est spécifiée, Gallery en attribuera une à l'aide de la vignette vidéo de la première vidéo de la collection. Pour ajouter une image de collection, cliquez sur Browse et sélectionnez une image à partir du système de fichiers. Vous pouvez également faire glisser et déposer une image dans la zone désignée.

La description de la collection est utilisée pour décrire la collection.

Foire aux questions

1. Comment les statistiques sont-elles présentées ?

Le module Analytics regroupe le nombre de visionnages de toutes vos vidéos YouTube et présente ses informations en tant qu'élément unique.

2. Une collection peut-elle contenir à la fois des vidéos YouTube et des vidéos Video Cloud ?

Non. Chaque collection Gallery ne peut contenir que des vidéos d'une seule source. Toutefois, les sites Gallery peuvent inclure plusieurs collections, et donc à la fois du contenu YouTube et du contenu Video Cloud.

3. Les vidéos YouTube d'un site Gallery doivent-elles provenir de ma propre chaîne ou puis-je utiliser des vidéos issues des chaînes d'autres éditeurs ?

Vous pouvez utiliser aussi bien des vidéos de votre propre chaîne que des vidéos d'une autre chaîne présentée que vous avez reliée. De cette façon, les entreprises disposant de plusieurs chaînes (toutes reliées sur YouTube) peuvent inclure des vidéos de n'importe quelles chaînes sur un seul site Gallery. Vous avez également la possibilité d'inclure dans votre site Gallery des vidéos provenant de chaînes qui ne vous appartiennent pas.

4. Comment les métadonnées des vidéos sont-elles utilisées dans Gallery ?

Les métadonnées YouTube suivantes sont utilisées dans Gallery :

  • ID
  • Titre
  • Description
  • URL de vignette de haute qualité
  • Durée
  • Date de publication

Ces métadonnées sont utilisées de la même manière que les métadonnées des vidéos Video Cloud, elles sont ajoutées aux <meta> balises dans la tête, utilisées dans la carte du site, etc.

5. Si le lecteur Brightcove configuré pour le site Gallery est un lecteur compatible avec Audience, le module Audience capturera-t-il les données de visionnage relatives aux vidéos YouTube ?

Oui. Les formulaires de client potentiel sont également pris en charge.

6. Les formulaires de client potentiel Gallery sont-ils pris en charge pour les vidéos YouTube ?

Oui.

7. Existe-t-il des restrictions d'utilisation concernant les vidéos YouTube sur un site Gallery ? Par exemple, si une vidéo YouTube n'autorise pas l'intégration, peut-on tout de même l'utiliser ? Et dans le cas où une vidéo est marquée comme « non listée » ou « privée » ?

Oui.
