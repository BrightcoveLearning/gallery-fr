Ajout de vidéos Video Cloud à une expérience de portail

{{ page.description }}

À l'aide de l'éditeur de collection Gallery, vous pouvez sélectionner les vidéos à afficher sur une expérience de portail. Avec Portal Experiences, les vidéos sont organisées en collections. Les collections peuvent contenir des vidéos provenant d'un compte Video Cloud ou d'une chaîne YouTube. Cette rubrique se concentre sur l'ajout de vidéos Video Cloud à une collection. Pour plus d'informations sur l'ajout de vidéos YouTube à une collection, reportez-vous à la section Ajout de vidéos YouTube à une expérience de portail.

Création d'une collection

Les vidéos dans Portal Experiences sont organisées en collections. Pour créer une collection, procédez ainsi :

  1. Cliquez sur VIDÉOS dans la navigation de gauche. La page Gérer les collections s'ouvre. Le côté droit de la page affiche un aperçu de la mise en page vidéo de la page d'accueil pour le type de modèle sélectionné.
  2. Saisissez le nom de la collection et appuyez sur Entrée. Le nom sera affiché aux spectateurs, alors assurez-vous d'utiliser un nom significatif. Cliquez sur modifier () pour modifier le nom de la collection.

Modification d'une collection

  1. Pour modifier une collection, cliquez sur le nom de la collection.
  2. La page Modifier la collection s'ouvre. La page Modifier la collection permet d'ajouter des vidéos à une collection et de configurer des propriétés de collection supplémentaires. Le menu déroulant Change Collection est une liste de toutes les collections créées. Sélectionnez la collection à modifier. L'aperçu sur la droite indique où la collection apparaît sur la page d'accueil.
  3. Collection Source indique la source du contenu vidéo de la collection :
    • Video Cloud - Le contenu vidéo Video Cloud sera utilisé
    • YouTube : le contenu vidéo Video Cloud sera utilisé. Pour plus d'informations, voir Ajout de vidéos YouTube à une expérience de portail.
  4. Avec une source de collection Video Cloud, les vidéos peuvent être issues d'une playlist ou utiliser une affectation personnalisée :
    • Playlist - les vidéos de la collection sont basées sur une liste de lecture Video Cloud
    • Personnalisé : les vidéos de la collection sont basées sur des balises vidéo, une recherche personnalisée ou des champs de métadonnées personnalisés
Note : Les valeurs entrées pour Inclure la balise et Exclure la balise dans le cadre des paramètres de recherche de > site FONCTIONNALITÉS DU SITE s'appliquent également à toutes les collections créées. Cela peut limiter les vidéos affichées lors de la création de collections.
Note : Si vous avez récemment apporté des modifications aux métadonnées vidéo ou aux listes de lecture, ces modifications peuvent prendre jusqu'à 6 minutes avant d'apparaître dans l'éditeur de collection et sur les expériences publiées.

Utilisation d'une affectation vidéo Playlist

Lorsque vous choisissez d'utiliser une affectation de vidéo Playlist, les vidéos de la collection seront basées sur une sélection Video Cloud.

  1. Sélectionnez l'option Playlist. Une liste de playlists dans votre compte Video Cloud s'affiche.
  2. Utilisez les commandes de navigation () ou la fonction de recherche pour localiser la liste de lecture souhaitée. Après avoir sélectionné une liste de lecture, la liste des vidéos de cette playlist s'affiche.
    Note : L'ordre des vidéos de la collection est basé sur les paramètres de la liste de lecture.
  3. (Facultatif) Selon le modèle sélectionné, des détails supplémentaires de collecte peuvent être configurés. Pour plus d'informations, consultez la section Ajout d'informations de collecte de cette rubrique.
  4. Cliquez sur Save pour enregistrer les modifications apportées.

Utilisation d'une affection de vidéo Custom

Lorsque vous choisissez d'utiliser un Personnalisé affectation vidéo, les vidéos de la collection seront basées sur des balises vidéo, une recherche personnalisée ou des champs de métadonnées personnalisés.

  1. Sélectionnez l'option Custom.
  2. Les paramètres de recherche peuvent être réglés sur Comprendre ou Exclure des vidéos basées sur Mots clés , Chercher ou un Champ personnalisé.
    Note : Lorsque vous utilisez l'option Champ personnalisé, vous devez entrer le nom interne du champ personnalisé, PAS le nom de l'affichage.
  3. Configurez les paramètres de recherche appropriés pour renvoyer les vidéos souhaitées. Dans l'exemple suivant, les vidéos qui ont été marquées avec video cloud essentials seront retournées.
  4. (Facultatif) Cliquez sur l'icône d'ajout () dans la section appropriée pour inclure ou exclure des paramètres supplémentaires.
  5. Définissez l'ordre de tri vidéo :
    • Les plus récentes - Vidéos commandées en fonction du moment où elles ont été activées, les plus récentes répertoriées en premier
    • Les plus anciennes en premier - Vidéos commandées en fonction du moment où elles ont été activées, les plus anciennes répertoriées en premier
    • Les plus consultés - Vues les plus populaires
    • A-Z - Trié par ordre croissant
    • Z-A - Trié par ordre décroissant
  6. (Facultatif) Selon le modèle sélectionné, des détails supplémentaires de collecte peuvent être configurés. Pour plus d'informations, consultez la section Ajout d'informations de collecte de cette rubrique.
  7. Cliquez sur Save pour enregistrer les modifications apportées.

Ajout d'informations de collecte

Selon le modèle sélectionné, vous pouvez configurer les informations de la collection. Pour ajouter des détails sur la collection, cliquez sur le bouton INFORMATIONS SUR LA COLLECTE section. La section Informations sur la collecte permet d'ajouter une image de collection et une description de collection. Certains réseaux sociaux utiliseront l'image et la description lorsqu'un lien vers l'expérience est affiché sur les sites de médias sociaux.

L' image de la collection est utilisée pour représenter la collection sur un site Galerie. Si aucune image de collection n'est spécifiée, Gallery en attribuera une à l'aide de la vignette vidéo de la première vidéo de la collection. Pour ajouter une image de collection, cliquez sur Parcourir et sélectionnez une image dans le système de fichiers. Vous pouvez également faire glisser et déposer une image dans la zone désignée.

La description de la collection est utilisée pour décrire la collection.
