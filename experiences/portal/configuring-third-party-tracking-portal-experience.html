Configuration du suivi tiers pour une expérience de portail

{{ page.description }}

Pour configurer les paramètres de suivi tiers pour une expérience de portail, modifiez l'expérience et cliquez sur CONFIGURATION DU SITE Suivi > tiers dans la navigation de gauche. Les expériences de portail peuvent être intégrées à d'autres sites tiers ou utiliser les services fournis par d'autres sites tiers :

  • Google Analytics
  • Gestionnaire de balises Google
  • Eloqua
  • Marketo

Pour accéder aux paramètres de suivi, modifiez l'expérience et cliquez sur CONFIGURATION DU SITE Suivi > tiers dans la navigation de gauche.

Note : Les paramètres disponibles sont basés sur le modèle de site sélectionné. Si le modèle ne prend pas en charge l'option, il n'apparaîtra pas.

Intégration avec Google Analytics

Gallery permet d'envoyer des données de suivi à Google Analytics. Lorsque vous créez une propriété à suivre via Google Analytics, Google vous fournit un ID de suivi ainsi qu'un code de suivi à ajouter à votre site. L'ID de suivi permet de dire à Google Analytics sur quel compte et quelle propriété envoyer les données. Il doit être saisi dans le champ Google Analytics. Le code de suivi peut être ajouté à votre site de l'une des façons suivantes :

  • En-tête/pied de page HTML personnalisé : si vous utilisez un en-tête/pied de page HTML personnalisé sur votre site, vous pouvez coller le code de suivi dans celui-ci.
  • Spécifier un fichier JavaScript personnalisé - Enregistrez le code de suivi dans un fichier .js hébergé sur votre domaine, puis pointez-le à l'aide de la propriété JavaScript personnalisée pour le site

En ajoutant l'ID de suivi dans Paramètres du site, vous obtenez l'analyse par défaut « page vue ». Pour le même résultat, Gallery ajoutera le JavaScript minimum requis. L'ajout de code de suivi dans l'en-tête personnalisé ou dans un fichier JavaScript personnalisé permet d'ajouter plus de données statistiques spécifiques. Par exemple, vous pouvez ajouter du code personnalisé pour le clic ou d'autres événements.

Pour en savoir plus sur Google Analytics et pour créer un identifiant de suivi pour votre site, consultez le site internet de Google Analytics.

Intégration avec Google Tag Manager

Gallery inclut la possibilité d'inclure du code de Google Tag Manager. Collez le code approprié dans le < diriger> ou < corps> sections.

Pour en savoir plus sur Google Tag Manager, visitez le Google site Internet.

Intégration à Eloqua

Gallery comprend les scripts de suivi des pages Eloqua pour permettre de suivre les visites sur votre site internet. Pour intégrer votre site à Eloqua, vous devez saisir votre identifiant de site Eloqua.

Intégration à Marketo

Gallery inclut des scripts de suivi des pages Marketo pour vous permettre de suivre les visites sur vos sites vidéo. Pour intégrer votre site à Marketo, vous devez saisir votre identifiant de compte Marketo.
