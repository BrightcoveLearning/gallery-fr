Configuration des détails d'une expérience de portail

{{ page.description }}

Le nom, la description et la langue de l'expérience peuvent être configurés dans le cadre des paramètres Vue d'ensemble. Pour configurer les paramètres Vue d'ensemble, modifiez l'expérience, puis cliquez sur APERÇU dans la navigation de gauche.

La page Vue d'ensemble s'affiche :

  • Type d'expérience : affiche le type d'expérience et le modèle utilisé
  • Titre et description de l'expérience - Lorsque vous créez une nouvelle expérience pour la première fois, vous êtes invité à entrer le titre de l' expérience et la description de l' expérience. Ces valeurs peuvent être modifiées ici. Le titre de l'expérience s'affiche dans l'onglet du navigateur à l'ouverture du site.
  • Experience Language - Définit le texte spécifique à la galerie sur le site sur cette langue. La prise en charge de la langue varie en fonction du modèle utilisé.

Lorsque vous avez terminé, cliquez sur Enregistrer pour enregistrer vos modifications.

Note : Après avoir apporté des modifications à une expérience, assurez-vous de republier l'expérience afin que ces modifications apparaissent sur les sites qui affichent l'expérience.
