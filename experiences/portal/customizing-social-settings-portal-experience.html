Personnalisation des paramètres sociaux pour une expérience de portail

{{ page.description }}

À l'aide de l'Éditeur de site Gallery, vous pouvez personnaliser les paramètres sociaux d'une expérience de portail :

  • Partage d'événements - Permet de partager un événement sur des sites de médias sociaux populaires
  • Partage vidéo : permet aux spectateurs de partager une vidéo sur des sites de médias sociaux populaires
  • Restez connecté - Fournissez aux téléspectateurs un lien direct vers vos sites de médias sociaux sur Facebook, Twitter et LinkedIn

Pour personnaliser les paramètres sociaux, cliquez sur Social dans la navigation de gauche.

Note : L'option Partage d'événements n'apparaît que lorsque le modèle Live Event Portal est utilisé.

Configuration des paramètres de partage d'événements

Les paramètres de partage d'événements permettent aux spectateurs de partager un événement sur des sites de médias sociaux populaires. Un menu de partage apparaîtra sur l'expérience.

Note : L'onglet Partage d'événements n'apparaît que lorsque le modèle Live Event Portal est utilisé.

Titre et description de l'événement : entrez les informations de l'événement. Ces informations seront affichées lorsqu'elles sont partagées sur les sites de médias sociaux.

Options de partage : sélectionnez les options de partage social affichées. À l'exception de Facebook, un message de partage personnalisé peut être configuré pour chaque option de partage social en cliquant sur le nom. Par exemple, pour configurer un message personnalisé pour les vidéos partagées sur LinkedIn, cliquez sur le bouton LinkedIn relier. Configurez le message, puis cliquez sur Enregistrer.

Des messages personnalisés apparaîtront dans les publications lorsque la vidéo est partagée socialement.

Configuration des paramètres de partage de vidéos

Les paramètres de partage vidéo permettent aux spectateurs de partager une vidéo sur des sites de réseaux sociaux populaires. Les icônes de partage apparaîtront à côté du lecteur.

Note : Si un lien vers une vidéo est partagé et que la vidéo est ensuite supprimée, le spectateur sera redirigé vers la page d'accueil du site lorsque vous cliquez sur le lien.

Pour configurer les options de partage vidéo qui seront affichées, cliquez sur le bouton Partage de vidéo languette.

Options de partage : sélectionnez les options de partage social à activer. À l'exception de Facebook, un message de partage personnalisé peut être configuré pour chaque option de partage social en cliquant sur le nom. Par exemple, pour configurer un message personnalisé pour les vidéos partagées sur LinkedIn, cliquez sur le bouton LinkedIn relier. Configurez le message, puis cliquez sur Enregistrer.

Des messages personnalisés apparaîtront dans les publications lorsque la vidéo est partagée socialement.

Couleur et arrière-plan de l'icône - Définissez la couleur de l'icône pour utiliser une icône Lumière sur fond sombre ou une icône Sombre sur fond clair. Tous les modèles ne prennent pas en charge cette option.

Note : Si vous activez les options de partage vidéo dans Gallery, nous vous recommandons de désactiver les paramètres sociaux dans les propriétés du lecteur.

Partage sur Twitter

Lorsque les internautes partagent le contenu vidéo sur Twitter, la vidéo est intégrée dans une carte de lecteur Twitter. Les lecteurs de carte permettent d'intégrer du contenu vidéo dans des tweets. Avant que le contenu vidéo ne soit lu incorporé dans un tweet, l'éditeur doit être approuvé par Twitter. Lorsque vous créez un site Gallery, une URL du formulaire lui est assignée site-xxx.bcvp0rtal.com. Le bcvp0rtal domaine a été approuvé par Twitter. En revanche, si vous hébergez votre site Gallery sur un domaine personnalisé, vous devez remplir le processus d'approbation de la carte Twitter avant que la lecture du contenu soit intégrée dans les tweets. Pour en savoir plus sur les cartes Twitter et le processus d'approbation, consultez la documentation de Twitter.

Partage vers e-mail

Lorsque les spectateurs partagent du contenu vidéo sur un e-mail, l'objet et le corps de l'e-mail sont encodés dans un mailto lien pour le bouton de partage de courrier électronique. Chaque navigateur et client de messagerie a une longueur de caractères maximale qui est prise en charge. Il faut veiller à ce que les e-mails générés ne dépassent pas les limites de caractères imposées par les navigateurs et les clients de messagerie.

Configuration des paramètres de connexion

Les paramètres Rester connecté donnent aux spectateurs un accès direct à vos pages Facebook, Twitter et LinkedIn. Des icônes apparaîtront dans le pied de page donnant aux spectateurs un accès direct à vos sites de médias sociaux.

Clique le Rester connecté onglet, ajoutez vos informations de médias sociaux selon vos besoins, puis cliquez sur sauvegarder.

Les icônes peuvent également être configurées pour utiliser une icône Lumière sur l'icône sombre ou Dark sur le thème de couleur claire .
