    Généralités sur les modèles de portail Galerie

    Cette rubrique fournit une référence à tous les modèles de portail disponibles.
    Remarque : Pour plus d'informations sur les modèles d'expérience dans la page, reportez-vous à Généralités sur les modèles d'expérience dans la galerie.

    Lorsque vous créez une nouvelle expérience Gallery, vous choisirez un modèle qui contrôle l'aspect général de l'expérience. En sélectionnant le type d'expérience que vous souhaitez créer, vous filtrez la liste des modèles.

    Lorsque vous cliquez sur Portail, le sélecteur de modèles peut être utilisé pour filtrer les modèles de portail en fonction des besoins de votre site.

    Remarque : Après avoir sélectionné un modèle, le modèle ne peut pas être modifié.

    Les modèles suivants sont disponibles lors de la création d'expériences de portail, de page de destination et d'événements.

    • Mosaïque
    • Classique
    • Tente de réception
    • Editeur
    • Découverte
    • Portail des événements en direct
    • Page de destination
    • Vitrine
    • Chronique
    • Catalogue

    Pour plus d'informations sur les modèles d'expérience dans la page, reportez-vous à Généralités sur les modèles d'expérience dans la galerie.

    Remarque : les modèles personnalisés peuvent être également développés avec l'aide de Brightcove Consulting. Contactez votre représentant pour plus d’informations.

    Les modèles de portail fournissent un ensemble de modèles de page qui peuvent être personnalisés. Les modèles de page peuvent être affichés en ouvrant l'éditeur de site Galerie, puis en cliquant sur PAGES dans le menu de navigation de gauche.

    Cliquez sur un nom de page pour ouvrir la mise en page dans l'éditeur. Dans l'exemple ci-dessus, les pages Accueil, Lecture et Catégorie sont disponibles.

    Remarque : certaines pages permettent également de modifier la présentation du lecteur. Pour plus d'informations, voir Modification de la disposition du lecteur sur une expérience de portail.

    Mosaic

    Le modèle Mosaic propose un tableau perpétuel d’images avec un lecteur vidéo intégré à la page. Les vidéos apparentées s'affichent au-dessous. Avec le modèle Mosaic, les visuels prédominent sur le site. Le modèle Mosaic donne un site hautement visuel. Il est recommandé si vous voulez placer le visuel au cœur de l’expérience utilisateur. Ce modèle convient notamment aux médiathèques de petite taille associées à une structure de catégories plutôt réduite. Les publicités de la page et les appels à l'action sont également pris en charge.

    Le modèle Mosaic contient deux modèles de mise en page.

    • Home
    • Catégorie

    Cliquez ici pour afficher un exemple de site utilisant le modèle Mosaic.

    Classic

    Le modèle Classic conjugue de façon transparente le contenu éditorial et la vidéo. Ainsi, l’utilisateur dispose d’une navigation fluide dans une médiathèque. Le modèle Classic est recommandé si vous disposez d'une poignée de contenus vidéo spécifiques à présenter et mettre en avant accompagnés d'un contenu rédactionnel. Ce modèle intègre une hiérarchie de contenus, particulièrement adaptée aux médiathèques de grande taille. Notez que lorsque vous utilisez le modèle Classic :

    1. La première collection s'applique au carrousel, ceci est la zone de défilement de la page
    2. La deuxième collection s'applique à la Section One de la page d'accueil.
    3. La troisième collection s'applique à la Section Two de la page d'accueil.
    4. Les collections restantes sont utilisées pour remplir le volet de navigation de gauche.

    Le modèle Classic contient trois modèles de mise en page.

    • Home
    • Category
    • Video Detail

    Les publicités de la page et les appels à l'action sont également pris en charge. Cliquez ici pour afficher un exemple de site utilisant le modèle Classic.

    Marquee

    Le modèle Marquee offre une expérience utilisateur cohérente basée sur une page unique. La vidéo présentée est en vedette au centre et la médiathèque en dessous. Le modèle Marquee est recommandé si vous voulez mettre en avant une vidéo précise.

    Le modèle Marquee contient trois modèles de mise en page.

    • Home
    • Lecture
    • Catégorie

    Cliquez ici pour afficher un exemple de site utilisant le modèle Marquee. Les publicités de la page et les appels à l'action sont également pris en charge.

    Le modèle de sélection de sélection offre également la possibilité de modifier la disposition du lecteur pour toutes les pages en cliquant sur le lien Modifier la mise en page en regard du nom de la page. La mise en page actuelle est indiquée par une coche.

    Les présentations suivantes sont disponibles :

    • Standard
    • Grand
    • Bannière latérale
    • Two Side Regions

    De l'espace de messagerie supplémentaire est ajouté au modèle lors de l'utilisation des mises en page Side Banner ou Two Side Regions. Les limites de dimensions de ces zones sont les suivantes :

    • Disposition de la bannière latérale - L'image est verrouillée à une hauteur de 50px puis mise à l'échelle en largeur au rapport d'aspect
    • Mise en page Deux régions latérales - L'image est verrouillée à une hauteur de 300px, puis mise à l'échelle en largeur selon le rapport d'aspect

    Pour plus d'informations sur la modification de la disposition du lecteur, voir Modification de la disposition du lecteur sur une expérience de portail.

    Publisher

    Le modèle Publisher compte une page d’accueil centrée sur la vidéo et sur les images. Publisher propose un système de vidéos facilement découvrables, de survol des descriptions vidéo avec le curseur et de catégories/sous-catégories de contenu. Les publicités de la page et les appels à l'action sont également pris en charge.

    Le modèle Publisher contient trois modèles de mise en page.

    • Home
    • Category
    • Video Detail

    Les publicités de la page et les appels à l'action sont également pris en charge. Cliquez ici pour afficher un exemple de site utilisant le modèle Publisher.

    Découverte

    Le modèle Discovery convient aux médiathèques de grande taille et permet au visiteur de parcourir le contenu dans un format quadrillé. Il s’agit d’un modèle essentiellement visuel offrant peu d’espace pour le contenu éditorial.

    Le modèle Discovery contient trois modèles de mise en page.

    • Home
    • Category
    • Détails de la vidéo

    Cliquez ici pour afficher un exemple de site utilisant le modèle Discovery.

    Portail des événements en direct

    Le modèle Live Event Portal est conçu pour diffuser du contenu en direct à votre audience. Le modèle Live Event Portal offre des états avant, pendant et après l'événement et prend en charge les annonces et les appels à l'action au niveau de la page. L’intégration de services tiers permet d’ajouter des fonctionnalités de discussion et de commentaires.

    Le modèle Live Event Portal fournit cinq modèles de mise en page.

    • Homepage : Pre-Event
    • Homepage : Live
    • Homepage : Post-Event
    • Archive vidéo
    • Détails de la vidéo

    Cliquez ici pour voir un exemple de site à l'aide du modèle Live Event Portal. Pour plus d'informations sur l'utilisation du modèle Portail d'événements en direct, reportez-vous à la section Création d'une expérience de portail d'événements en direct.

    Page de destination

    Le modèle Landing Page permet de diffuser une vidéo unique avec des appels à l’action et des formulaires de clients potentiels.

    Le modèle Page de destination fournit uniquement une mise en page d' accueil .

    Cliquez ici pour afficher un exemple de site utilisant le modèle Landing Page. Pour plus d'informations sur l'utilisation du modèle de page de destination, reportez-vous à la section Création d'une expérience de page de destination.

    Le modèle Showcase convient aux médiathèques de grande taille et permet au visiteur de parcourir le contenu dans un format quadrillé. Il s’agit d’un modèle essentiellement visuel offrant peu d’espace pour le contenu éditorial.

    Le modèle Showcase contient trois modèles de mise en page.

    • Home
    • Category
    • Video Detail

    Les publicités de la page et les appels à l'action sont également pris en charge. Cliquez ici pour afficher un exemple de site utilisant le modèle Showcase.

    Le modèle Chronicle propose trois modèles de mise en page.

    • Home
    • Category
    • Video Detail

    Catalogue

    Le modèle de catalogue convient aux grandes bibliothèques de contenu et permet au spectateur de parcourir le contenu dans un format de grille. Il s'agit d'un modèle graphique qui prend en charge jusqu'à 4 niveaux de hiérarchie et prend en charge les images personnalisées et les descriptions pour les collections.

    Le modèle Catalogue fournit trois modèles de mise en page.

    • Accueil
    • Category
    • Video Detail
