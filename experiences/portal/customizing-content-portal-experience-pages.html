Personnalisation du contenu des pages d'expérience du portail

{{ page.description }}

Des objets tels que du texte, des images et des liens peuvent être ajoutés aux pages Expérience du portail, ce qui vous permet de personnaliser et de marquer votre expérience vidéo.

Note : Le processus d'ajout d'objets à une page varie légèrement en fonction du modèle. Les captures d'écran de cette rubrique ont été prises à l'aide du modèle Marquee.
Note : Certaines pages offrent également la possibilité de modifier la disposition du lecteur. Pour plus d'informations, voir Modification de la disposition du lecteur sur une expérience de portail.

Ajout d'objets à des pages

À l'aide de l'Éditeur de site, vous pouvez ajouter des objets tels que des images, du texte et des liens vers une page. Commencez par modifier une expérience, puis cliquez sur PAGES dans la navigation de gauche. Le modèle de page affichera des icônes plus () indiquant que les objets qui peuvent être ajoutés à la page. Passez le curseur sur l'icône plus et cliquez dessus pour ajouter l'objet approprié.

Le modèle de page peut également contenir des objets texte qui font partie du modèle. En passant le curseur sur eux, un bouton Modifier s'affiche. Cliquez sur l'objet ou sur le bouton Modifier pour modifier l'objet.

Notez que les objets pouvant être ajoutés ou modifiés sont basés sur le modèle sélectionné lors de la création de l'expérience.

Ajout de texte à une page

Le texte peut être ajouté à une page de deux manières :

  • En cliquant sur le +Ajouter une image ou du texte relier
  • En cliquant sur l'un des espaces réservés de texte sur la page

Ajoutez du texte en cliquant sur le +Ajouter une image ou du texte relier

Pour ajouter du texte à la page en cliquant sur le +Ajouter une image ou du texte relier:

  1. Cliquer sur .
  2. Saisissez votre texte. La barre de mise en forme peut être utilisée pour formater le texte. Cliquez sur Save une fois l'opération terminée.

Ajouter du texte en cliquant sur un espace réservé

Les pages peuvent contenir de nombreux objets texte dans le cadre du modèle. Cliquez sur l'objet texte pour le modifier.

Pour ajouter votre texte :

  1. Cliquez sur un des objets du paramètre substituable.
  2. Saisissez votre texte et cliquez sur Save.

Ajout d'une image à l'en-tête de page

Les images seront redimensionnées (jamais mises à l'échelle) en fonction du modèle. Le chargeur d'images peut afficher les dimensions d'image recommandées en fonction de l'endroit où l'image est utilisée. Les images JPEG, PNG et GIF sont prises en charge. Pour ajouter une image à l'en-tête de page :

  1. Cliquer sur .
  2. Clique le Image languette. La boîte de dialogue du navigateur d'images s'ouvre.
  3. Cliquez sur Parcourir, puis parcourez et sélectionnez une image dans le système de fichiers. Cliquez sur Save une fois l'opération terminée.

Ajout de liens de navigation à l'en-tête de page

Les liens de navigation peuvent être ajoutés en haut de la page et permettent aux utilisateurs d'accéder à d'autres sections de votre site. Pour ajouter un lien de navigation à une page :

  1. Cliquez sur +Ajouter Menu Links dans l'en-tête de page.
  2. Entrez le texte et le lien. Cochez New Window pour que le lien s'ouvre dans une nouvelle fenêtre de navigateur. Cliquez sur Save une fois l'opération terminée.

Vous pouvez saisir d'autres liens de navigation en cliquant sur +Add Link. Pour modifier un lien, cliquez dessus.

Ajout de liens de navigation au pied de page

Les liens de navigation peuvent être également ajoutés au pied de page. Ceci est utile pour avoir un titre et une série de liens ci-dessous. Par exemple :

L'objet de navigation de pied de page se compose de deux parties, un espace réservé pour le titre et +Ajouter des liens de menu pour les liens associés.

Pour ajouter des liens de navigation au pied de page :

  1. Cliquez sur ESPACE RÉSERVÉ, entrez une valeur pour l'en-tête et cliquez sur Enregistrer.
  2. Cliquez sur +Ajouter des liens de menu et entrez le Texte , Relier URL et Ordre de la liste. Vous pouvez également indiquer l'ordre des liens en définissant la valeur List Order lors de la création du lien.

Vous pouvez entrer des liens de navigation supplémentaires en cliquant sur +Ajouter des liens de menu.

Suppression des objets d'une page

Le texte, les images et les liens peuvent être supprimés de la page de deux manières.

Pour supprimer le texte ou les images de l'en-tête de page

  1. Survolez l'objet et cliquez sur l'icône Supprimer.

Pour supprimer des liens de navigation de l'en-tête de page

  1. Cliquez sur le lien dans l'en-tête de navigation.
  2. Cliquez sur Delete.

Pour supprimer les liens de navigation du pied de page

  1. Cliquez sur le lien dans le pied de page.
  2. Cliquez sur Delete.

Pour supprimer le titre du pied de page

  1. Cliquez sur le titre dans le pied de page.
  2. Désélectionnez le titre du texte et cliquez sur Save.
