Personnalisation des paramètres vidéo pour une expérience de portail

{{ page.description }}

Pour personnaliser les paramètres Vidéo, modifiez l'Expérience du portail et cliquez sur VIDÉO ET LECTURE > VIDÉO dans la navigation de gauche. Les paramètres Vidéo permettent de configurer les éléments suivants pour une expérience de portail :

  • Tri
  • Téléchargement de la vidéo
  • Lien connexe
  • Description longue
  • Sécurité vidéo

Assurez-vous de republier l'expérience après avoir apporté des modifications aux paramètres vidéo.

Activation du tri vidéo

Activer le tri des vidéos (non disponible pour les collections basées sur une liste de lecture) - Si cette option est cochée, les spectateurs auront la possibilité de trier les vidéos de la collection. La fonction de tri n'est pas disponible pour les collections qui ont été créées en utilisant des sélections. Ce paramètre n'est disponible qu'avec les modèles suivants :

  • Catalogue
  • Découverte
  • Portail des événements en direct
  • Marquee
  • Mosaic
  • Publisher
  • Classique

Configuration des téléchargements vidéo

Les paramètres de téléchargement vidéo permettent aux spectateurs de télécharger des vidéos. Le lien de téléchargement peut être utilisé pour télécharger le rendu de la plus haute qualité.

Les options suivantes sont disponibles :

  • Ne pas autoriser les téléchargements vidéo - Les téléchargements vidéo ne sont pas autorisés
  • Fournir un lien de téléchargement pour chaque vidéo - Un lien de téléchargement sera fourni pour chaque vidéo
  • Fournissez un lien de téléchargement basé sur une valeur de champ personnalisée - Les téléchargements vidéo seront fournis en fonction d'une valeur de champ personnalisée affectée à la vidéo. Lorsque vous sélectionnez cette option, vous définissez un champ personnalisé, puis définissez ce champ sur true pour toute vidéo pouvant être téléchargée. Pour obtenir des étapes complètes sur la configuration d'un lien de téléchargement pour chaque vidéo, reportez-vous à la section Téléchargement de vidéos à partir d'une expérience de portail.
Note : La vidéo doit contenir au moins un rendu MP4 disponible.

Affichage des liens connexes

Afficher le lien associé pour chaque vidéo - Si cette option est cochée, le lien associé de la vidéo s'affiche à côté du nom de la vidéo. Pour plus d'informations sur la configuration d'un lien connexe pour une vidéo, voir Modification des propriétés vidéo à l'aide du module multimédia.

Affichage de la description longue

Utilisez une description longue (si disponible) sur la page détaillée de la vidéo - Si cette option est cochée, la description longue de la vidéo sera affichée sur la page détaillée de la vidéo (au lieu de la brève description).

Configuration de la sécurité vidéo

Restreindre les vidéos sur site à celles sélectionnées via une recherche personnalisée ou une liste de lecture intelligente - Si cette option est cochée, lorsque la recherche est activée pour l'expérience, seules les vidéos des collections seront recherchées (recherche renvoie uniquement les vidéos ajoutées à une collection à l'aide d'une liste de lecture intelligente ou d'une recherche personnalisée). En outre, si le spectateur tente d'accéder à une vidéo qui a été supprimée d'une liste de lecture, elle sera redirigée vers la page d'accueil de l'expérience.

Si cette option n'est pas cochée, lorsque la recherche est activée pour une expérience, toutes les vidéos du compte Video Cloud sont recherchées (sauf si l'option include/exclude tag est utilisée).

Note : Si cette option n'apparaît pas, contactez votre responsable de compte Brightcove pour l'activer.
