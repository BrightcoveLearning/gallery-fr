Configuration des appels à l'action sur une expérience de portail

{{ page.description }}

Selon le modèle utilisé pour créer l'expérience du portail, certains modèles fournissent des zones désignées pour le contenu des appels à l'action. Il peut s'agit de HTML personnalisé, d'images ou de publicités. Sur les modèles, ces zones peuvent être étiquetées Banner, Video Grid CTA/AD, Hero CTA/AD, Catégorie CTA/AD ou Video CTA/AD et sont désignées avec une couleur de fond gris clair. Pour configurer les appels à l'action pour une page, cliquez sur une zone d'appel à l'action d'une page ou cliquez sur APPELS À L'ACTION dans la navigation de gauche, puis sélectionnez un modèle de page. La page Appels à l'action s'ouvre.

Notez que la page Appels à l'action diffère selon le modèle sélectionné. Dans la boîte de dialogue ci-dessus, le modèle Showcase a été utilisé lors de la création du site. Le modèle de page d'accueil prend en charge les appels à l'action dans les domaines suivants :

  • A - En-tête de page
  • B - Au milieu ou à la fin du flux vidéo à l'aide de CTA répétables
  • C - Lorsque la superposition s'ouvre pour lire une vidéo
  • D - Sur les appareils mobiles, sous la navigation du site
  • E - Sur les appareils mobiles, au milieu ou à la fin du flux vidéo à l'aide de CTA répétables

Pour configurer un appel à l'action, cliquez dans la zone appropriée et sélectionnez le type d'action pour la zone :

  • Aucune
  • Publicité
  • HTML personnalisé
  • Image

Configuration des CTA reproductibles

Lors de la configuration des appels à l'action, certaines zones désignées ont pris en charge les DEC reproductibles.

Pour configurer la région CTA reproductible, sélectionnez-la, puis sélectionnez quand le CTA doit être affiché.

Sélectionnez ensuite le type d'objet à afficher dans le spot, Publicité, HTML personnalisé ou Image. Pour ajouter des points supplémentaires, cliquez sur Ajouter un autre point répétitif.

Configuration d'une annonce

Pour configurer une publicité, sélectionnez Advertisement. La taille suggérée pour la publicité change en fonction de la zone sélectionnée. Collez le code d'annonce d'en-tête de document et le code unité d'annonce.

Cliquez sur Save pour enregistrer les modifications apportées.

Pour les pages sur lesquelles une seule vidéo peut être lue (pages Live et Video Detail), si vous choisissez Advertisement, deux options de personnalisation supplémentaires sont disponibles :

  • Champs personnalisés - Vous pouvez choisir d'afficher le code d'unité publicitaire basé sur une valeur de champ personnalisée. Par exemple, une vidéo avec une valeur personnalisée de « soccer » affichera des annonces de soccer. Les vidéos sans valeur de champ personnalisé afficheront des publicités en utilisant le Autrement code de bloc d'annonces.
  • Activer les annonces complémentaires - Si le serveur publicitaire est configuré pour fournir des annonces complémentaires, celles-ci s'affichent dans la zone d'appel à l'action lorsque la lecture d'une vidéo déclenche l'annonce associée.
    Note : Pour afficher une annonce de sauvegarde lorsque ni le champ personnalisé ni les critères d'annonces complémentaires ne sont remplis, vous devez fournir un code d'unité d'annonce sous Sinon, afficher, sinon un point vide sera affiché.

Configuration du code HTML personnalisé

Pour configurer l'affichage du HTML personnalisé, sélectionnez Custom HTML. La taille suggérée pour le code HTML change en fonction de la zone sélectionnée. Coller dans le code HTML personnalisé à afficher.

Cliquez sur Save pour enregistrer les modifications apportées.

Lors de la configuration du code HTML personnalisé, selon le modèle et la région configurés, il peut y avoir une option supplémentaire pour personnaliser le CTA qui s'affiche. Par exemple, lors de la configuration du CTA de superposition du modèle Showcase, il existe une option pour :

  • Afficher le même appel à l'action pour toutes les vidéos - Le même CTA est utilisé pour toutes les vidéos
  • Afficher l'appel à l'action basé sur des valeurs personnalisées - Le CTA peut changer en fonction des métadonnées personnalisées associées à la vidéo

Notez que plusieurs configurations peuvent également être ajoutées.

Configuration d'une image

Pour configurer l'affichage d'une image, sélectionnez Image. La taille suggérée pour l'image change en fonction de la zone sélectionnée. Glissez-déposez une image à partir du système de fichiers ou cliquez sur Choose File, localisez l'image à charger et sélectionnez-la. Pour ajouter un hyperlien d'accès à l'image, fournissez une URL dans le champ Link image to.

Cliquez sur Save pour enregistrer les modifications apportées.
