Personnalisation des paramètres du modèle de catalogue

{{ page.description }}

Lorsqu'une expérience de portail est créée à l'aide du modèle Catalogue, plusieurs paramètres spécifiques au modèle peuvent être configurés. Pour configurer les paramètres du modèle de catalogue, modifiez l'expérience et cliquez sur Paramètres du > modèle APPARENCE ET COMPORTEMENT dans la navigation de gauche.

Les paramètres suivants peuvent être configurés :

  • Navigation principale
  • Carrousel de fonctionnalités
  • Boutons de pagination de carrousel
  • Vignettes vidéo
  • Icônes de lecture de vignettes
  • Description de la vidéo

Navigation principale

Le paramètre Navigation principale contrôle la mise en page de la navigation sur le site :

  • Navigation à gauche : lorsqu'il y a plus de 3 collections, les noms des collections apparaissent dans la navigation de gauche. Lorsque vous utilisez cette option, un objet texte réservé HOME est ajouté à la mise en page pour permettre aux utilisateurs de revenir à la page d'accueil. Ce texte peut être personnalisé si nécessaire.
  • Navigation dans le menu réduit - Un menu de hamburger pliable s'affiche

Carrousel des vidéos à la une

Le paramètre Carrousel d'entités contrôle l'affichage des vidéos de la collection actuelle :

  • Grille de vignettes - La vignette des vidéos apparaît sous le lecteur
  • Vidéo uniquement - Une seule vidéo apparaît. < > des icônes apparaissent sur le lecteur pour accéder à d'autres vidéos de la collection

Boutons de pagination du carrousel

Le paramètre Boutons de pagination du carrousel détermine quand les boutons de pagination du carrousel sont affichés :

  • Toujours afficher les boutons de pagination - Les boutons de pagination sont toujours affichés
  • Afficher les boutons de pagination en survol uniquement - Les boutons de pagination ne sont affichés que lorsque la souris est au-dessus du lecteur ou des informations vidéo

Vignettes de vidéos

Le paramètre Miniatures vidéo contrôle la taille des images miniatures de la vidéo :

  • Petit
  • Grand

Icônes de lecture en vignette

Le paramètre Icônes de lecture des vignettes détermine si un bouton de lecture est affiché dans le lecteur :

  • Ne pas afficher les icônes de lecture - Aucune icône de lecture n'est affichée
  • Afficher les icônes de lecture par défaut - L'icône de lecture s'affiche

Description de la vidéo

Le paramètre Description vidéo contrôle la longueur de la description vidéo affichée :

  • Description tronquée : affiche une description vidéo tronquée
  • Afficher la description complète - Affiche la description complète de la vidéo
