Utilisation de polices personnalisées dans une expérience de portail

{{ page.description }}

L'éditeur de site Gallery permet de personnaliser le style d'une expérience de portail en modifiant le thème, les couleurs et les polices. Il est également possible d'utiliser vos propres polices personnalisées dans une expérience de portail. Dans cette rubrique, la police Acme du répertoire des polices Google sera utilisée pour personnaliser une expérience de portail.

Liaison vers un fichier de polices personnalisé

Le répertoire Google Font fournit des polices open source qui peuvent être utilisées sur les pages Web. Recherchez une police à utiliser, puis cliquez sur Sélectionnez ce style. Dans cette rubrique, la police Acme sera utilisée.

Selon l'aide de Google, les lignes suivantes doivent être intégrées dans le document.

<link rel="preconnect" href="https://fonts.gstatic.com"> 
<link href="https://fonts.googleapis.com/css2?family=Acme&display=swap" rel="stylesheet">

Pour inclure ces lignes dans une expérience Portal, procédez comme suit :

  1. Modifiez l'expérience.
  2. Cliquez sur APPARENCE ET COMPORTEMENT > En-tête et pied de page dans la barre de navigation de gauche.
  3. Collez le <link> balise dans le En-tête HTML personnalisé champ.
  4. Cliquez sur Save.

Création de CSS pour la police personnalisée

Pour utiliser la police personnalisée, une style balise est ajoutée au champ HTML En-tête personnalisé du portail pour styliser les objets texte appropriés.

Les outils de développement fournis avec votre navigateur sont utiles pour obtenir les noms de classe des objets sur la page que vous souhaitez styliser (objets pour lesquels nous voulons modifier la police). Notez que les noms des éléments seront modifiés en fonction du modèle utilisé. Dans l'exemple ci-dessous, nous allons modifier la police de la vidéo en vedette et la police utilisée pour les noms de vidéo qui apparaissent dans la grille vidéo sur la page. Les captures d'écran sont prises en utilisant Google Chrome.

  1. À l'aide des outils de développement de votre navigateur, inspectez l'élément sur l'expérience pour laquelle vous souhaitez modifier la police. Dans ce cas, le titre vidéo présenté est en cours d'inspection. Localisez le nom de classe de l'objet.
  2. Ajoutez le CSS approprié au champ HTML d'en-tête personnalisé pour styliser le titre de la vidéo en vedette avec la nouvelle police. 
      
      <style>
        .video-name {
         font-family:Acme;
       }
      </style>
  3. Localisez le nom de classe des titres vidéo dans la grille.
  4. Ajoutez le CSS approprié à l'intérieur de la <style> balise existante pour styliser les titres vidéo dans la grille avec la nouvelle police. 
    
      .bc-video-browser .bc-video-grid-items .bc-video-grid-item .caption {
       font-family:Acme;
    }
  5. Confirmez que le champ HTML d'en-tête personnalisé contient les éléments suivants. 
    <link rel="preconnect" href="https://fonts.gstatic.com"> 
    <link href="https://fonts.googleapis.com/css2?family=Acme&display=swap" rel="stylesheet">
  <style>
      .video-name {
      font-family:Acme;
    }

    .bc-video-browser .bc-video-grid-items .bc-video-grid-item .caption {
    font-family:Acme;
  }
</style>
    Note : Il est également possible d'enregistrer le CSS dans un fichier séparé et de l'associer à l'expérience à l'aide du champ URL CSS personnalisé.
  6. Cliquez sur Save.
  7. Cliquez sur Aperçu > réactif pour vérifier que la police mise à jour s'affiche.
  8. Lorsque vous avez terminé, Publiez l'expérience.

Le fichier CSS complet du modèle peut être consulté en cliquant sur le Afficher le CSS lien dans le cadre du APPARENCE ET COMPORTEMENT> Personnalisé Les paramètres. Si vous souhaitez modifier toutes les instances d'une police, vous pouvez rechercher toutes les occurrences de font-family dans le modèle CSS et ajouter les entrées appropriées à votre <style> balise.
