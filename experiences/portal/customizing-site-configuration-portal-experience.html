Personnalisation de la configuration du site d'une expérience de portail

{{ page.description }}

Pour configurer les paramètres de configuration de site pour une expérience de portail, modifiez l'expérience et cliquez sur CONFIGURATION DU SITE dans la navigation de gauche. Les paramètres suivants sont disponibles :

  • URL : configuration de l'expérience pour utiliser un domaine personnalisé
  • Optimisation des moteurs de recherche - Configurez les paramètres pour améliorer le référencement
  • Contrôle d'accès - Configurer les restrictions d'authentification unique et IP
  • Suivi tiers - Configurer l'intégration avec Google, Eloqua et Marketo

Utilisation d'un domaine personnalisé

L'utilisation d'un domaine personnalisé vous permet d'héberger une expérience de portail sur votre propre URL. Pour plus d'informations sur l'utilisation d'un domaine personnalisé, voir Affectation d'un domaine personnalisé à une expérience de portail.

Configuration des paramètres SEO

Les paramètres d'optimisation du moteur de recherche permettent de configurer l'image du site, les mots-clés et les méta-balises et d'accéder à la carte du site et aux URL du flux de recherche. Pour plus d'informations sur la configuration des paramètres d'optimisation du moteur de recherche, voir Configuration des paramètres d'optimisation du moteur de recherche pour une expérience de portail.

Configuration des paramètres de contrôle d'accès

Les paramètres de contrôle d'accès permettent de contrôler qui peut accéder à une expérience de portail. Gallery peut restreindre l'accès à l'aide d'un code d'accès simple, s'intégrer à des solutions d'authentification unique et restreindre l'accès aux sites en fonction de l'adresse IP du spectateur. Pour plus d'informations sur le contrôle de l'accès aux sites Gallery, reportez-vous à Contrôler l'accès à une expérience de portail.

Configuration des paramètres de suivi

Les expériences de portail peuvent être intégrées à d'autres sites tiers ou utiliser les services fournis par d'autres sites tiers, tels que :

  • Google Analytics
  • Google Tag Manager
  • Eloqua
  • Marketo

Pour plus d'informations sur l'intégration avec des sites tiers, consultez Configuration du suivi tiers pour une expérience de portail.
