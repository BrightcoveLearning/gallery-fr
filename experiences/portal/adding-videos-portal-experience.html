Ajout de vidéos à une expérience de portail

{{ page.description }}

À l'aide de l'éditeur de collection Gallery, vous pouvez sélectionner les vidéos à afficher sur Portal Experiences. Avec Portal Experiences, les vidéos sont organisées en collections. Les collections peuvent contenir des vidéos de votre compte Video Cloud ou d'une chaîne YouTube.

Pour ajouter des vidéos à une expérience de portail, cliquez sur l'icône Vidéos dans la navigation de gauche.

Création d'une collection

Les vidéos d'une expérience de portail sont organisées en collections. Pour créer une collection, procédez ainsi :

  1. Cliquez sur l'icône Videos dans le volet de navigation de gauche. La page Gérer les collections s'ouvre. Le côté droit de la boîte de dialogue affiche un aperçu de la mise en page vidéo de la page d'accueil pour le type de modèle sélectionné.
  2. Saisissez le nom de la collection et appuyez sur Entrée. Le nom sera affiché aux spectateurs, alors assurez-vous d'utiliser un nom significatif. Cliquez sur modifier () pour modifier le nom de la collection.
  3. (Facultatif) Pour indiquer qu'il s'agit d'une collection d'en-têtes, cliquez sur l'icône de dossier (). Les collections d'en-têtes ne sont associées à aucune vidéo et sont utilisées pour l'organisation.

Modification d'une collection

  1. Pour modifier une collection, cliquez sur le nom de la collection.
  2. La page Modifier la collection s'ouvre. La page Modifier la collection permet d'ajouter des vidéos à une collection et de configurer des propriétés de collection supplémentaires.
  3. Le menu déroulant Change Collection est une liste de toutes les collections créées. Sélectionnez la collection à modifier.
  4. Collection Source indique la source du contenu vidéo de la collection :
    • Video Cloud : le contenu vidéo Video Cloud sera utilisé. Pour plus d'informations, consultez Ajout de vidéos Video Cloud à une expérience de portail.
    • YouTube : le contenu vidéo Video Cloud sera utilisé. Pour plus d'informations, consultez Ajout de vidéos YouTube à une expérience de portail.
  5. Cliquez sur Save pour enregistrer les modifications apportées.

Suppression d'une collection

Pour supprimer une collection, procédez ainsi :

  1. Cliquez sur l'icône Videos dans le volet de navigation de gauche.
  2. Clique le Effacer icône () associé à la collection.
  3. Cliquez sur Supprimer pour confirmer la suppression.
  4. Lorsque vous avez terminé, cliquez sur Done.

Commande et imbrication de collections

Pour modifier l'ordre d'affichage des collections sur votre site, cliquez sur l'icône Vidéos dans la navigation de gauche.

Déplacement d'une collection

Cliquez et faites glisser sur le nom de la collection ou déplacez l'icône () et faites glisser la collection vers une nouvelle position. La ligne de séparation indique si les collections sont au même niveau ou imbriquées. Dans cet exemple, la collection Brightcove Player est déplacée vers le haut. La ligne de séparation indique que toutes les collections sont au même niveau.

Imbriquer une collection

Cliquez et faites glisser sur le nom de la collection ou déplacez l'icône () et faites glisser la collection vers une nouvelle position. Faire glisser vers la droite mettra en retrait la ligne de séparation indiquant que cette collection sera imbriquée. Dans cet exemple, le Module de téléchargement la collection est déplacée et la ligne de séparation indique qu'elle sera imbriquée sous le Studio Nuage Vidéo collection.

Une fois la collection imbriquée, l'imbrication sera reflétée dans la boîte de dialogue Gérer les collections.

Note : Le nombre de niveaux pouvant être imbriqués est basé sur le modèle.
Note : Les modèles Showcase, Mosaic et Chronicle ne prennent pas en charge l'imbrication.
