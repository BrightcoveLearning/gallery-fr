Téléchargement de vidéos à partir d'une expérience de portail

{{ page.description }}
 

Les expériences de portail peuvent être configurées pour afficher un lien de téléchargement permettant aux spectateurs de télécharger du contenu vidéo. Les sites peuvent être configurés pour fournir un lien de téléchargement pour toutes les vidéos ou vidéos spécifiques en fonction d'une valeur de champ personnalisée. Lorsqu'une vidéo peut être téléchargée, un lien de téléchargement s'affiche à côté de celle-ci. Le format rendu MP4 de la plus haute qualité disponible est téléchargé.

Note : La vidéo doit contenir au moins un rendu MP4 disponible.
Note : Les téléchargements vidéo ne sont pas pris en charge sur les appareils iOS.
Note : Les vidéos YouTube ne peuvent pas être téléchargées.

Pour fournir un lien de téléchargement pour toutes les vidéos, consultez Personnalisation des paramètres vidéo pour une expérience de portail. Pour ajouter la prise en charge vidéo de téléchargement à une expérience de portail, vous devez effectuer les opérations suivantes :

  1. Créer un champ personnalisé pour indiquer le contenu téléchargeable
  2. Étiqueter le contenu vidéo qui peut être téléchargé
  3. Configurer une expérience de portail pour autoriser les téléchargements

Création d'un champ personnalisé pour indiquer le contenu téléchargeable

Les vidéos pouvant être téléchargées sont indiquées en définissant la valeur d'un champ personnalisé. Commencez par créer un champ personnalisé dans votre compte. Il permettra d'indiquer qu'une vidéo peut être téléchargée.

  1. Connectez-vous à Video Cloud Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMIN, puis sur Champs vidéo. Notez que vous devez être un administrateur de compte pour accéder au menu ADMIN.
  3. Cliquez sur Ajouter un champ personnalisé.
  4. Créez un champ personnalisé qui permettra d'indiquer si une vidéo est téléchargeable. Le nom et la valeur affectée n'ont aucune importance. Dans l'exemple ci-dessous, un nouveau champ personnalisé downloadable est créé. Le Type est défini sur Liste, avec 2 valeurs, True et False. Les vidéos qui peuvent être téléchargées auront cette valeur définie sur True dans le cadre des propriétés de la vidéo.
    Note : Pour plus d'informations sur la création de champs personnalisés, voir Création de champs de métadonnées personnalisés.
  5. Cliquez sur Enregistrer le champ.

Étiqueter le contenu vidéo qui peut être téléchargé

Toute vidéo téléchargeable doit se voir affecter une valeur dans le champ personnalisé créé.

  1. Ouvrez le module Médias.
  2. Pour signaler une vidéo comme étant téléchargeable, cliquez sur la vidéo pour ouvrir les propriétés vidéo.
  3. Recherchez la section CHAMPS PERSONNALISÉS et cliquez sur Modifier.
  4. Définissez la valeur du champ personnalisé selon le cas. Dans cet exemple, la valeur de Downloadable est définie sur True.
  5. Cliquez sur Save.
  6. Modifiez la valeur du champ personnalisé pour toutes les vidéos supplémentaires si nécessaire.

Configuration d'une expérience de portail pour autoriser les téléchargements

Pour configurer une expérience de portail afin d'autoriser les téléchargements vidéo, procédez comme suit.

  1. Ouvrez le module Gallery.
  2. Recherchez l'expérience et cliquez sur Modifier.
  3. Cliquez sur VIDÉO ET LECTURE > VIDÉO dans la navigation de gauche.
  4. Pour indiquer que les vidéos à télécharger sont basées sur un champ personnalisé, dans la Téléchargement vidéo rubrique, sélectionnez le Fournir un lien de téléchargement basé sur une valeur de champ personnalisé option.
  5. Entrez le nom du champ personnalisé et la valeur qui ont été créés précédemment. Dans cet exemple, le champ personnalisé créé a été nommé downloadable et sa valeur est true pour les vidéos téléchargeables.
  6. Cliquez sur Enregistrer, puis republiez l'expérience.
  7. Vérifiez qu'un lien Download Video s'affiche avec les vidéos.
