Affectation d'un domaine personnalisé à une expérience de portail

{{ page.description }}
 

L'attribution d'un domaine personnalisé à une expérience de portail Gallery permet d'accéder au site à une URL facile à mémoriser (par opposition à une site-slug.bcvp0rtal.com URL). Lorsque vous utilisez des domaines personnalisés, vous pouvez également attribuer un chemin facultatif à l'URL. L'utilisation de chemins vous permet d'utiliser différents modèles, de créer des microsites, des campagnes, des portails internes, etc., tout en utilisant le même domaine. Par exemple, videos.brightcovegallery.com peut être étendu pour inclure également des chemins comme videos.brightcovegallery.com/event ou videos.brightcovegallery.com/product. Les chemins d'accès permettent aux éditeurs qui ont plusieurs sites et SSL de n'avoir à configurer SSL que pour un domaine plutôt que pour chaque site-slug individuel.

Note : Les domaines personnalisés doivent être créés dans le cadre des paramètres de la galerie avant de pouvoir être affectés à un site. Pour plus d'informations, reportez-vous à la section Configuration de domaines personnalisés et SSL.

Pour attribuer un domaine personnalisé à une expérience de portail, procédez comme suit :

  1. Modifiez l'expérience.
  2. Cliquez sur > URL DE CONFIGURATION DU SITE dans la navigation de gauche. Le domaine initial attribué au site sera affiché.
  3. (Facultatif) Décochez la case Active pour désactiver le site-xxxx.bcvp0rtal.com domain. En règle générale, les éditeurs désactivent le site-xxxx.bcvp0rtal.com lorsqu'ils utilisent un domaine personnalisé afin qu'il ne soit pas indexé par les moteurs de recherche.
  4. Cliquez sur Add Custom Domain.
  5. Par défaut, les domaines seront définis sur Active. Le fait de décocher la case Active empêche Gallery d'effectuer un routage sur l'entrée, mais ne libère pas la combinaison chemin/domaine à utiliser sur un autre site.
  6. Sélectionnez un domaine à partir du menu déroulant. La liste des domaines personnalisés, initialement configurés dans les paramètres Gallery, s'affiche. Par défaut, les domaines seront actifs.
  7. Le cas échéant, entrez un chemin d'accès pour le domaine.
  8. Pour ajouter des domaines à votre site, cliquez sur Add Custom Domain.
  9. Lorsque vous avez terminé, cliquez sur Enregistrer.

Cliquez sur l'icône Supprimer () pour supprimer un domaine personnalisé.

Note : Pour que toute modification de domaine (y compris l'activation ou la suppression d'un domaine) prenne effet, le site doit être publié.

La liste des domaines personnalisés sera répertoriée dans la boîte de dialogue de publication.

Pour désactiver ou supprimer l'un des domaines personnalisés, modifiez le site, cliquez sur CONFIGURATION DU SITE> URL et décochez la case actif case à cocher pour désactiver le domaine ou cliquez sur l'icône de suppression () pour supprimer le domaine personnalisé.
