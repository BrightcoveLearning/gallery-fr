Contrôle de l'accès à une expérience de portail

{{ page.description }}

Les profils de contrôle d'accès sont utilisés pour contrôler l'accès aux expériences de portail. Les profils de contrôle d'accès sont créés dans le cadre des paramètres de la Galerie et peuvent restreindre l'accès à un site en :

  • Limitation de l'accès en fonction de l'adresse IP
  • Obligation de l'utilisateur à entrer des informations d'identification (SSO)
  • Obligation de l'utilisateur à entrer un code d'accès

Pour plus d'informations sur la création de profils de contrôle d'accès, voir Configuration des profils de contrôle d'accès. Une fois les profils créés, ils peuvent être affectés à Portal Experiences pour contrôler l'accès. Pour affecter un profil de contrôle d'accès à un site, procédez comme suit :

  1. Modifiez votre expérience de portail.
  2. Cliquez sur le contrôle > d'accès CONFIGURATION DU SITE dans la navigation de gauche.
  3. Sélectionnez un profil de contrôle d'accès pour le site. Par défaut, tous les sites sont publics.
  4. Cliquez sur Save.

Assurez-vous de republier le site après avoir apporté des modifications aux paramètres de contrôle d'accès.
