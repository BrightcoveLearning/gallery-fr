Généralités sur les modèles d'expérience dans la page Galerie

{{ page.description }}
Note : Pour plus d'informations sur les autres modèles Gallery, reportez-vous à la section Présentation des modèles Gallery Portal.

Lorsque vous créez une nouvelle expérience dans la page, vous choisirez un modèle qui contrôle l'apparence de l'expérience. Après avoir sélectionné un modèle, le modèle ne peut pas être modifié. Les modèles suivants sont disponibles.

  • Carrousel
  • Grille
  • Liste de lecture horizontale
  • Vidéo unique
  • Films
  • Liste de lecture verticale
  • Événement en direct dans la page
Note : Des modèles personnalisés peuvent également être développés avec l'aide de Brightcove Consulting. Pour plus d'informations, contactez votre responsable de compte.

Carrousel

Le modèle Carrousel peut être utilisé pour afficher un nombre illimité de vidéos et offre plusieurs options de mise en page. Les vidéos apparentées s'affichent au-dessous. L'interactivité en joueur et les composants compagnons sont disponibles.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle Carrousel.

Grille

Le modèle Grille peut être utilisé pour afficher un nombre illimité de vidéos dans une grille de vignettes vidéo de taille égale. L'interactivité en joueur et les composants compagnons sont disponibles.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle Grille.

Liste de lecture horizontale

Le modèle de liste de lecture horizontale peut être utilisé pour afficher un nombre illimité de vidéos. La liste de lecture affiche des vignettes vidéo de taille égale sous le lecteur. L'interactivité en joueur et les composants compagnons sont disponibles.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle de liste de lecture horizontale.

Vidéo unique

Le modèle Single Video peut être utilisé pour mettre en évidence une vidéo et fonctionne bien pour une annonce de produit ou un événement. L'interactivité en joueur et les composants compagnons sont disponibles.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle Single Video.

Films

Le modèle Filmstrip va lire des vidéos dans une lightbox et est idéal pour mettre en évidence le contenu vidéo dans un espace limité. L'interactivité en joueur et les composants compagnons sont disponibles.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle Filmstrip.

Liste de lecture verticale

Le modèle de liste de lecture verticale peut être utilisé pour afficher un nombre illimité de vidéos. La liste de lecture affiche des vignettes vidéo uniformément dimensionnées à droite du lecteur. L'interactivité en joueur et les composants compagnons sont disponibles.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle de liste de lecture verticale.

Événement en direct dans la page

Le modèle Live Event In-Page est conçu pour diffuser du contenu en direct à votre audience. Le modèle Live Event In-Page offre des états avant, pendant et après l'événement et prend en charge les intégrations tierces pour les fonctionnalités de chat et de commentaire. L'interactivité en joueur et les composants compagnons sont disponibles.

Pour plus d'informations sur l'utilisation du modèle Événement en ligne, reportez-vous à la section Création d'une expérience en page d'événement en direct.

Cliquez ici pour voir un exemple d'expérience à l'aide du modèle Live Event In-Page.
