Ajout de vidéos à une expérience sur la page

{{ page.description }}

Les vidéos Video Cloud et YouTube peuvent être ajoutées à une expérience en page. Pour ajouter des vidéos à une expérience en page, modifiez l'expérience, puis cliquez sur VIDÉOS dans la navigation de gauche.

Note : Pour que des vidéos YouTube puissent être ajoutées à une expérience, vous devez configurer les paramètres YouTube.

Cliquez sur Ajouter des vidéos pour ouvrir le sélecteur vidéo, puis sélectionnez une source:

  • Video Cloud - Le contenu vidéo Video Cloud sera utilisé
  • YouTube - Le contenu vidéo YouTube sera utilisé (voir la section Détails de l'intégration YouTube de ce document pour plus de détails)
Note : Il est possible d'ajouter des vidéos de Video Cloud et YouTube à la même expérience In-Page.

Si la source YouTube a été sélectionnée, un sélecteur de chaîne apparaîtra vous permettant de sélectionner la chaîne YouTube à utiliser.

Lorsque Video Cloud Source est sélectionnée, sélectionnez une méthode pour ajouter les vidéos :

  • Sélectionner des vidéos- Les vidéos seront sélectionnées et utilisées manuellement
  • Sélectionner une liste de lecture- Les vidéos d'une liste de lecture seront utilisées
  • Utiliser l'ID de référence- Les vidéos d'un ID de référence de liste de lecture seront utilisées

Une fois les vidéos sélectionnées, elles seront affichées dans le corps de la page. Dans cet exemple, une playlist Video Cloud a été utilisée.

Note : Après avoir apporté des modifications à une expérience, assurez-vous de republier l'expérience afin que ces modifications apparaissent sur les sites qui affichent l'expérience.

Sélection manuelle des vidéos

Pour sélectionner manuellement des vidéos, sélectionnez une méthode de sélection des vidéos. Avec une source Video Cloud, le sélecteur vidéo affiche une liste des 100 dernières vidéos mises à jour dans votre compte Video Cloud. Les nouvelles vidéos seront récupérées à mesure que vous page vers le bas. Avec une source YouTube, toutes les vidéos de la chaîne sélectionnée seront affichées. Cliquez sur une vidéo pour la sélectionner, cliquez à nouveau dessus pour la désélectionner. Utilisez le champ de recherche pour rechercher des vidéos. Dans cet exemple, une source Video Cloud a été sélectionnée.

Lorsque vous avez terminé, cliquez sur Ajouter.

Recherche de vidéos

Le champ de recherche peut être utilisé pour rechercher des vidéos. Lors de la recherche de vidéos Video Cloud, Gallery recherche le nom de la vidéo, les descriptions courtes et longues, les balises et les champs personnalisés. Lors de la recherche de vidéos YouTube, Gallery utilise l'API de recherche YouTube.

Cliquez sur Autres options de recherche pour mieux contrôler les champs recherchés et pour inclure et exclure du texte. Notez que ces options sont uniquement disponibles lorsqu'une source Video Cloud est utilisée.

Utilisation d'une liste de lecture

Pour utiliser les vidéos d'une liste de lecture, sélectionnez une méthode de sélection de liste de lecture. Avec une source Video Cloud, le sélecteur vidéo affiche une liste des 20 listes de lecture les plus récentes mises à jour dans votre compte Video Cloud. Avec une source YouTube, le sélecteur vidéo affiche une liste de toutes les playlists de la chaîne sélectionnée. Le champ de recherche peut être utilisé pour rechercher des listes de lecture. Dans cet exemple, une source Video Cloud a été sélectionnée.

Notez qu'une seule liste de lecture peut être sélectionnée. Lorsque vous avez terminé, cliquez sur Sélectionner.

Utilisation d'un ID de référence de playlist

Pour utiliser les vidéos dans une playlist en spécifiant un ID de référence de playlist, sélectionnez un Méthode de Utiliser l'ID de référence. Cette option est uniquement prise en charge lors de l'utilisation d'une source Video Cloud. Le sélecteur vidéo fournit un champ permettant d'entrer l' ID de référence de la liste de lecture. En utilisant un ID de référence de playlist, les vidéos de l'expérience peuvent facilement changer en attribuant à nouveau l'ID de référence à une autre playlist à l'aide du module Media. L'expérience n'a pas besoin d'être modifiée et republiée lorsque vous devez modifier les vidéos qui apparaissent dans l'expérience.

Notez qu'un seul ID de référence de liste de lecture peut être utilisé. Lorsque vous avez terminé, cliquez sur Sélectionner.

Ajout de l'interactivité aux vidéos

Les interactions et les composants associés peuvent être ajoutés à une vidéo et affichés au fur et à mesure de la lecture de la vidéo. Les interactions incluent des liens et des cartes et sont affichées dans le lecteur lors de la lecture d'une vidéo. Les composants complémentaires incluent du HTML, des images et des objets texte et sont affichés en dehors du lecteur.

Les vidéos qui ont été configurées avec l'interactivité affichent une icône dans la colonne Interactivité indiquant le type d'interaction. Dans l'exemple suivant, les interactions entre les liens et les cartes ainsi qu'une image compagnon ont été configurées.

Pour plus d'informations sur l'ajout et la configuration d'interactions et de composants complémentaires, voir :

  • Ajout d'interactions à une expérience dans la page
  • Configuration des composants compagnons pour une expérience dans la page

Suppression de vidéos d'une expérience dans la page

Le processus de suppression de vidéos d'une expérience en page dépend de la méthode sélectionnée pour ajouter les vidéos :

  • Sélectionnez Vidéos - Cochez la case en regard de la ou des vidéos à supprimer, puis cliquez sur Supprimer les vidéos. Vous pouvez également cliquer sur l'icône de suppression () dans le Actions colonne.
  • Sélectionner la liste de lecture - Cliquez sur Supprimer la liste de lecture pour supprimer toutes les vidéos
  • Utiliser l'ID de référence : cliquez sur Supprimer la liste de lecture pour supprimer toutes les vidéos

Lorsqu'une vidéo est supprimée, les interactions ou les composants compagnons configurés sur cette vidéo ne seront pas supprimés. Si la vidéo est ensuite réajoutée à l'expérience, les interactions et les composants associés réapparaîtront.

Détails de l'intégration YouTube

  • Les vidéos YouTube ne peuvent pas être téléchargées
  • Avant d'ajouter des vidéos YouTube à un site vidéo, vous devez configurer les paramètres YouTube.
  • Les vidéos YouTube doivent avoir un paramètre de confidentialité de Public
  • Actuellement, Gallery ne collecte pas de données d'analyse pour les vidéos YouTube. Les vidéos YouTube dans une expérience en page n'affichent aucune donnée d'analyse dans le tableau de bord Experience Analytics
  • Les vidéos YouTube jouées dans Brightcove Players avec le plugin IMA n'afficheront pas de publicités sur les appareils mobiles
  • ' L'utilisation de YouTube par les clients est soumise aux conditions générales de YouTube
  • Lors de la recherche, toutes les collections doivent être composées de vidéos YouTube et les vidéos doivent être la propriété de la chaîne sélectionnée.
