Examen de l'analyse des appels à l'action de l'expérience sur la page

{{ page.description }}

L'analyse des appels à l'action fournit des analyses sur les clics et les conversions de formulaires pour une expérience en page. Pour consulter l'analyse des appels à l'action, modifiez l'expérience et cliquez sur EXPERIENCE ANALYTICS > Calls-to-Action dans la navigation de gauche.

Sélection d’une période

Pour spécifier la période du rapport, vous pouvez cliquer sur une date rapide ou utiliser le contrôle calendrier.

Utilisation d'une date rapide

Une série de dates rapides se trouvent en haut de la page. Les dates suivantes sont disponibles :

  • Aujourd'hui
  • Hier
  • 7 derniers jours
  • Les 30 derniers jours
  • Ce mois-ci - Le mois civil à ce jour - par exemple, si aujourd'hui est le 17 mars, ce rapport s'affiche du 1er mars au 17 mars
  • Le mois dernier - le mois civil précédent - par exemple, si aujourd'hui est le 14 février, ce rapport s'affiche du 1er janvier au 31 janvier.

Cliquez sur une date rapide pour mettre à jour le rapport.

Utilisation du champ Calendrier

Vous pouvez également sélectionner une date de début et de fin en cliquant sur le contrôle calendrier.

Cliquez sur Appliquer pour mettre à jour le rapport.

Vérification de l'analyse des clics

L'analyse des clics suit l'interactivité dans une expérience en page. Les objets suivants déclencheront le suivi des analyses de clics :

  • Interactions de liaison configurées avec une URL de lien
  • Interactions de carte configurées avec une URL de lien
  • Composants d'image configurés avec des URL cliquables

Pour consulter l'analyse des clics, cliquez sur Experience Analytics > Appels à l'action dans la navigation gauche, puis cliquez sur Clickthrough dans l'en-tête de page.

L'analyse des clics affiche les données suivantes pour l'expérience :

  • Nom de l'appel à l'action : nom de l'appel à l'action :
    • Liens : affiche le titre du lien
    • Cartes - Affiche le titre de la carte
    • Images : affiche le nom attribué lors de la création
    • HTML personnalisé : affiche le nom attribué lors de la création
  • Mise à jour : date et heure de la dernière mise à jour de l'interaction
  • Type : type d'interaction, de lien ou de carte
  • Impressions - Nombre de fois où l'interactivité a été affichée (avec des cartes, une impression est enregistrée lorsque l'icône de la carte () apparaît ; la carte n'a pas besoin d'être affichée pour qu'une impression soit enregistrée)
  • Clickthrough : nombre de fois où un lien a été cliqué, un lien d'une carte a été cliqué ou une URL de clickthrough sur un composant d'image a été cliqué
  • Taux de clics (%) - Pourcentage de spectateurs ayant cliqué sur un lien (Clickthrough divisé par Impressions)

Examen de l'analyse des conversions de formulaires

L'analyse des conversions de formulaires suit l'activité de soumission de formulaire et de lien HTML personnalisée dans une expérience en page. Les objets suivants déclencheront le suivi de l'analyse des conversions de formulaires :

  • Formulaires qui sont soumis à partir des composants associés HTML
  • Formulaires qui sont soumis à partir de composants HTML personnalisés
  • URL cliquables dans des composants HTML personnalisés

Pour consulter l'analyse des conversions de formulaires, cliquez sur Experience Analytics > Appels à l'action dans la navigation de gauche, puis cliquez sur Conversions de formulaire dans l'en-tête de page.

L'analyse des conversions de formulaires affiche les données suivantes pour l'expérience :

  • Nom de l'appel à l'action : nom du composant (le nom est attribué lorsque le composant est ajouté à l'expérience)
  • Mise à jour : date et heure de la dernière mise à jour du composant
  • Type - Type d'interaction (les composants associés HTML sont indiqués par un () icône
  • Impressions : nombre de fois où le composant a été affiché
  • Conversions de formulaires : nombre de fois où un formulaire a été envoyé ou qu'une URL dans un composant HTML personnalisé a été cliquée
  • Taux de conversion de formulaire (%) - Pourcentage de téléspectateurs ayant soumis un formulaire (Conversions de formulaires divisées par Impressions)
