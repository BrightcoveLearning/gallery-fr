Configuration des détails pour une expérience sur la page

{{ page.description }}

Le nom de l'expérience, la description et la taille de la mise en page peuvent tous être configurés dans le cadre des paramètres d'aperçu. Pour configurer les paramètres d'aperçu, modifiez l'expérience, puis cliquez sur APERÇU dans la navigation de gauche.

La page Vue d'ensemble s'affiche :

  • Type d'expérience : affiche le modèle utilisé pour l'expérience en page
  • Titre et description de l'expérience - Lorsque vous créez une nouvelle expérience pour la première fois, vous êtes invité à entrer le titre de l' expérience et la description de l' expérience. Ces valeurs peuvent être modifiées ici.
  • Langue de l'expérience : sélectionnez la langue qui sera utilisée pour afficher les messages de l'interface utilisateur
  • Taille de mise en page : détermine la taille de l'expérience dans la page. Voir Définition de la taille de l'expérience pour plus de détails.

Lorsque vous avez terminé, cliquez sur Enregistrer pour enregistrer vos modifications.

Note : Après avoir apporté des modifications à une expérience, assurez-vous de republier l'expérience afin que ces modifications apparaissent sur les sites qui affichent l'expérience.

Définition de la taille de l'expérience

Les expériences en page peuvent être dimensionnées de l'une des trois façons suivantes :

  • Responsive - L'expérience se redimensionnera automatiquement pour s'adapter à son conteneur.
    Note : Lorsque vous sélectionnez cette option, le code d'intégration recommandé doit être utilisé. Le code intégré No JavaScript n'est pas réactif.
  • Taille fixe - La taille de l'expérience sera dimensionnée en fonction du spécifié largeur et la taille valeurs.
  • Largeur fixe -La largeur de l'expérience sera définie sur la valeur spécifiée largeur valeur.

Lorsque vous avez terminé, cliquez sur Enregistrer pour enregistrer vos modifications.

Note : Pour modifier la taille de la mise en page, l'expérience doit être republiée et tout le code de publication intégré existant sera mis à jour. Pour plus d'informations, reportez-vous à la section Aperçu et publication d'une expérience intégrée à la page.
