Utilisation des remplacements de vidéo dynamique dans une expérience sur la page

{{ page.description }}

Il se peut que vous souhaitiez modifier les vidéos dans une expérience en page. Par exemple, les vidéos d'une expérience doivent peut-être être modifiées en fonction d'une sélection effectuée par l'utilisateur ou d'une navigation vers d'autres parties d'un site. Au lieu de créer de nouvelles expériences en page, chacune avec un ensemble de vidéos différent, il est possible de modifier le code d'intégration pour attribuer les vidéos appropriées à une expérience. Les modifications apportées au code d'intégration varient en fonction du type de code intégré utilisé :

  • Code d'intégration recommandé
  • Aucun code d'intégration JavaScript

Code d'intégration recommandé

Le code d'intégration typique de l'expérience dans la page (option recommandée) ressemble à ce qui suit :

<div data-experience="597a2aa114b2ed00103052af"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Pour remplacer dynamiquement les vidéos associées à une expérience en page, le code d'intégration peut être modifié pour ajouter un attribut à la div balise afin de spécifier les vidéos ou la liste de lecture à utiliser.

Spécification de vidéos individuelles

Pour spécifier des vidéos individuelles à utiliser, ajoutez le data-video-ids paramètre à la div balise, puis spécifiez une liste d'ID vidéo délimitée par des virgules.

<div data-experience="597a2aa114b2ed00103052af" 
     data-video-ids="12345,56789"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Spécification d'une liste de lecture

Pour spécifier une liste de lecture à utiliser, ajoutez le data-playlist-id paramètre à la div balise, puis spécifiez l'ID de playlist à utiliser.

<div data-experience="597a2aa114b2ed00103052af" 
     data-playlist-id="12345"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

Aucun code d'intégration JavaScript

Le code d'intégration typique de l'expérience dans la page (Aucune option JavaScript) ressemble à ce qui suit :

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Pour remplacer dynamiquement les vidéos associées à une expérience en page, le code d'intégration peut être modifié pour ajouter un paramètre à l' src attribut.

Spécification de vidéos individuelles

Pour spécifier des vidéos individuelles à utiliser, ajoutez le videoIds paramètre, puis spécifiez une liste d'ID vidéo délimitée par des virgules.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?videoIds=12345,56789""
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

Spécification d'une liste de lecture

Pour spécifier une liste de lecture à utiliser, ajoutez le playlistId paramètre, puis spécifiez l'ID de playlist à utiliser.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?playlistId=12345"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
