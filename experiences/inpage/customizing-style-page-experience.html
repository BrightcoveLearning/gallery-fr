Personnalisation du style d'une expérience sur la page

{{ page.description }}

Le style contrôle les polices, les tailles de police et les couleurs utilisées dans une expérience en page. Pour personnaliser le style, modifiez l'expérience, puis cliquez sur STYLE dans la navigation de gauche.

Note : Après avoir apporté des modifications à une expérience In-Page, assurez-vous de republier l'expérience afin que les modifications apparaissent sur les sites qui affichent l'expérience.

États du dispositif de commutation

Les liens situés en haut de la page peuvent être utilisés pour personnaliser les styles dans différents états de périphérique, y compris le bureau, la tablette ou le mobile.

Par défaut, le mode Bureau s'affiche lors de la personnalisation des styles. Les états du périphérique peuvent être utilisés pour prévisualiser à quoi ressemblera une expérience sur le type de périphérique sélectionné.

Changement d'états d'expérience

Lesexpériences en page ont des états visuels différents qui changent lorsque le spectateur interagit avec l'expérience. Le style de chacun de ces états peut être personnalisé en cliquant sur les liens en haut de la page.

Les états suivants sont disponibles pour personnaliser :

  • AVANT LA LECTURE - Comment l'expérience apparaîtra lors du premier rendu, avant le début de la lecture vidéo
  • LECTURE - Comment l'expérience apparaîtra lors de la lecture d'une vidéo
  • APRÈS LECTURE - Comment l'expérience apparaîtra après la lecture d'une vidéo (en supposant que la lecture automatique de la vidéo suivante soit désactivée)

Pour styliser chacun de ces états, cliquez sur un état, puis personnalisez les styles selon vos besoins.

Sélection d'un thème

Les thèmes fournissent un ensemble prédéfini de couleurs et de polices qui peuvent être utilisés pour styliser rapidement une expérience en page. Gallery fournit un ensemble de thèmes par défaut, mais vous pouvez également créer vos propres thèmes personnalisés qui sont partagés sur un compte. Une fois qu'un thème est appliqué à une expérience, vous pouvez utiliser les options de style pour personnaliser davantage l'apparence. Le thème actuel est affiché en haut du menu de style. Dans cet exemple, le thème Lumière est sélectionné.

Pour appliquer un thème à une expérience en page, procédez comme suit :

  1. Cliquez sur le thème actuel dans la navigation de gauche pour ouvrir le sélecteur de thème.
  2. Cliquez sur un thème pour le sélectionner. L'aperçu sera mis à jour pour afficher les couleurs du thème.
  3. Lorsque vous avez terminé, cliquez sur le lien < Retour aux styles en haut de la page.

Cliquez sur le lien Réinitialiser le thème par défaut pour appliquer le thème par défaut à l'expérience.

Détermination de l'objet à styliser

Pour déterminer quel objet styliser, essayez de survoler un objet sur l'expérience. La plupart du temps, une boîte verte entoure l'objet. Le style de cet objet sera activé et tous les autres styles qui ne s'appliquent pas seront grisés. Par exemple, le survol de la description de la superposition met en évidence à la fois le RECOUVRIR et DESCRIPTION DE LA SUPERPOSITION modes. Le style OVERLAY contrôle l'apparence de la superposition, y compris le titre de la superposition et la description de la superposition. L' OVERLAY DESCRIPTION contrôle l'apparence de l'objet de description vidéo. Avec les superpositions, l'ensemble de la superposition peut être stylé ou les styles du titre et de la description peuvent être stylisés individuellement.

Si aucune case verte n'est présente lors du survol d'un objet, cela indique que l'objet est considéré comme faisant partie du GLOBAL style et peut être stylisé en utilisant le GLOBAL modes. Les objets texte qui ont été ajoutés à l'expérience peuvent être stylisés à l'aide de la GLOBAL style à moins que des styles n'aient été modifiés à l'aide de l'éditeur de texte lorsque le texte a été placé sur l'expérience.

Modification des styles

Le panneau de style est utilisé pour personnaliser le style des objets sur une expérience. Le panneau de style peut être utilisé pour personnaliser :

  • Couleur du texte et couleur d'arrière-plan
  • Police
  • Taille de police

Un fichier CSS personnalisé peut également être spécifié.

Modification des valeurs de couleur de texte et d'arrière-plan

Pour sélectionner une valeur de couleur, procédez comme suit :

  1. Cliquez sur le cercle de couleur pour ouvrir le sélecteur de couleurs.
  2. Utilisez le curseur de couleur pour définir une plage de couleurs.
  3. Utilisez le curseur d'opacité pour définir l'opacité de la couleur.
  4. Cliquez sur le carré de couleur pour sélectionner une couleur.
  5. Cliquez sur le X pour fermer le sélecteur de couleurs.

Les couleurs peuvent également être spécifiées en entrant des codes de couleur :

  • Les valeurs de couleur HEX - HEX sont spécifiées avec : #RRGGBB. RR (rouge), GG (vert) et BB (bleu) sont des entiers hexadécimaux compris entre 00 et FF spécifiant l'intensité de la couleur.
  • Les valeurs de couleur RGBA - RGBA sont spécifiées avec : rgba (rouge, vert, bleu, alpha). Le paramètre alpha est un nombre compris entre 0,0 (entièrement transparent) et 1,0 (entièrement opaque).
  • Les valeurs de couleur HSLA - HSLA sont spécifiées avec : hsla (teinte, saturation, légèreté, alpha).

Cliquez sur l'icône haut/bas () pour basculer entre les modes de couleur.

Sélection d'une police

La police actuelle sera affichée dans le panneau de style. Utilisez la liste déroulante pour sélectionner une nouvelle police. Cliquez sur le X pour fermer le sélecteur de polices.

Sélection d'une taille de police

La taille de police actuelle s'affiche dans le panneau de style. Utilisez la liste déroulante pour sélectionner une nouvelle taille de police. Cliquez sur le X pour fermer le sélecteur de taille de police.

Tous les styles qui ont été modifiés affichent une flèche d'annulation.

Cliquez sur la flèche d'annulation () pour réinitialiser le style à sa valeur d'origine. Vous pouvez également réinitialiser tous les styles en cliquant sur Réinitialiser toutes les valeurs par défaut en bas du panneau de style.

Ajout de polices personnalisées

Il est également possible d'utiliser des polices personnalisées dans une expérience. Le répertoire Google Font fournit des polices open source qui peuvent être utilisées sur les pages Web. Recherchez une police à utiliser (cet exemple utilise la police Acme), puis cliquez sur Sélectionnez ce style. Les informations sur la police seront affichées. Le lien de la feuille de style pour la police sélectionnée sera nécessaire.

Pour ajouter une police personnalisée au sélecteur de polices :

  1. Modifiez l'expérience et cliquez sur STYLE dans la navigation de gauche.
  2. Cliquez sur l'une des listes déroulantes de police dans la navigation de gauche.
  3. Clique sur le Ajouter une police personnalisée relier.
  4. Entrez l'URL de votre fichier CSS de police personnalisé. Sur la base de l'exemple ci-dessus, l'URL du fichier CSS de police est la suivante : 
    https://fonts.googleapis.com/css2 ?family=Acme&display=swap
  5. Cliquez sur Ajoutez des polices à votre bibliothèque.
  6. Vérifiez que la police a été ajoutée en haut du sélecteur de police.
Note : Une collection de polices open source est disponible à l'adresse https://fonts.google.com.
Note : Si le fichier CSS se trouve derrière les CORs ou comporte un autre type de restriction de fichier, la police n'apparaîtra pas correctement sur l'expérience de création, les pages d'aperçu ou les liens partageables car cette URL n'a pas été mise en liste blanche pour permettre l'utilisation de la police. Si vous utilisez une police personnalisée, assurez-vous que la police est accessible par l'expérience.
Note : Les expériences In-Page ne prennent actuellement pas en charge les polices intégrées base64. La solution de contournement consiste à ajouter un fichier CSS personnalisé via une URL CSS qui remplace plutôt la police de l'expérience.

Utilisation d'un fichier CSS personnalisé

Pour utiliser les styles d'un fichier CSS personnalisé :

  1. Cliquez sur CSS personnalisé dans la navigation de gauche.
  2. Entrez l'URL de votre fichier CSS ou collez dans le code CSS.
  3. Cliquez sur Save.
