Ajout de composants à une expérience sur la page

{{ page.description }}

Des composants peuvent être ajoutés à une expérience en page, ce qui vous permet de personnaliser et de marquer l'expérience. Pour ajouter des composants à une expérience en page, modifiez l'expérience, puis cliquez sur MISE EN PAGE dans la navigation de gauche.

Cliquez sur l'icône plus () où vous souhaitez ajouter un composant, puis sélectionnez le type de composant à ajouter.

Les types de composants suivants peuvent être ajoutés :

  • Publicité
  • Chat
  • HTML personnalisé
  • Image
  • Texte
  • 3Play
  • Flux Twitter
Note : Après avoir apporté des modifications à une expérience, assurez-vous de republier l'expérience afin que ces modifications apparaissent sur les sites qui affichent l'expérience.

Ajout d'un composant de publicité

Pour ajouter une annonce à une expérience :

  1. Cliquez sur Publicité dans le sélecteur de composants.
  2. Entrez un nom pour l'annonce.
  3. Saisissez le code annonce en-tête et le code unité d'annonce. Lecode publicitaire d'en-tête sera partagé entre toutes les publicités de l'expérience.
  4. Sélectionnez le moment où la publicité doit être affichée.
  5. Cliquez sur Save. La publicité apparaîtra sur l'expérience.
Note : Si vous disposez d'une expérience dans la page avec plusieurs annonces, assurez-vous que chaque code d'unité d'annonce possède un div identifiant unique. Vous devrez peut-être modifier l'ID du paramètre div et le conteneur de adblock1 pour correspondre à cet identifiant.

Ajout d'un composant de chat

Un widget de chat tiers peut être ajouté à une expérience pour fournir des fonctionnalités de chat. Pour ajouter un composant de chat à une expérience :

  1. Cliquez sur Chat sur le sélecteur de composants.
  2. Entrez un nom pour le composant chat.
  3. Entrez le code HTML du chat.
  4. Sélectionnez le moment où le chat doit être affiché.
  5. Cliquez sur Save. Le chat sera rendu sur l'expérience.

Ajout d'un composant HTML personnalisé

Les composants personnalisés peuvent être utilisés pour ajouter votre propre code HTML personnalisé à une expérience. Par exemple, si vous avez créé un formulaire de prospect personnalisé à l'aide d'une plateforme d'automatisation marketing (MAP), le formulaire de prospect HTML du MAP peut être ajouté à une expérience en page à l'aide du composant HTML personnalisé.

Pour ajouter du code HTML personnalisé à une expérience :

  1. Cliquez sur Personnaliser dans le sélecteur de composants.
  2. Entrez un nom pour le composant HTML.
  3. Entrez le code HTML personnalisé à ajouter à l'expérience.
  4. Sélectionnez le moment où le code HTML doit être affiché.
  5. Cliquez sur Save. Le code HTML sera rendu sur l'expérience.

Si aucun code HTML n'est entré, un espace réservé HTML sera affiché.

Note : L'environnement de création de la Galerie élimine les scripts des composants HTML personnalisés pour des raisons de sécurité. Les scripts s'exécutent lorsque l'expérience en page est prévisualisée et publiée. Cela peut entraîner une apparence différente des composants HTML personnalisés, surtout s'ils contiennent beaucoup de contenu JavaScript.

Ajout d'un composant d'image

Pour ajouter une image à une expérience :

  1. Cliquez sur Image dans le sélecteur de composants.
  2. Entrez un nom pour l'image.
  3. Pour spécifier une image, vous pouvez :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  4. Définissez la taille de l'image.
    • L'image sera redimensionnée en conteneur - Les images dont la taille est plus grande que le conteneur dans lequel elles se trouvent seront réduites pour s'adapter. Les images plus petites ne seront pas mises à l'échelle.
    • Largeur et hauteur : entrez manuellement une largeur et une hauteur pour l'image.
  5. Définissez l' alignement de l'image.
  6. Saisissez une URL de clics. Il s'agit de l'URL à laquelle vous accédez lorsque l'image est cliquée. Vous pouvez également ouvrir le lien dans une nouvelle fenêtre.
  7. Sélectionnez le moment où l'image doit être affichée.
  8. Cliquez sur Enregistrer pour ajouter l'image à l'expérience.

Si aucune image n'est sélectionnée, un espace réservé d'image s'affiche.

Ajout d'un composant de texte

Pour ajouter du texte à une expérience :

  1. Cliquez sur Texte dans le sélecteur de composants.
  2. Entrez le texte à ajouter à l'expérience. La barre d'outils peut être utilisée pour formater le texte.
  3. Entrez un nom pour le texte.
  4. Sélectionnez le moment où le texte doit être affiché.
  5. Cliquez sur Enregistrer pour ajouter le texte. Le texte apparaîtra sur l'expérience.

Si aucun texte n'est saisi, un espace réservé au texte sera affiché.

Ajout d'un composant 3Play

Les composants 3Play peuvent être utilisés pour ajouter une transcription interactive 3Play à une expérience. Ce composant ne fonctionnera que si vos sous-titres vidéo ont été générés par 3Play Media. Lorsque ce composant est ajouté à une expérience, un lien Afficher la transcription interactive apparaît. Lorsque vous cliquez dessus, la transcription vidéo s'affiche. Dans l'exemple suivant, un composant 3Play a été ajouté sous le lecteur vidéo et a été configuré pour apparaître pendant le En jouant Etat.

Pour un exemple en direct, consultez ce lien et lisez une vidéo.

Note : Pour que le composant 3Play soit activé pour votre compte, contactez votre responsable de compte ou le support Brightcove.

Pour ajouter un composant 3Play à une expérience :

  1. Cliquez sur 3Play sur le sélecteur de composants.
  2. Entrez un nom pour le composant 3Play.
  3. Saisissez votre ID de projet 3Play. Pour trouver votre ID de projet 3Play, connectez-vous à votre compte 3Play et cliquez sur le lien Aperçu du compte dans l'en-tête de la page. L' ID du projet apparaîtra sur cette page.
  4. Sélectionnez quand le composant 3Play doit être affiché.
  5. Cliquez sur Enregistrer pour ajouter le composant 3Play. Le composant 3Play apparaîtra sur l'expérience.

Ajout d'un composant de flux Twitter

Pour ajouter un flux Twitter à une expérience :

  1. Cliquez sur Flux Twitter dans le sélecteur de composants.
  2. Entrez un nom pour le flux Twitter.
  3. Créez un widget sur le site Twitter. Si vous avez besoin d'aide pour créer un widget Twitter, consultez la documentation de développement de Twitter.
  4. Copiez le code HTML du widget Twitter et collez-le dans les paramètres du flux Twitter.
  5. Sélectionnez le moment où le flux Twitter doit être affiché.
  6. Cliquez sur Enregistrer pour ajouter le flux Twitter. Le flux Twitter apparaîtra sur l'expérience.
