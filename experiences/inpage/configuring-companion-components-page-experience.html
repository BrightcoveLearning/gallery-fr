Configuration des composants compagnons pour une expérience dans la page

{{ page.description }}

Tous les modèles d'expérience en page offrent la possibilité d'avoir des composants complémentaires. Les composants complémentaires comprennent :

  • HTML personnalisé
  • Images
  • Texte

Avec les composants compagnons, le contenu en dehors du lecteur peut changer en fonction de la vidéo en cours de lecture. Par exemple, lorsqu'une vidéo produit est lue, il peut y avoir une image cliquable à droite du lecteur qui ouvrira plus de détails sur ce produit. Les composants compagnons sont similaires aux interactions de carte et de lien, la différence étant que les cartes et les liens s'affichent sur le lecteur vidéo lui-même. Les composants compagnons s'affichent à droite du lecteur vidéo lorsque la vidéo est en état LECTURE.

Note : Pour plus d'informations sur l'ajout d'interactions de liens et de cartes, voir Ajout d'interactions à une expérience In-Page.
Note : Après avoir apporté des modifications à une expérience, assurez-vous de republier l'expérience afin que ces modifications apparaissent sur les sites qui affichent l'expérience.

Affichage des composants compagnons

Pour afficher les composants qui ont été configurés, modifiez l'expérience, puis cliquez sur VIDEOS dans la navigation de gauche. Les vidéos qui ont été configurées avec des interactions afficheront une icône dans la colonne Interactivité indiquant le type d'interaction.

Les composants d'accompagnement incluent HTML (), images () et texte ().

Note : Relier () et carte () peuvent apparaître, indiquant que des liens et des cartes sont configurés dans la vidéo. Pour plus d'informations sur les liens et les cartes, reportez-vous à la section Ajout d'interactions à une expérience In-Page.

Configuration d'un composant HTML compagnon

Pour configurer un composant HTML compagnon, procédez comme suit :

  1. Modifiez l'expérience, puis cliquez sur VIDÉOS dans la navigation de gauche.
  2. Cliquez sur une vidéo pour la sélectionner.
  3. Cliquez sur Ajouter de l'interactivité> HTML d'accompagnement ou cliquez sur le HTML () icône dans le Actions colonne. L'éditeur d'interactivité s'ouvrira.
  4. Entrez un nom pour le composant HTML.
  5. Collez dans votre code HTML.
  6. Pour configurer l'heure de début et la durée du composant, cliquez sur le bouton Configurer relier.
  7. Sélectionnez une heure et une durée de début:
    • Afficher pour toute la vidéo - Le composant apparaîtra pour l'ensemble de la vidéo
    • Sélectionnez une heure et une durée de début : vous définissez manuellement l'heure et la durée de début du composant.
  8. Définissez la durée du composant.
    Note : La durée peut également être définie à l'aide du curseur sous le joueur. Faites glisser la flèche gauche jusqu'au moment où le composant doit apparaître, puis faites glisser la flèche droite jusqu'à ce que le composant disparaisse.
  9. Cochez Jusqu'à la fin de la vidéo si vous souhaitez définir une heure de début uniquement et faire apparaître le composant jusqu'à ce que la vidéo se termine.
  10. Cochez Appliquer cette interaction à toutes les vidéos de la mise en page si vous souhaitez que ce composant apparaisse sur toutes les vidéos de l'expérience.
  11. Cliquez sur Save pour enregistrer les modifications apportées.
Note : Pour attribuer l'interaction à une autre vidéo, sélectionnez une autre vidéo à l'aide de la liste déroulante du sélecteur vidéo au-dessus du lecteur.

Configuration d'un composant d'image compagnon

Pour configurer un composant d'image compagnon, procédez comme suit :

  1. Modifiez l'expérience, puis cliquez sur VIDÉOS dans la navigation de gauche.
  2. Cliquez sur une vidéo pour la sélectionner.
  3. Cliquez sur Ajouter de l'interactivité> Image associée ou cliquez sur le image () icône dans le Actions colonne. L'éditeur d'interactivité s'ouvrira.
  4. Entrez un nom pour le composant d'image.
  5. Pour spécifier une image, vous pouvez :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  6. Définissez la taille de l'image.
    • L'image sera redimensionnée en conteneur - Les images dont la taille est plus grande que le conteneur dans lequel elles se trouvent seront réduites pour s'adapter. Les images plus petites ne seront pas mises à l'échelle.
    • Largeur et hauteur : entrez manuellement une largeur et une hauteur pour l'image.
  7. Définissez l' alignement de l'image.
  8. Saisissez une URL de clics. Il s'agit de l'URL à laquelle vous accédez lorsque l'image est cliquée. Vous pouvez également ouvrir le lien dans une nouvelle fenêtre.
  9. Pour configurer l'heure de début et la durée du composant, cliquez sur le bouton Configurer relier.
  10. Sélectionnez une heure et une durée de début:
    • Afficher pour toute la vidéo - Le composant apparaîtra pour l'ensemble de la vidéo
    • Sélectionnez une heure et une durée de début : vous définissez manuellement l'heure et la durée de début du composant.
  11. Définissez la durée du composant.
    Note : La durée peut également être définie à l'aide du curseur sous le joueur. Faites glisser la flèche gauche jusqu'au moment où le composant doit apparaître, puis faites glisser la flèche droite jusqu'à ce que le composant disparaisse.
  12. Cochez Jusqu'à la fin de la vidéo si vous souhaitez définir une heure de début uniquement et faire apparaître le composant jusqu'à ce que la vidéo se termine.
  13. Cochez Appliquer cette interaction à toutes les vidéos de la mise en page si vous souhaitez que ce composant apparaisse sur toutes les vidéos de l'expérience.
  14. Cliquez sur Save pour enregistrer les modifications apportées.
Note : Pour attribuer l'interaction à une autre vidéo, sélectionnez une autre vidéo à l'aide de la liste déroulante du sélecteur vidéo au-dessus du lecteur.

Édition/Suppression de composants compagnons

Le survol d'une icône dans la colonne Interactivité affiche toutes les interactions de ce type. Dans cet exemple, un composant d'image a été configuré.

Pour modifier un composant, survolez une icône dans le Interactivité colonne et cliquez sur modifier () icône à côté du composantt ou cliquez sur le titre de la vidéo. L'éditeur d'interactivité s'ouvrira. Modifiez le composant, puis Enregistrez les modifications.

Pour supprimer un composant, survolez une icône dans le Interactivité colonne et cliquez sur supprimer () à côté du composant.

Les composants peuvent également être modifiés/supprimés en cliquant sur l'icône de modification () dans Actions colonne. Cela affichera une liste de toutes les interactions pour la vidéo.

Cliquez sur l'icône Modifier pour modifier le composant ou supprimer l'icône pour supprimer l'interaction.

Les composants compagnons sont également accessibles à partir du DISPOSITION menu. Sélectionnez l'état PLAYING et les icônes compagnons s'afficheront sur l'expérience.
