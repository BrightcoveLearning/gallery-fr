Personnalisation d'une expérience sur la page

{{ page.description }}

Des objets tels que du texte, des images, des annonces, du HTML personnalisé et des flux Twitter peuvent être ajoutés à une expérience en page vous permettant de personnaliser et de marquer l'expérience. Pour ajouter des objets à une expérience en page, modifiez l'expérience, puis cliquez sur MISE EN PAGE dans la navigation de gauche. Lorsque vous modifiez une expérience dans la page, le mode LAYOUT s'ouvre par défaut.

Sélection d'un type de périphérique

Le sélecteur de périphérique situé en haut de la page peut être utilisé pour modifier l' expérience en page à l'aide d'une mise en page de bureau, tablette ou mobile.

Par défaut, la mise en page Bureau s'affiche lorsque l'expérience est modifiée. La sélection de la disposition mobile affichera la disposition mobile.

Notez que la prévisualisation d'une expérience ouvrira l'expérience dans un autre onglet du navigateur et vous permettra de prévisualiser l'expérience sans les éléments de l'interface utilisateur de la Galerie.

Changement d'états d'expérience

J'expérience n pages ont différents états visuels qui vont changer au fur et à mesure que le spectateur interagit avec elle. L'apparence de chacun de ces états peut être personnalisée en cliquant sur les liens en haut de la page.

Les états suivants sont disponibles pour personnaliser :

  • AVANT LA LECTURE - Comment l'expérience apparaîtra lors du premier rendu, avant le début de la lecture vidéo
  • LECTURE - Comment l'expérience apparaîtra lors de la lecture d'une vidéo
  • APRÈS LE JEU - Comment l'expérience apparaîtra après la lecture d'une vidéo
Note : Le modèle Événement Live In-Page est organisé différemment de la plupart des autres modèles en page. Pour plus d'informations, reportez-vous à la section Création d'une expérience Live Event In-Page.

Dans le EN JOUANT et APRÈS LE JEU états, en cliquant sur le < Arrière lien renverra l'expérience au AVANT DE JOUER Etat. L'expérience entrera dans le APRÈS LE JEU état, que l'option de lecture automatique de la prochaine vidéo soit activée ou non.

Note : Tous les modèles n'afficheront pas de lien « Précédent ».

Pour modifier les différents états d'expérience, cliquez sur un état, puis ajoutez des composants à l'expérience selon vos besoins. Le menu Dispositions propose d'autres options de personnalisation pour chaque état. Voir le Personnaliser l'apparence rubrique pour plus de détails.

Personnalisation de l'apparence

Le menu LAYOUT fournit des paramètres supplémentaires qui peuvent être utilisés pour personnaliser l'expérience dans la page.

  • Présentation
  • Infos vidéo
  • Vignettes
  • Partage

Notez que les options de mise en page disponibles sont basées sur le modèle sélectionné.

Paramètre de présentation

Le paramètre Présentation contrôle la présentation de l'expérience. Cliquez sur Présentation dans la navigation de gauche, sélectionnez une présentation, puis cliquez sur Enregistrer.

Note : Le modèle de liste de lecture horizontale ne fournit aucune option de présentation.

Paramètres d'informations vidéo

Les Informations sur la vidéo le réglage des contrôles où le nom de la vidéo et brève description apparaîtra. Cliquez sur Infos vidéo dans la navigation de gauche, choisissez un emplacement, puis cliquez sur Enregistrer.

Paramètre des vignettes

Le paramètre Miniatures contrôle l'emplacement du nom de la vidéo par rapport à la miniature de la vidéo. Cliquez sur Miniatures dans la navigation de gauche, choisissez un emplacement, puis cliquez sur Enregistrer.

Paramètres de partage

Le paramètre Partage contrôle l'emplacement des icônes de partage social. Cliquez sur Partage dans la navigation de gauche, choisissez un emplacement, puis cliquez sur Enregistrer.

Ajout de composants à une expérience en page

À l'aide de l'éditeur de mise en page, vous pouvez ajouter des composants à l'expérience. Commencez par cliquer sur l'icône plus () où vous souhaitez ajouter un composant, puis sélectionnez le type de composant à ajouter.

Pour plus d'informations sur les composants pouvant être ajoutés à une expérience en page, reportez-vous à la section Ajout de composants à une expérience dans la page.

Modification et suppression de composants

Pour modifier ou supprimer un composant dans une expérience en page, passez la souris sur le composant. Une icône de modification/suppression apparaîtra. Cliquez sur Modifier pour ouvrir l'éditeur de composants dans lequel le composant peut être modifié. Cliquez sur Supprimer pour supprimer le composant de l'expérience.

Note : Vous pouvez également modifier un composant en double-cliquant dessus.

Réorganisation des composants

Les composants qui se trouvent sur une expérience peuvent être déplacés à l'aide du glisser-déposer. Pour modifier l'ordre des composants, cliquez et faites-le glisser vers la bonne position. La ligne verte indique où se trouve le composant.

Les composants peuvent également être déplacés vers un autre emplacement de l'expérience.

Prévisualisation d'une expérience dans la page

La prévisualisation d'une expérience ouvrira l'expérience dans un nouvel onglet de navigateur afin qu'elle puisse être visualisée en dehors de l'environnement de création de la Galerie. L'aperçu peut être utilisé pour s'assurer que l'expérience est rendue comme prévu. L'aperçu est réactif afin que vous puissiez rétrécir ou élargir les fenêtres de votre navigateur pour voir les points de rupture. Pour prévisualiser une expérience, cliquez sur le Aperçu bouton.

Publication d'une expérience dans la page

La publication d'une expérience prendra les modifications qui ont été apportées et les publiera dans la production In-Page Experience. Toutes les modifications apportées à une expérience doivent être publiées avant qu'elles apparaissent. Pour publier une expérience dans la page, cliquez sur Publier et intégrer. Pour plus d'informations sur l'aperçu et la publication d'une expérience dans la page, reportez-vous à la section Prévisualisation et publication d'une expérience dans la page.
