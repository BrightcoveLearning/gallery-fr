Création de thèmes personnalisés pour les expériences sur la page

{{ page.description }}

Les thèmes fournissent un ensemble prédéfini de couleurs et de polices qui peuvent être utilisés pour styliser rapidement les expériences en page. Gallery fournit un ensemble de thèmes par défaut, mais vous pouvez également créer vos propres thèmes personnalisés. Tous les thèmes personnalisés créés sont partagés sur un compte Video Cloud. Une fois qu'un thème est appliqué à une expérience en page, vous pouvez utiliser les options de style pour personnaliser davantage l'apparence de l'expérience.

Création d'un thème personnalisé

Pour créer un thème personnalisé, procédez comme suit :

Note : Si vous avez déjà des thèmes, il peut être plus rapide de dupliquer un thème, puis d'apporter des modifications plutôt que de créer un nouveau thème.
  1. Modifiez l'expérience, puis cliquez sur STYLE dans la navigation de gauche. Le thème actuel est affiché en haut du menu de style.
  2. Cliquez sur le thème. Une liste de tous les thèmes sera affichée.
  3. Clique le Ajouter un thème personnalisé relier.
  4. Entrez un nom pour le thème.
  5. Modifiez les styles selon les besoins. Toutes les modifications apportées aux paramètres de style seront enregistrées automatiquement.
  6. Lorsque vous avez terminé, cliquez sur < Retour aux styles.

Application d'un thème à une expérience en page

Pour appliquer un thème à une expérience en page, procédez comme suit :

  1. Modifiez le projet, puis cliquez sur STYLE dans la navigation de gauche. Le thème actuel est affiché en haut du menu de style.
  2. Cliquez sur le thème. Une liste de tous les thèmes sera affichée. Cliquez sur un nouveau thème pour le sélectionner.
  3. Lorsque vous avez terminé, cliquez sur < Retour aux styles.

Modification d'un thème

Lorsque vous modifiez un thème, toutes les modifications de style sont appliquées à toutes les expériences dans la page existantes à laquelle le thème a été appliqué, sauf si un paramètre de style spécifique a été modifié dans l'expérience. Par exemple, un thème d'entreprise est créé, Thème d'entreprise, et la couleur du texte de superposition est définie sur rouge. Une nouvelle expérience dans la page est créée et la couleur du texte de superposition devient bleue. Si quelqu'un applique le thème d'entreprise à cette nouvelle expérience, toutes les modifications de style seront remplacées par les styles spécifiés dans le thème d'entreprise, à l' exception de la couleur du texte superposé comme ce style. a été modifié plus tôt.

Pour modifier un thème, procédez comme suit :

Note : Les thèmes intégrés ne peuvent pas être modifiés.
  1. Modifiez l' expérience, puis cliquez sur STYLE dans la navigation de gauche. Le thème actuel est affiché en haut du menu de style.
  2. Cliquez sur le thème. Une liste de tous les thèmes sera affichée.
  3. Cliquez sur l'icône de modification () pour le thème que vous souhaitez modifier.
  4. Modifiez les styles selon les besoins. Toutes les modifications apportées aux paramètres de style seront enregistrées automatiquement.
  5. Lorsque vous avez terminé, cliquez sur < Retour aux styles.

Duplication d'un thème

Pour dupliquer un thème, procédez comme suit :

  1. Modifiez l' expérience, puis cliquez sur STYLE dans la navigation de gauche. Le thème actuel est affiché en haut du menu de style.
  2. Cliquez sur le thème. Une liste de tous les thèmes sera affichée.
  3. Cliquez sur l'icône dupliqué () pour le thème que vous souhaitez dupliquer.
  4. Entrez un nouveau nom pour le thème et cliquez sur Enregistrer.
  5. Modifiez les styles selon les besoins. Toutes les modifications apportées aux paramètres de style seront enregistrées automatiquement.
  6. Lorsque vous avez terminé, cliquez sur < Retour aux styles.

Suppression d'un thème

Pour supprimer un thème, procédez comme suit :

Note : Les thèmes intégrés ne peuvent pas être supprimés.
  1. Modifiez l' expérience, puis cliquez sur STYLE dans la navigation de gauche. Le thème actuel est affiché en haut du menu de style.
  2. Cliquez sur le thème. Une liste de tous les thèmes sera affichée.
  3. Cliquez sur l'icône Supprimer () pour le thème que vous souhaitez supprimer. Vous serez invité à confirmer la suppression.
  4. Lorsque vous avez terminé, cliquez sur < Retour aux styles.
