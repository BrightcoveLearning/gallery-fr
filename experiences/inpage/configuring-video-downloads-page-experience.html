Configuration des téléchargements vidéo pour une expérience sur la page

{{ page.description }}

Les expériences dans la galerie peuvent être configurées pour afficher un lien de téléchargement permettant aux spectateurs de télécharger du contenu vidéo. Les expériences en page peuvent être configurées de manière à ce qu'un lien de téléchargement apparaisse pour toutes les vidéos ou vidéos spécifiques en fonction d'une valeur de champ personnalisée. Lorsqu'une vidéo peut être téléchargée, un lien de téléchargement s'affiche à côté de celle-ci. Le format associé de la plus haute qualité disponible est téléchargé.

Note : Les téléchargements vidéo ne sont pas pris en charge sur les appareils iOS.

Pour fournir un lien de téléchargement pour toutes les vidéos d'une expérience, consultez le Configuration des paramètres pour une expérience sur la page Documentation. Si vous souhaitez autoriser le téléchargement sélectif uniquement des vidéos désignées, vous devez effectuer les opérations suivantes :

  1. Créer un champ personnalisé pour indiquer le contenu téléchargeable
  2. Attribuer une valeur appropriée au champ personnalisé
  3. Configurer l'expérience en page pour autoriser les téléchargements

Création d'un champ personnalisé pour indiquer le contenu téléchargeable

Les vidéos pouvant être téléchargées sont indiquées en définissant la valeur d'un champ personnalisé. Commencez par créer un champ personnalisé dans votre compte. Il permettra d'indiquer qu'une vidéo peut être téléchargée.

  1. Connectez-vous à Video Cloud Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMIN, puis sur Champs vidéo. Notez que vous devez être un administrateur de compte pour accéder au menu ADMIN.
  3. Cliquez sur Add Custom Fields.
  4. Créez un champ personnalisé qui permettra d'indiquer si une vidéo est téléchargeable. Le nom et la valeur affectée n'ont aucune importance. Dans l'exemple ci-dessous, un nouveau champ personnalisé downloadable est créé. Le Type est défini sur Liste, avec 2 valeurs, true et false. Les vidéos téléchargeables auront la valeur true dans leurs propriétés.
  5. Cliquez sur Enregistrer les modifications, puis sur Enregistrer.

Affectation de valeurs de champ personnalisées

Toute vidéo téléchargeable doit se voir affecter une valeur dans le champ personnalisé créé.

  1. Ouvrez le module Médias.
  2. Pour signaler une vidéo comme étant téléchargeable, cliquez sur la vidéo pour ouvrir les propriétés vidéo.
  3. Recherchez la section Champs personnalisés et cliquez sur Modifier.
  4. Définissez la valeur du champ personnalisé. Dans cet exemple, la valeur de Downloadable est true.
  5. Cliquez sur Save.
  6. Modifiez les propriétés de la vidéo pour toutes les vidéos supplémentaires si nécessaire.
Note : Utilisez la fonction Quick Edit du module Media pour modifier simultanément les valeurs de champ personnalisées de plusieurs vidéos.

Configuration d'une expérience dans la page pour autoriser les téléchargements

Pour configurer une expérience en page afin d'autoriser les téléchargements vidéo en fonction d'une valeur de champ personnalisée, procédez comme suit.

  1. Ouvrez le module Gallery.
  2. Recherchez votre expérience et cliquez sur Modifier.
  3. Cliquez sur SETTINGS > Video dans la navigation de gauche.
  4. Dans le Téléchargements de vidéos rubrique, sélectionnez le Fournir un lien de téléchargement pour chaque vidéo en fonction d'une valeur de champ personnalisé option.
  5. Entrez le nom du champ personnalisé et la valeur que vous avez créés à l'étape 4 ci-dessus. Dans cet exemple, le champ personnalisé créé a été nommé downloadable et sa valeur est true pour les vidéos téléchargeables.
  6. Cliquez sur Enregistrer, puis republiez l'expérience.
  7. Vérifiez qu'un lien de téléchargement s'affiche pour toute vidéo dont le champ personnalisé téléchargeable est défini sur true.
