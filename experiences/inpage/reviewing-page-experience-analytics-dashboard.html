Examen du tableau de bord d'analyse de l'expérience sur la page

{{ page.description }}

Le tableau de bord analytique fournit une vue d'ensemble de haut niveau du nombre de partages sociaux, des types d'appareils utilisés pour visualiser une expérience et des vidéos les plus visionnées pour une expérience en page. Pour afficher le tableau de bord analytique, modifiez l'expérience et cliquez sur Tableau de > bord ANALYTICS PROJECT dans la navigation de gauche.

Sélection d’une période

Pour spécifier la période du tableau de bord, vous pouvez cliquer sur une date rapide ou utiliser le contrôle Calendrier.

Utilisation d'une date rapide

Une série de dates rapides se trouvent en haut de la page. Les dates suivantes sont disponibles :

  • Aujourd'hui
  • Hier
  • 7 derniers jours
  • Les 30 derniers jours
  • Ce mois-ci - Le mois civil à ce jour - par exemple, si aujourd'hui est le 17 mars, ce rapport s'affiche du 1er mars au 17 mars
  • Le mois dernier - le mois civil précédent - par exemple, si aujourd'hui est le 14 février, ce rapport s'affiche du 1er janvier au 31 janvier.

Cliquez sur une date rapide pour mettre à jour le tableau de bord.

Utilisation du champ Calendrier

Vous pouvez également sélectionner une date de début et de fin en cliquant sur le contrôle calendrier.

Cliquez sur Appliquer pour mettre à jour le tableau de bord.

Affichage des partages sociaux

La section Partage social affiche le nombre de partages sociaux par plateforme pour l'expérience pour la période sélectionnée. Notez que les partages sont comptabilisés lorsque quelqu'un clique sur une icône de partage dans l'expérience, mais il n'y a aucun moyen de déterminer s'il a réellement partagé une vidéo.

Le survol du graphique affichera les données pour la date donnée.

Cliquez sur Voir les détails du partage social pour ouvrir l'analyse d'engagement vidéo dans laquelle vous pouvez afficher le nombre de partages sociaux par vidéo.

Affichage des types de périphériques

La section Type de périphérique affiche les types de périphériques utilisés pour afficher l'expérience pour la période sélectionnée.

Cliquez sur Voir l'affichage vidéo par type d'appareil pour ouvrir l'analyse des performances vidéo et afficher les données de la vue vidéo par type d'appareil.

Visualisation des vidéos les plus tendances

La section Tendances les plus populaires affiche les 5 vidéos les plus tendances en fonction du nombre de vues pour la période sélectionnée.

Cliquez sur Voir les détails de la vue vidéo pour ouvrir l'analyse des performances vidéo et afficher les données de la vue vidéo par vidéo.
