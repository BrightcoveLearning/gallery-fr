Ajout du fichier de configuration à l'expérience

{{ page.description }}

L'expérience d'événement virtuel nécessite l'inclusion d'un fichier de configuration avec le site. Le fichier de configuration inclut les paramètres du site, les options de mise en page, le calendrier des événements, les conférenciers et d'autres contenus qui seront affichés. Après avoir créé une nouvelle expérience d'événement virtuel, l'ajout du fichier de configuration est la première chose à faire

Pour ajouter le fichier de configuration à une expérience d'événement virtuel, procédez comme suit.

  1. Accédez au lien suivant :
    https://virtual-events-experience.s3.amazonaws.com/v1/config.txt
  2. Copiez le contenu dans le Presse-papiers.
  3. Ouvrez le module Gallery.
  4. Modifiez l'expérience.
  5. Cliquez sur APPARENCE ET COMPORTEMENT > En-tête et pied de page dans la navigation de gauche.
  6. Collez le fichier de configuration dans le En-tête HTML personnalisé champ.
  7. Cliquez sur Enregistrer.

Assurez-vous de republier l'expérience pour que les modifications apparaissent.

Mise à jour du policyKey et AssetBaseUrl valeurs

Pour que le compte à rebours s'affiche sur les vidéos simulive, la valeur PolicyKey du fichier de configuration doit être mise à jour avec une clé de stratégie activée pour la recherche. La valeur fournie ne fonctionnera pas pour votre compte. Si vous ne le faites pas, aucun compte à rebours ne sera affiché avec les vidéos. Pour obtenir une clé de stratégie compatible avec la recherche, reportez-vous à l'exemple de code du document développeur : Générez une clé de stratégie. Une fois la nouvelle clé de stratégie créée, mettez à jour la policyKey valeur dans le fichier de configuration.

Le chemin d'accès de la valeur AssetBaseURL est utilisé pour charger les fichiers utilisés par le modèle. Cela peut être laissé seul dans la plupart des cas, sauf si un degré élevé de personnalisation est effectué. La valeur du chemin d'accès qui est là doit être supprimée.

Une fois que les valeurs PolicyKey et AssetBaseURL ont été mises à jour dans le fichier de configuration, republiez l'expérience.
