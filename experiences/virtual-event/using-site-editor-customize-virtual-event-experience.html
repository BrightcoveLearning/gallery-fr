Utilisation de l'éditeur de site pour personnaliser une expérience d'événement virtuel

{{ page.description }}

L'éditeur de site de galerie est utilisé pour personnaliser l'apparence et la convivialité des expériences. Cliquez sur le titre de l'expérience pour ouvrir l'éditeur de site. L'Éditeur de site offre la possibilité de :

  • Ajouter des objets aux pages
  • Ajouter des vidéos à un site
  • Personnaliser le style du site
  • Configurer les appels à l'action
  • Personnaliser l'apparence et le comportement
  • Personnaliser les fonctionnalités du site
  • Personnaliser la vidéo et la lecture
  • Personnaliser la configuration du site
  • Configurer les détails du site

Vous accédez à ces options en cliquant sur les menus dans la navigation de gauche.

Ajout d'objets à une page

Des objets tels que du texte, des images et des liens peuvent être ajoutés aux pages, ce qui vous permet de personnaliser et de personnaliser votre expérience vidéo. Pour savoir comment ajouter des objets aux pages, voir Personnalisation du contenu des pages d'expérience d'événement virtuel.

Ajout de vidéos à un site

Pour ajouter des vidéos à un site, cliquez sur VIDÉOS dans la navigation de gauche. Pour plus d'informations sur l'ajout de vidéos à un site vidéo, consultez Ajout de vidéos à une expérience d'événement virtuel.

Personnalisation du style du site

Pour personnaliser les couleurs et les polices utilisées sur le site, ou pour sélectionner un autre thème, cliquez sur STYLE dans la navigation de gauche. Pour plus d'informations sur la personnalisation du style, consultez Personnalisation du style d'une expérience de portail.

Configuration des appels à l'action

Les appels à l'action permettent d'ajouter du code HTML personnalisé, des images ou des publicités sur une page. Les modèles ne prennent pas tous en charge les appels à l'action. Pour configurer les appels à l'action, cliquez sur APPELS À ACTION dans la navigation de gauche.

Pour plus d'informations sur la configuration des appels à l'action, reportez-vous à la section Configuration des appels à l'action sur une expérience de portail.

Personnalisation de l'apparence et du comportement

Les paramètres d'apparence et de comportement sont utilisés pour configurer un en-tête / pied de page personnalisé, utiliser des fichiers CSS et JavaScript personnalisés et configurer une image ou une vidéo d'arrière-plan. Pour configurer les paramètres d'apparence et de comportement, cliquez sur APPARENCE ET COMPORTEMENT dans la navigation de gauche.

Pour plus d'informations sur les paramètres d'apparence et de comportement, reportez-vous à Personnalisation de l'apparence et du comportement d'une expérience d'événement virtuel.

Personnalisation des fonctionnalités du site

Les paramètres des fonctionnalités du site sont utilisés pour activer la recherche de site, configurer les options de partage social et autoriser les commentaires sur les vidéos. Pour configurer les fonctionnalités du site, cliquez sur FONCTIONNALITÉS DU SITE dans la navigation de gauche.

Pour plus d'informations sur la personnalisation des fonctionnalités du site, consultez Personnalisation des fonctionnalités du site d'une expérience de portail.

Personnalisation des paramètres vidéo et de lecture

Les paramètres vidéo et de lecture permettent de sélectionner un lecteur personnalisé pour l'expérience, de configurer les formulaires d'intégration d'audience et de prospect et de configurer des options de vidéos telles que les téléchargements et les liens associés. Pour personnaliser les paramètres vidéo et de lecture, cliquez sur VIDÉO ET LECTURE dans la navigation de gauche.

Pour plus d'informations sur la configuration des paramètres vidéo et de lecture, consultez Personnalisation des paramètres vidéo et de lecture d'une expérience de portail.

Personnalisation de la configuration du site

Les paramètres de configuration du site permettent d'héberger le site dans un domaine personnalisé, de le sécuriser à l'aide de SSL ou de restreindre l'accès au site. Pour configurer la configuration du site, cliquez sur CONFIGURATION DU SITE dans la navigation de gauche.

Pour plus d'informations sur les paramètres de configuration du site, consultez Personnalisation de la configuration de site d'une expérience de portail.

Configuration des détails du site

Les paramètres d'aperçu permettent de modifier le nom, la description et la langue du site. Pour configurer les détails du site, cliquez sur APERÇU dans la navigation de gauche.

Pour plus d'informations sur la configuration des détails du site, reportez-vous à la section Configuration des détails d'une expérience de portail.
