Création d'un fichier d'agenda d'événement

{{ page.description }}

En règle générale, l'agenda de l'événement est initialement créé dans un outil tiers, par exemple, Microsoft Excel ou Google Sheets. Le module Agenda nécessite qu'un fichier d'agenda externe .csv soit chargé pour créer initialement l'agenda de l'événement. Une fois le fichier agenda téléchargé, il peut être modifié à l'aide du module Agenda.

Note : Un exemple de fichier de modèle d'agenda peut être téléchargé ici.
Note : Il n'y a aucun moyen de créer un agenda dans le module Agenda. Vous devez charger un fichier .csv pour remplir initialement l'agenda de l'événement.

Exigences de nommage des colonnes

Lors de la création du fichier d'agenda initial, vous pouvez télécharger un exemple de modèle d'agenda et le modifier ou utiliser un fichier d'agenda existant que vous avez peut-être créé. Le module Agenda recherche l'existence de noms de colonnes spécifiques. Ces colonnes seront mappées aux champs appropriés dans le module agenda. Les autres colonnes pouvant se trouver dans votre fichier agenda seront ignorées par le module Agenda. Afin de télécharger un fichier d'agenda, les colonnes suivantes sont requises :

  • name- Nom de la session
  • start_date- Date de début de la session (format MM/JJ/AAAA)
  • start_time- Heure de début de la session
  • timezone- Fuseau horaire pour start_time (en utilisant l'ID de zone des fuseaux horaires IANA)
  • duration- Durée de la session en minutes
  • track- Valeur qui identifie la piste (c'est-à-dire technique, commerciale, marketing) ; un nom de piste ALL indique que la session couvre toutes les pistes (c.-à-d. déjeuner, pauses)

Les colonnes suivantes sont facultatives, mais si elles existent dans votre fichier d'agenda, elles seront également importées dans le module Agenda :

  • description- Description de la session
  • speaker1_first- Prénom de l'orateur
  • speaker1_last- Nom de famille de l'intervenant
  • speaker1_email- Adresse e-mail du haut-parleur (utilisée pour identifier de manière unique un haut-parleur)
  • speaker2_first- Prénom du haut-parleur 2
  • speaker2_last- Nom de famille de l'orateur 2
  • speaker2_email- Adresse e-mail du haut-parleur 2 (utilisée pour identifier de manière unique un haut-parleur)
  • speaker3_first- Prénom de l'orateur 3
  • speaker3_last- Nom de famille Speaker 3
  • speaker3_email- Adresse e-mail du haut-parleur 3 (utilisée pour identifier de manière unique un haut-parleur)
  • session_type- Type de session (session, keynote, réunion, autre)
  • video _id- ID vidéo (pour session, keynote et autres types de session)
  • url- URL de la réunion (pour les types de session de réunion)
  • related _label- Étiquette du lien associé qui apparaîtra dans le module Agenda
  • related_link- Lien URL vers un document de session, des diapositives, etc.
Note : Le téléchargement de l'agenda prend en charge jusqu'à trois haut-parleurs, des haut-parleurs supplémentaires peuvent être ajoutés à une session à l'aide du module Agenda.

Lorsque vous êtes prêt à importer votre agenda dans le module Agenda, exportez-le sous forme de fichier .csv. Le module Agenda peut uniquement importer un fichier .csv. Pour plus d'informations sur la façon de télécharger votre fichier d'agenda dans le module Agenda, voir Téléchargement et personnalisation de l'agenda de l'événement.
