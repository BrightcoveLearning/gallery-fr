    Intégration de Pigeonhole à une expérience d'événement virtuel

    Dans cette rubrique, vous apprendrez comment intégrer Pigeonhole à une expérience d'événement virtuel.

    Pigeonhole peut être utilisé pour ajouter facilement des questions et réponses en direct à une ou plusieurs vidéos dans une expérience d'événement virtuel. Généralement, un nouveau pigeonhole est créé pour l'événement en direct, puis des sessions peuvent être créées pour chaque vidéo à laquelle vous souhaitez ajouter des questions et réponses. Une fois les sessions créées, le code d'intégration iframe est associé à chaque vidéo à l'aide d'un champ personnalisé Video Cloud. Le code intégré entraînera l'affichage du module Q&A Pigeonhole à côté du lecteur vidéo pour les vidéos choisies.

    Création d'un nouvel événement Pigeonhole

    Vous devez créer un nouveau Pigeonhole pour votre événement en direct, puis ajouter des sessions pour chaque vidéo qui aura Q&R activé.

    1. Connectez-vous à votre compte Pigeonhole.
    2. Cliquez + Ajouter Pigeonhole et cliquez sur votre compte à utiliser.
    3. Entrez un nom d'événement et un code d'accès.
    4. Saisissez les dates de début et de fin de l'événement. Ces dates contrôlent combien de temps le Pigeonhole sera actif.
    5. Sélectionnez une option de fuseau horaire et de marque personnalisée (le cas échéant).
    6. Cliquez sur Continue.
    7. Maintenant que l'événement est créé, le point Agenda sera sélectionné dans la navigation de gauche. Cliquez sur Ajouter une session , puis sur Créer un nouveau.
    8. Sélectionnez un type de session de questions-réponses.
    9. Entrez un nom de session.
    10. Saisissez les dates et heures de début et de fin .
    11. Configurez les paramètres Q&R selon les besoins.
    12. Cliquez sur Ajouter des questions et réponses. Confirmez que la session a été créée.
    13. Cliquez sur Plus de paramètres dans la navigation de gauche.
    14. Cliquez sur l'onglet Intégrations en haut de la page.
    15. Faites défiler la page jusqu'à la section Autres .
    16. Recherchez Brightcove Gallery Incorporer et cliquez sur Intégrer.
    17. ( Facultatif) Si votre expérience Gallery est sécurisée à l'aide de l'authentification unique, sélectionnez Autoriser la connexion automatique des participants à l'aide des attributs SAML. Cela récupère le nom et l'adresse e-mail de l'utilisateur et les connectera automatiquement sur Pigeonhole Live lorsque le widget Pigeonhole Live se charge. Si cette option n'est pas cochée, les utilisateurs apparaîtront comme anonymes à l'intérieur de Pigeonhole. Cliquez sur Save.
    18. Dans la boîte de dialogue Incorporer Brightcove Gallery, sélectionnez votre session Q&R dans la liste déroulante.
    19. Cliquez sur Obtenir le code d'intégration.
    20. Cliquez sur le code d'intégration et copiez-le dans le presse-papiers. Il s'agit du code qui sera intégré à l'expérience Galerie.
    21. Cliquez sur Close (Fermer).

    Création d'un champ personnalisé

    Pour que le modèle d'événement virtuel sache quel trou de pigeonhole afficher pour une vidéo (le cas échéant), un champ personnalisé est utilisé. Pour créer un champ personnalisé, procédez comme suit.

    1. Connectez-vous à Video Cloud Studio.
    2. Dans l'en-tête de navigation, cliquez sur ADMIN et alors Champs vidéo. Notez que vous devez être un administrateur de compte pour accéder au menu ADMIN.
    3. Créez un nouveau champ personnalisé nommé Chat Incorporer.
    4. Confirmez que le nom interne est chat_embed.
    5. Définissez le Type sur Texte.
    6. Enregistrez le champ personnalisé.

    Pour plus d'informations sur la création de champs personnalisés dans Video Cloud, voir Création de champs de métadonnées personnalisés.

    Ajout de Q&R à une vidéo

    Pour qu'un pigeonhole apparaisse en regard d'une vidéo dans le modèle Événement virtuel, procédez comme suit.

    1. Ouvrez le module Médias.
    2. Cliquez sur une vidéo pour ouvrir les propriétés de la vidéo.
    3. Recherchez la section CHAMPS PERSONNALISÉS et cliquez sur Modifier.
    4. Collez le code d'intégration Pigeonhole dans le champ personnalisé Chat Intégrer .
    5. Cliquez sur Save.
