Personnalisation du contenu des pages d'expérience d'événement virtuel

{{ page.description }}

Des objets tels que du texte, des images et des liens peuvent être ajoutés aux pages Virtual Event Experience, ce qui vous permet de personnaliser et de personnaliser votre expérience vidéo.

Ajout d'objets à des pages

À l'aide de l'éditeur de site, vous pouvez ajouter des objets tels que des images, du texte, des liens et un message de bienvenue à une page. Commencez par modifier une expérience, puis cliquez sur PAGES dans la navigation de gauche. Le modèle de page affichera des icônes plus () indiquant que les objets qui peuvent être ajoutés à la page.

Passez le curseur sur l'icône plus et cliquez dessus pour ajouter l'objet approprié.

Le modèle de page peut également contenir des objets texte qui font partie du modèle. En passant le curseur sur eux, un bouton Modifier s'affiche. Cliquez sur l'objet ou sur le bouton Modifier pour modifier l'objet.

Ajout d'un logo à l'en-tête de page

Le logo peut être du texte ou un fichier image. Pour les images, les images JPEG et PNG sont prises en charge. Pour ajouter un logo à l'en-tête de la page, cliquez sur + Ajouter un logo dans l'en-tête de la page.

Ajout d'un logo textuel

  1. Utilisez l'éditeur de texte sur le Texte onglet pour saisir et mettre en forme le texte de l'en-tête.
  2. Cliquez sur Save.

Ajout d'un logo d'image

  1. Clique le Image languette. La boîte de dialogue du navigateur d'images s'ouvre.
  2. Pour spécifier une image, vous pouvez :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  3. Cliquez sur Save une fois l'opération terminée.

Ajout de liens de menu

Des liens de menu peuvent être ajoutés à l'en-tête de la page pour permettre aux utilisateurs de naviguer vers d'autres parties de votre site. Pour ajouter des liens de menu à l'en-tête de page :

  1. Cliquez sur +Ajouter un menu dans l'en-tête ou cliquez sur MENU dans l'en-tête, puis cliquez sur l'icône plus (). L'affichage changera en fonction de la largeur de la fenêtre de votre navigateur.
  2. Entrez le texte et le lien. Cochez New Window pour que le lien s'ouvre dans une nouvelle fenêtre de navigateur.
  3. Cliquez sur Save.

Des liens de navigation supplémentaires peuvent être saisis en cliquant sur dans l'en-tête. Pour ajouter des liens de pied de page supplémentaires, cliquez sur + Ajouter des liens de menu. Pour modifier un lien, cliquez dessus.

Ajouter un message de bienvenue

Un message de bienvenue peut être ajouté pour accueillir les visiteurs sur le site et peut contenir du texte et une image. Le HTML personnalisé peut être utilisé pour des messages de bienvenue plus avancés. Pour ajouter un message de bienvenue, cliquez sur + Message de bienvenue.

Ajouter un SMS

Les Texte peut être utilisé pour ajouter du texte et/ou une image. Sur la page, l'image apparaîtra sous le texte.

  1. Saisissez votre texte. La barre de mise en forme peut être utilisée pour formater le texte.
  2. Pour ajouter une image, vous pouvez :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  3. Cliquez sur Save.

Utiliser du HTML personnalisé

Le HTML personnalisé est utile lorsque vous souhaitez avoir une mise en page plus avancée qui ne peut pas être créée à l'aide de l'éditeur WYSIWYG sur le Texte languette.

  1. Clique le HTML personnalisé languette.
  2. Collez le code HTML personnalisé pour le message de bienvenue.
  3. Cliquez sur Save.

Ajout de liens de navigation au pied de page

Pour ajouter des liens de navigation au pied de page :

  1. Cliquez sur ESPACE RÉSERVÉ, entrez une valeur pour l'en-tête et cliquez sur Enregistrer.
  2. Cliquez sur +Ajouter des liens de menu et entrez le Texte , Relier URL et Ordre de la liste. Vous pouvez également indiquer l'ordre des liens en définissant la valeur List Order lors de la création du lien.

Vous pouvez entrer des liens de navigation supplémentaires en cliquant sur +Ajouter des liens de menu.

Suppression des objets d'une page

Le texte, les images et les liens peuvent être supprimés de la page de deux manières.

Pour supprimer le texte ou les images de l'en-tête de page

  1. Survolez l'objet et cliquez sur l'icône Supprimer.

Pour supprimer des liens de navigation de l'en-tête de page

  1. Cliquez sur le lien dans l'en-tête de navigation.
  2. Cliquez sur Delete.

Pour supprimer les liens de navigation du pied de page

  1. Cliquez sur le lien dans le pied de page.
  2. Cliquez sur Delete.

Pour supprimer le message de bienvenue ou tout texte

  1. Cliquez sur l'objet texte.
  2. Désélectionnez le titre du texte et cliquez sur Save.
