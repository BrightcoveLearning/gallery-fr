Personnalisation des commanditaires de l'événement

{{ page.description }}

Les images des commanditaires peuvent être ajoutées à la page d'accueil de l'événement virtuel.

Les informations du sponsor sont personnalisées en modifiant l' sponsors objet dans le fichier de configuration.

  • name- Nom du sponsor
  • image_url- Logo du sponsor, les images seront redimensionnées à 100 px de large
  • link_url- URL de destination de l'image

L' sponsors objet du fichier de configuration dispose également de paramètres permettant de contrôler l'emplacement dans la page d'accueil des informations du sponsor. Pour plus d'informations sur le positionnement de la section Sponsors, voir Positionnement du contenu sur la page d'accueil.

Suppression de l'opacité et dimensionnement des images de logo

Par défaut, les images de logo sont stylisées avec un niveau d'opacité de 80%. Le style CSS suivant peut être utilisé pour définir l'opacité sur 100%. Les largeur maximale La propriété peut être utilisée pour modifier la taille des images.


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
Modifiez l'expérience et ajoutez le CSS en haut de la En-tête HTML personnalisé . champ.
