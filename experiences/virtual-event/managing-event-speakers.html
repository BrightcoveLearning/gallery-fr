Gestion des conférenciers d'événements

{{ page.description }}

Des images de conférencier et une courte biographie peuvent être ajoutées à la page d'accueil de l'événement virtuel.

Les intervenants de l'événement sont gérés à l'aide du module Agenda. Tous les orateurs de l'événement qui ont été inclus dans le téléchargement du fichier d'agenda de l'événement apparaîtront sous le Conférenciers d'événements référencement.

Note : Un maximum de 8 haut-parleurs peuvent être affichés sur la page d'accueil.
Note : Un maximum de 4 haut-parleurs peuvent être affichés d'affilée.

Pour accéder au module Agenda, procédez comme suit :

  1. Cliquez sur une expérience d'événement virtuel pour ouvrir l'éditeur de site de la galerie.
  2. Cliquez sur CARACTÉRISTIQUES DU SITE> Ordre du jour.
  3. Dans la navigation de gauche, cliquez sur Haut-parleurs. Par défaut, le Conférenciers d'événements le lien est sélectionné, affichant tous les intervenants pour l'événement en cours.

En cliquant sur le Tous les haut-parleurs Le lien affichera tous les conférenciers qui ont été affectés à toutes les expériences d'événement virtuel. La liste des orateurs peut être triée en cliquant sur un nom d'en-tête de colonne.

Création d'un haut-parleur

Pour créer un nouveau haut-parleur, procédez comme suit.

  1. Cliquez sur Créer un haut-parleur.
  2. Entrez dans le haut-parleur Prénom et Nom de famille.
  3. Entrez dans le haut-parleur E-mail.
  4. Entrez dans le haut-parleur Titre et Société.
  5. Entrez dans le haut-parleur Biographie.
  6. Pour le Photo du visage image, cliquez Télécharger une image ou l'icône de téléchargement et parcourez et sélectionnez une image de haut-parleur.
  7. Cliquez sur Save.

Modification d'un haut-parleur

Pour modifier un nouveau haut-parleur, procédez comme suit.

  1. Cliquez sur un haut-parleur pour ouvrir le Modifier le haut-parleur panneau.
  2. Apportez les modifications nécessaires.
  3. Cliquez sur Save.

Supprimer un haut-parleur

Pour supprimer un haut-parleur, procédez comme suit.

  1. Cliquez sur un haut-parleur pour ouvrir le Modifier le haut-parleur panneau.
  2. Cliquez sur Supprimer le haut-parleur.
  3. Cliquez sur Supprimer l'enceinte pour confirmer la suppression.

Recherche de conférenciers

Les Rechercher des haut-parleurs en haut de la page peut être utilisé pour rechercher des orateurs. Clique le Conférenciers d'événements ou Tous les haut-parleurs lien pour indiquer les locuteurs à rechercher. Les champs prénom, nom, titre et société de l'intervenant seront recherchés.
