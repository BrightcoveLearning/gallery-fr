Utilisation du contenu statique

{{ page.description }}

Les objets de contenu statique sont utilisés pour injecter du HTML personnalisé dans le modèle. Cet exemple de bannière de héros a été construit à l'aide d'une combinaison de texte et d'images et injecté en haut de la page.

Le contenu statique peut être n'importe quel HTML et peut être injecté n'importe où dans le modèle. Les objets de contenu statique sont ajoutés à l' static_content objet dans le fichier de configuration. Pour plus d'informations sur la façon de positionner le contenu statique sur la page, voir Positionnement du contenu sur la page d'accueil.
