Personnaliser l'apparence et le comportement d'une expérience d'événement virtuel

{{ page.description }}

Pour configurer les paramètres Apparence et Comportement, modifiez l'expérience et cliquez sur APPARENCE ET COMPORTEMENT dans la navigation de gauche. Les paramètres suivants sont disponibles :

  • En-tête et pied de page : configuration d'un en-tête/pied de page personnalisé pour l'expérience
  • Personnalisé - Utiliser des fichiers CSS et JavaScript personnalisés
  • Image et vidéo d'arrière-plan : configuration d'une image ou d'une vidéo d'arrière-plan

Configuration d'en-têtes/pieds de page HTML

Gallery offre la possibilité d'utiliser un en-tête et/ou pied de page personnalisé pour le site. Cela offre la possibilité de personnaliser une expérience d'événement virtuel afin qu'elle corresponde plus étroitement à l'apparence de vos sites d'entreprise. Un en-tête/pied de page personnalisé peut également être configuré au niveau du compte. Pour plus d'informations sur la configuration d'un en-tête/pied de page personnalisé au niveau du compte, reportez-vous à la section Configuration des paramètres de la galerie.

Pour configurer une expérience d'événement virtuel afin d'utiliser un en-tête ou un pied de page personnalisé, procédez comme suit.

  1. Concevez l'en-tête/pied de page à utiliser pour votre site. Il peut contenir n'importe quel code HTML, JavaScript ou CSS valide.
  2. Cliquez sur APPARENCE ET COMPORTEMENT > En-tête et pied de page dans la navigation de gauche.
  3. Collez le code HTML de l'en-tête dans le champ Custom Header HTML.
  4. Sélectionnez l'option Inclure l'en-tête par défaut sous l'en-tête personnalisé pour afficher l'en-tête de modèle par défaut sous votre en-tête personnalisé.
  5. Collez le code HTML du pied de page dans le champ Custom Footer HTML.
  6. Sélectionnez l'option Inclure le pied de page par défaut sous le pied de page personnalisé pour afficher le pied de page par défaut sous votre pied de page personnalisé.
  7. Cliquez sur Enregistrer.

Remarques sur l'utilisation d'en-têtes et de pieds de page personnalisés

  • Gallery ne fait aucune validation sur le HTML fourni pour l'en-tête/pied de page
  • Il n'y a pas de limite de caractères sur l'en-tête et le pied de page HTML.
  • Les fichiers CSS et JavaScript externes sont pris en charge.
  • Le code doit suivre les principes de conception réactive afin qu'il puisse évoluer correctement entre les appareils
  • Des conflits CSS ont été constatés en cas d'utilisation de Bootstrap (dont Gallery se sert également). Vérifiez et corrigez tout conflit éventuel.

Configuration des paramètres personnalisés

Les paramètres personnalisés permettent de personnaliser l'aspect du site. Pour configurer les paramètres personnalisés, cliquez sur APPARENCE ET COMPORTEMENT > Personnalisé dans la navigation de gauche.

Les paramètres suivants sont disponibles :

  • URL CSS personnalisée : permet de spécifier l'emplacement d'un fichier .css personnalisé pour personnaliser le style. Cliquez sur le lien Afficher CSS pour afficher le fichier CSS actuellement utilisé pour styliser le site.
  • URL JavaScript personnalisée : permet de spécifier l'emplacement d'un fichier .js personnalisé afin de personnaliser le style et/ou le comportement du site.
  • URL Favicon personnalisée : permet de spécifier l'emplacement du favicon à utiliser pour le site. Les navigateurs qui fournissent la prise en charge de favicon affichent généralement le favicon dans l'onglet du navigateur de la page et à côté du nom de la page dans une liste de signets.
  • Icône d'application mobile personnalisée - Permet de sélectionner une image qui sera utilisée comme icône d'écran d'accueil lorsque le site est enregistré sur l'écran d'accueil de votre appareil. Brightcove vous recommande de télécharger une haute résolution (nous recommandons 1024 px x 1024 px), une image carrée et Gallery va mettre l'image à l'échelle et utiliser une version la mieux adaptée à votre appareil.
  • En-têtes de réponse personnalisés : entrez les en-têtes HTTP qui seront envoyés en réponse à des demandes adressées au portail. Par exemple, des en-têtes peuvent être ajoutés pour ajouter une stratégie de sécurité de contenu (CSP) ou HTTP-Strict-Transport-Security (HSTS).

Configurer une image ou une vidéo d'arrière-plan

La galerie offre la possibilité d'afficher une image ou une vidéo d'arrière-plan personnalisée pour le site. Pour configurer une image ou une vidéo d'arrière-plan, cliquez sur APPARENCE ET COMPORTEMENT> Image d'arrière-plan et vidéo dans la navigation de gauche.

Ajout d'une image d'arrière-plan

  1. Pour spécifier une image, vous pouvez :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  2. Cliquez sur Enregistrer.

Cliquez sur Supprimer pour supprimer l'image.

Ajouter une vidéo d'arrière-plan

Une vidéo Video Cloud peut également être utilisée comme vidéo d'arrière-plan. Collez un ID vidéo Video Cloud dans le ID vidéo champ et cliquez Sauvegarder.

Note : Si l'ID vidéo n'est pas valide, l'image d'arrière-plan téléchargée ou l'image d'arrière-plan par défaut s'affiche.
