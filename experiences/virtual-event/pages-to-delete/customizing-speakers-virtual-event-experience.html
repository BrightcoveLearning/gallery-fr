Personnalisation des conférenciers

Dans cette rubrique, vous allez apprendre à personnaliser les conférenciers qui apparaissent sur la page d'accueil Virtual Event Experience.

Des images de conférencier et une courte biographie peuvent être ajoutées à la page d'accueil de l'événement virtuel.

Les informations de haut-parleur sont personnalisées en modifiant l' speakers objet dans le fichier de configuration.

  • name- Nom de l'orateur
  • description- Informations sur les haut-parleurs qui s'affichent sous leur image. Ceci est limité à environ 3 lignes de 15 mots.
  • image_url- Image du haut-parleur. Recommandez une largeur maximale de 200px. Les images ne seront pas redimensionnées
  • link_url- (facultatif) URL de destination de l'image (peut être liée à la page LinkedIn du haut-parleur par exemple)

Par défaut, le modèle CSS configure les images de haut-parleur avec un border-radius de 50%, ce qui rendra les images circulaires.

L' speakers objet du fichier de configuration dispose également de paramètres permettant de contrôler l'emplacement dans la page d'accueil des informations sur le haut-parleur. Pour plus d'informations sur le positionnement de la section haut-parleurs, voir Positionnement du contenu sur la page d'accueil.
