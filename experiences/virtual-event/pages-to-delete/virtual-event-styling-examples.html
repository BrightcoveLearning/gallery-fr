Exemples de style d'événement virtuel

Cette rubrique fournit un exemple de code qui peut être utilisé pour personnaliser le modèle d'événement virtuel.

Icônes de partage social

Par défaut, si les paramètres de partage vidéo sont activés pour l'expérience, ils n'apparaîtront pas à côté des vidéos sur les pages détaillées de la vidéo. Le CSS suivant affichera les icônes sociales sur les pages détaillées de la vidéo.


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}
