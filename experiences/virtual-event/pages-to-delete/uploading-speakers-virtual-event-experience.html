Téléchargement d'un fichier de haut-parleurs

Dans cette rubrique, vous apprendrez à télécharger une liste externe de conférenciers pour remplir initialement la liste des conférenciers pour une expérience d'événement virtuel.

Vous avez peut-être créé une liste de conférenciers dans Microsoft Excel ou dans une autre application de feuille de calcul. Le module Agenda prend en charge le téléchargement d'un fichier de haut-parleurs .csv externe afin que tous les haut-parleurs n'aient pas besoin d'être saisis manuellement dans le module Agenda.

Note : Brightcove recommande de télécharger le fichier des haut-parleurs avant de télécharger un fichier d'agenda externe.

Exigences relatives aux fichiers d'importation

Pour importer votre liste de conférenciers dans le module agenda, la liste doit avoir été créée dans un tableur (par exemple, Microsoft Excel, Apple Numbers ou Google Sheets. Ces applications permettent d'exporter les données vers un fichier .csv. Le module Agenda peut uniquement importer un fichier .csv.

Le module agenda recherche l'existence de certains noms de colonnes. Ces colonnes seront mappées aux champs appropriés dans le module agenda. Au minimum, les champs suivants doivent exister dans le fichier .csv de vos haut-parleurs :

  • last-name- Prénom de l'orateur
  • first-name- Nom de famille de l'intervenant
  • email- Adresse e-mail du conférencier

Les colonnes suivantes sont facultatives, mais si elles existent dans votre fichier .csv, elles seront également importées dans le module agenda :

  • company- Entreprise de haut-parleurs
  • title- Titre du conférencier
  • bio- Bio des haut-parleurs
  • social-url- URL sociale du haut-parleur (c'est-à-dire l'URL LinkedIn)
  • homepage-url- Site Web personnel du conférencier
