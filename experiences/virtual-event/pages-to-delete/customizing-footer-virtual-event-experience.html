Personnalisation du pied de page d'une expérience d'événement virtuel

Dans cette rubrique, vous apprendrez comment personnaliser le pied de page des pages Expérience d'événement virtuel.

Le pied de page apparaît en bas de toutes les pages. Le pied de page est construit à partir des valeurs du fichier de configuration. Chaque élément de pied de page dans le fichier de configuration sera associé :

  • URL du lien
  • Étiquette
  • Groupe

Les groupes sont utilisés pour contrôler l'emplacement de l'étiquette/lien dans le pied de page. Il y a quatre groupes disponibles :

  • Groupe 0 - Utilisé pour les légendes HTML
  • Groupe 1 - Utilisé pour le texte légal et les informations sur les droits d'auteur
  • Groupes 2 et 3 - Utilisé pour les listes de liens (recommandé d'utiliser un maximum de 5 liens)

Les icônes sociales qui apparaissent dans le pied de page peuvent être modifiées/supprimées en modifiant l' social objet dans le fichier de configuration. Pour plus d'informations, voir Configuration des icônes sociales pour une expérience d'événement virtuel.

Ajout de liens au pied de page

Les liens de pied de page sont configurés en modifiant l' footer_links objet dans le fichier de configuration.

  • link_url- URL de destination
  • label- L'étiquette doit apparaître dans le pied de page
  • group- Regrouper le lien apparaît dans
