Vue d'ensemble du modèle d'événement virtuel

{{ page.description }}

En-tête

L'en-tête apparaîtra en haut de toutes les pages. L'en-tête est construit à partir des valeurs du fichier de configuration. L'en-tête est composé des composants suivants :

  • Logo en-tête
  • Liens
  • Rechercher

Pour plus d'informations sur la personnalisation de l'en-tête, voir Personnalisation du contenu des pages d'expérience d'événement virtuel.

Message de bienvenue

Le message de bienvenue peut être utilisé pour accueillir les visiteurs sur le site et peut contenir du texte et une image. Le HTML personnalisé peut être utilisé pour des messages plus avancés. Le message de bienvenue dans cet exemple a été construit en utilisant une combinaison de texte et d'images.

Pour plus d'informations sur l'ajout d'un message de bienvenue, voir Personnalisation du contenu des pages d'expérience d'événement virtuel.

Collection en vedette

La collection en vedette est la première collection vidéo qui apparaît dans le modèle. La collection en vedette est généralement utilisée pour présenter des vidéos en direct. Les flèches de navigation sont disponibles afin que les utilisateurs puissent naviguer dans les vidéos en vedette. Un widget Q&R peut également être associé à chaque vidéo. Pour plus d'informations sur la création de collections et l'ajout de vidéos, consultez Ajout de vidéos à une expérience d'événement virtuel.

Note : La collection Featured prend en charge un maximum de 24 vidéos.

Collecte primaire

La collection principale est la deuxième collection vidéo qui apparaît dans le modèle. Des flèches de navigation sont disponibles afin que les utilisateurs puissent naviguer dans les vidéos. Pour plus d'informations sur la création de collections et l'ajout de vidéos, consultez Ajout de vidéos à une expérience d'événement virtuel.

Note : La collection Primary prend en charge un maximum de 24 vidéos.

Ordre du jour

L'ordre du jour est généralement utilisé pour répertorier le calendrier des événements en direct. L'ordre du jour affiche le nom de la session, le ou les orateurs ainsi que la date et l'heure de la session. Une minuterie de compte à rebours apparaît à côté de la session et un bouton Affichage apparaît lorsque la session est en cours. Pour plus d'informations sur la création de l'agenda de l'événement, voir Gestion de l'agenda de l'événement.

Haut-parleurs

La section Conférenciers est automatiquement renseignée avec tous les conférenciers de l'événement. Tous les conférenciers de la session apparaîtront dans la section Conférenciers. La section Haut-parleurs affichera :

  • Image du haut-parleur
  • Prénom
  • Titre
  • Société
  • Court bio

Pour plus d'informations sur la gestion des conférenciers de l'événement, voir Gestion des conférenciers d'événements.

Collecte secondaire

La collection secondaire est la troisième collection vidéo qui apparaît dans le modèle. Des flèches de navigation sont disponibles afin que les utilisateurs puissent naviguer dans les vidéos. Pour plus d'informations sur la création de collections et l'ajout de vidéos, consultez Ajout de vidéos à une expérience d'événement virtuel.

Note : La collection Secondary prend en charge un maximum de 24 vidéos.

Pistes

La zone des pistes est une série de collections qui est généralement utilisée pour afficher des vidéos VOD. Cliquez sur une piste pour afficher les vidéos de cette piste. Pour plus d'informations sur la création de collections et l'ajout de vidéos, consultez Ajout de vidéos à une expérience d'événement virtuel.

Contenu statique

La grille en savoir plus est créée à l'aide d'un objet texte statique dans le fichier de configuration. Les objets texte statiques sont utilisés pour injecter du HTML personnalisé dans le modèle.

Le contenu statique peut être n'importe quel HTML et peut être injecté n'importe où dans le modèle. Pour plus d'informations sur l'utilisation d'objets de contenu statique, voir Utilisation du contenu statique.

Sponsors

La section commanditaires peut être utilisée pour mettre en évidence les commanditaires de l'événement. Pour chaque sponsor, vous pouvez afficher :

  • Logo du sponsor
  • Lien vers la page Web du sponsor

Pour plus d'informations sur la personnalisation de la section sponsors, voir Personnalisation des sponsors de l'événement.

Pied de page

Le pied de page apparaît en bas de toutes les pages. Le pied de page est construit à partir des valeurs du fichier de configuration. Le pied de page est composé des composants suivants :

  • Zone de légende principale
  • Texte juridique et droit d'auteur
  • Trois colonnes de liens
  • Icônes sociales
