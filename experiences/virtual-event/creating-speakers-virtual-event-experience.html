Création d'un fichier d'intervenants d'événement

{{ page.description }}

Vous avez peut-être créé une liste d'intervenants dans un outil tiers, par exemple Microsoft Excel ou Google Sheets. Le module Agenda permet de télécharger un fichier .csv de conférenciers d'événements. Une fois le fichier des intervenants téléchargé, les informations sur les intervenants peuvent être modifiées à l'aide du module Agenda.

Note : Un exemple de fichier de modèle de haut-parleurs peut être téléchargé ici.

Exigences de nommage des colonnes

Lors de la création du fichier de haut-parleurs, vous pouvez télécharger un exemple de modèle de haut-parleurs et le modifier ou utiliser un fichier de haut-parleurs existant que vous avez peut-être créé. Le module Agenda recherche l'existence de noms de colonnes spécifiques. Ces colonnes seront mappées aux champs de haut-parleur appropriés. Les autres colonnes pouvant se trouver dans votre fichier de haut-parleur seront ignorées lors de l'importation. Pour télécharger un fichier de conférencier, les colonnes suivantes sont requises :

  • first_name- Prénom de l'orateur
  • last_name- Nom de famille de l'intervenant

Les colonnes suivantes sont facultatives, mais si elles existent dans votre fichier de haut-parleurs, elles seront également importées :

  • id- Ne peut pas être défini par l'utilisateur. Il s'agit de l'ID qui serait renseigné si l'utilisateur exportait une liste de conférenciers au format CSV. Ceci est utilisé pour mettre à jour les entrées existantes dans la base de données.
  • email- Adresse e-mail du conférencier
  • title- Titre du conférencier
  • company- Entreprise de haut-parleurs
  • bio- Bio des haut-parleurs
  • image- Image du haut-parleur

Lorsque vous êtes prêt à importer vos enceintes dans le module Agenda, exportez-les sous forme de fichier .csv. Le module Agenda peut uniquement importer un fichier .csv.

Note : Pour plus d'informations sur la façon de charger votre fichier d'agenda dans le module Agenda, reportez-vous à la section Chargement et personnalisation de l'agenda des événements.
