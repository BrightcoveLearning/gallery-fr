Gestion de l'agenda de l'événement

{{ page.description }}

Le module Agenda prend en charge le téléchargement d'un fichier d'agenda .csv externe, de sorte que tous les points de l'ordre du jour ne doivent pas être saisis manuellement dans le module Agenda. Le téléchargement d'un fichier d'agenda est la façon dont l'agenda de l'événement est initialement créé. Une fois qu'un agenda est téléchargé, il peut être modifié à l'aide du module Agenda.

Note : Pour plus d'informations sur la création d'un fichier d'agenda à télécharger, reportez-vous à la section Création d'un fichier d'agenda d'événements.

Téléchargement d'un fichier d'agenda

Avant d'importer votre agenda existant dans le module Agenda, le fichier agenda doit être exporté vers un fichier .csv. Le module Agenda peut uniquement importer un fichier .csv.

À l'aide de l'éditeur de site Gallery, vous pouvez télécharger et configurer l'agenda de l'événement. Pour télécharger un fichier d'agenda dans une expérience d'événement virtuel, procédez comme suit.

  1. Cliquez sur le nom de l'expérience pour ouvrir l'éditeur de site Gallery.
  2. Dans la navigation de gauche, cliquez sur CARACTÉRISTIQUES DU SITE> Ordre du jour.
  3. Cliquez sur Télécharger l'ordre du jour.
  4. Parcourez et sélectionnez le fichier d'agenda à télécharger.
  5. Confirmez que l'ordre du jour est répertorié sur le Mon agenda page.

Si des erreurs surviennent lors de l'import de l'agenda, un message s'affichera. Après l'importation, le Statut sera DÉSACTIVÉ. Les agendas dont le statut est DÉSACTIVÉ ne seront pas affichés dans l'expérience d'événement virtuel.

Modification du titre de l'ordre du jour

Le titre de l'agenda sera affiché sur la page d'accueil de l'expérience au-dessus de l'agenda. Par défaut, le titre de l'agenda sera défini sur le nom du fichier d'agenda qui a été importé.

Pour modifier le titre de l'ordre du jour, cliquez sur l'icône de modification, saisissez un titre, puis appuyez sur Entrer.

Consulter l'ordre du jour

Une fois qu'un fichier d'agenda a été téléchargé, il apparaîtra sur le Mon agenda page. Pour consulter les détails de l'ordre du jour, cliquez sur le Nom de l'événement. L'agenda s'ouvrira en affichant les détails.

Cliquez sur une date sur le côté gauche de la page pour afficher l'ordre du jour de cette journée. Les pistes sont affichées dans les en-têtes de colonne en haut de la page.

Ajout d'une session

Pour ajouter une nouvelle session, procédez comme suit.

  1. Cliquez sur l'agenda pour ouvrir l'éditeur d'agenda.
  2. Cliquez sur un créneau horaire libre.
  3. Saisissez les informations de la session.
    • Nom de la session
    • Haut-parleurs - Cliquez sur Sélectionner des haut-parleurs et sélectionnez un ou plusieurs haut-parleurs pour la session.
    • Date de début
    • Heure de début
    • Durée
    • Description
    • Piste - Sélectionnez une piste à laquelle attribuer la session
    • Type de session
      • Session
      • Discours principal
      • Réunion
      • Autres
    • ID vidéo : disponible avec Session, Keynote et autres types de session
    • URL : disponible avec le type de session de réunion
    • Image de couverture
    • Contenu associé - Peut être utilisé pour fournir un lien vers le contenu de session (documents ou diapositives)
      • Label - Étiquette pour le contenu
      • URL - Lien vers le contenu
  4. Cliquez sur Save.

Confirmez que la session est ajoutée à l'ordre du jour. S'il y a plusieurs sessions dans la même piste programmées en même temps, elles seront mises en surbrillance et apparaîtront les unes à côté des autres.

Modification et suppression de sessions

Pour modifier ou supprimer une session, cliquez sur une session pour ouvrir le panneau Modifier la session. Pour modifier la session, apportez des modifications et cliquez sur sauvegarder. Pour supprimer la session, cliquez sur Supprimer la session.

Ajouter des conférenciers à une session

Pour ajouter des orateurs à une session, procédez comme suit.

  1. Cliquez sur une session pour ouvrir l'éditeur de session.
  2. Cliquez sur Sélectionnez les haut-parleurs.
  3. Sélectionnez un ou plusieurs haut-parleurs ou cliquez sur Créer un haut-parleur pour créer un nouveau haut-parleur.
    Note : Pour plus d'informations sur la création de haut-parleurs, voir Gestion des haut-parleurs d'événements.
  4. Cliquez sur Save.
Les Chercher peut être utilisé pour rechercher des locuteurs.

Ajout d'une nouvelle piste

Les pistes sont utilisées pour regrouper les sessions associées. Par exemple, la piste marketing comprendrait toutes les sessions liées au marketing. Un agenda aura généralement plusieurs pistes. Pour ajouter une nouvelle piste, procédez comme suit.

  1. Cliquez sur l'agenda pour ouvrir l'éditeur d'agenda.
  2. Clique le Ajouter une nouvelle piste lien en haut de l'agenda. Vous devrez peut-être faire défiler l'agenda vers la droite pour que le lien apparaisse.
  3. Entrer le Nom de la piste et appuyez sur Entrer.

Édition d'une piste

Pour modifier une piste, cliquez sur le nom de la piste, apportez des modifications et appuyez sur Entrer pour enregistrer les modifications.

Réorganiser les pistes

Pour modifier l'ordre des pistes dans l'agenda, procédez comme suit.

  1. Cliquez sur l'agenda pour ouvrir l'éditeur d'agenda.
  2. Survolez le nom de la piste à déplacer, cliquez sur l'icône de saisie et faites glisser la piste vers le nouvel emplacement.

Activer et désactiver un agenda

L'état de l'agenda détermine si l'agenda sera affiché ou non dans l'expérience d'événement virtuel associée. Le statut est affiché dans l'en-tête de la page sous le nom de l'agenda.

  • ACTIVÉ - L'agenda sera affiché dans l'expérience d'événements virtuels associée
  • DÉSACTIVÉ - L'agenda ne sera pas affiché dans l'expérience d'événements virtuels associée.

Pour activer un agenda, cliquez sur Activer en haut de la page. Pour désactiver un agenda, cliquez dessus et cliquez sur Désactiver. Notez qu'après avoir activé/désactivé un agenda, l'expérience n'a pas besoin d'être republiée.
