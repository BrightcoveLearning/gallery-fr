Premiers pas avec le module Agenda

{{ page.description }}

Pour accéder au module Agenda, cliquez sur une expérience d'événement virtuel pour ouvrir l'éditeur de site de la galerie, puis cliquez sur CARACTÉRISTIQUES DU SITE> Ordre du jour.

Sur la page d'accueil, vous pouvez :

  • Télécharger un agenda d'événement
  • Gérer l'agenda de l'événement
  • Gérer les conférenciers de l'événement
Pour revenir à l'éditeur de site Gallery depuis le module Agenda, cliquez sur le bouton Retour à mon événement lien sur le côté gauche de la page.

Téléchargement d'un agenda d'événement

Pour créer initialement un agenda d'événements, un fichier agenda .csv doit être téléchargé. Pour plus d'informations sur le téléchargement d'un agenda d'événement, voir Téléchargement d'un fichier d'agenda.

Gestion de l'agenda de l'événement

Pour modifier l'agenda de l'événement, cliquez dessus et apportez les modifications nécessaires. Pour plus d'informations, voir Gestion de l'agenda de l'événement.

Gestion des conférenciers de l'événement

Pour gérer les intervenants associés à un événement, cliquez sur Haut-parleurs dans la navigation de gauche. Pour plus d'informations sur la gestion des haut-parleurs, reportez-vous à Gestion des conférenciers d'événements.
