Création d'une expérience de page de destination

{{ page.description }}

Le modèle Page de destination vous permet de créer une expérience qui affiche une seule vidéo et peut éventuellement capturer des informations de visionneur. Par exemple, vous pouvez créer un message électronique pour un nouveau produit avec un lien vers un site de page de destination. Le site afficherait une vidéo produit et permettrait aux spectateurs d'entrer leurs informations pour recevoir plus d'informations.

Le processus de création et de publication d'une expérience Page de destination est similaire à ceux qui utilisent les autres modèles. Toutefois, lors de la création d'une expérience Page de destination, il existe des options de mise en page et des paramètres de site documentés ici dont vous pouvez profiter.

Création d'une expérience de page de destination

Pour créer une expérience de page de destination, procédez comme suit.

  1. Ouvrez le module Galerie et cliquez sur le lien Expériences en haut de la page.
  2. Cliquez sur Nouvelle expérience. Une liste de modèles s'affiche.
  3. Recherchez le modèle de page de destination et cliquez sur Choisir.

L'Éditeur du site Gallery s'ouvrira.

Modification de la mise en page de destination

La mise en page de la page de destination peut être configurée pour afficher éventuellement un formulaire d'appel à l'action ou un formulaire de prospect. Pour modifier la mise en page de la page de destination, procédez comme suit :

  1. Dans l'éditeur de site Gallery, cliquez sur PAGES dans la navigation de gauche.
  2. Clique le Changer la disposition menu déroulant.
  3. Cliquez sur un lien d'option de mise en page :
    • Sidebar CTA/AD and Lead Form - Affiche un formulaire de prospect et un appel à l'action dans la barre latérale droite à droite du lecteur
    • Sidebar CTA/AD - Affiche uniquement un appel à l'action dans la barre latérale droite à droite du lecteur
    • No Sidebar CTA/AD - Rien n'est affiché dans la barre latérale droite à droite du lecteur

L'aperçu dans l'Éditeur de site Gallery sera mis à jour pour afficher la mise en page. Une seule vidéo, des images, du texte, des liens et des appels à l'action peuvent être ajoutés au besoin. Pour plus d'informations sur l'utilisation de l'Éditeur de site Gallery pour personnaliser le site, reportez-vous à Utilisation de l'éditeur de site pour personnaliser une expérience de portail.

Utilisation du sélecteur vidéo unique

Lorsque vous utilisez le modèle de page de destination, seule une vidéo peut être affichée. Le modèle de page de destination utilise la boîte de dialogue de sélection vidéo unique par rapport à l'éditeur de collection utilisé par les autres modèles. Pour sélectionner une vidéo, procédez comme suit :

  1. Cliquez sur VIDÉOS dans la navigation de gauche.
  2. Sélectionnez une source vidéo, Video Cloud et YouTube sont pris en charge.
  3. Sélectionnez les critères vidéo :
    • Playlist - La vidéo se trouve dans une liste de lecture. Sélectionnez une liste de lecture, puis sélectionnez la vidéo.
    • Rechercher - Saisissez les critères de recherche de la vidéo. Les champs ID vidéo, nom et description courte et longue seront recherchés.
  4. Sélectionnez une seule vidéo.
  5. Cliquez sur Save pour enregistrer les modifications apportées.

Configuration d'un formulaire de prospecte

Pour utiliser un formulaire prospect, un lecteur compatible Audience configuré avec un formulaire de prospect doit être sélectionné.

Pour sélectionner un lecteur compatible Audience à utiliser sur la page de destination, cliquez sur > Lecteur VIDÉO ET LECTURE et Formulaire de prospect.

Les lecteurs compatibles avec le public sont indiqués par une icône de public (). Une fois qu'un lecteur compatible Audience est sélectionné, la boîte de dialogue se met à jour pour afficher le formulaire de prospect et les informations de connexion Audience.

Pour plus d'informations sur la création de formulaires de prospect et l'utilisation des lecteurs compatibles avec l'audience, consultez Utilisation d'un formulaire de prospect avec une expérience de portail.

Configuration d'un sous-domaine

Les domaines et sous-domaines personnalisés fournissent un moyen facile de publier le site sur une URL unique. Par exemple, vous pouvez créer plusieurs pages de destination et les faire publier sur mycustomurl.com/bike_campaign et mycustomurl.com/shoes_campaign. Pour plus d'informations sur la création de domaines personnalisés, voir Configuration de domaines personnalisés et SSL et Affectation d'un domaine personnalisé à une expérience de portail.

Exportation de données de piste

Toutes les données saisies dans les formulaires de prospect peuvent être exportées à l'aide du module Audience. Pour plus d'informations, voir Exportation des données de prospect de la galerie

Note : Seuls les utilisateurs disposant de privilèges Administrateur pourront exporter les données prospect.
