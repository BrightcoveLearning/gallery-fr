Création de l'état post-événement pour une expérience événementielle sur le portail

{{ page.description }}

Le modèle Live Event Portal fournit des modèles de page de pages détaillées avant, événement en direct, post-événement, archivage vidéo et vidéo. Ces modèles de page sont personnalisables pour que le site ait une apparence différente avant, pendant et après l'événement. L'état post-événement est ce que le site affichera une fois l'événement en direct terminé.

Pour créer l'état de la page d'accueil post-événement pour un site, créez une nouvelle expérience Live Event Portal Experience. Dans l'Éditeur de site Gallery, cliquez sur PAGES dans la navigation gauche, puis cliquez sur Page d'accueil : Post-Event.

L'état de la page d'accueil post-événement fournit les zones de contenu suivantes.

  • A - Logo et nom de la maison - Permet de renvoyer le visualiseur à la page d'accueil. Vous disposez de deux paramètres substituables qui vous permettent par exemple d'afficher une image suivie de texte.
  • B - Lien vers l'archive vidéo - Permet d'afficher toutes les collections de vidéos créées pour le site.
  • C - Paramètres Twitter - Permet d'utiliser un widget de flux Twitter. Voir Configuration d'un flux Twitter.
  • D - Image d'arrière-plan, texte et liens - Permet d'ajouter une image d'arrière-plan, du texte de sous-titre et des liens
  • E - Nom de l'événement en direct - Message texte statique
  • F - Message en direct terminé - Message texte statique
  • G - Paramètres d'inscription aux e-mails - Permet de configurer l'inscription aux e-mails. Consultez Configuration de l'inscription à la messagerie.
  • H - Ajouter des vidéos - Permet d'ajouter des collections de vidéos au site. Voir Ajout de vidéos pour plus d'informations.
  • I - Video Grid Box - Permet d'ajouter des appels à l'action, y compris du HTML ou des images personnalisées. Pour plus d'informations, reportez-vous à la section Configuration des appels à l'action sur une expérience de portail.
  • J - Objets de pied de page - Utilisé pour ajouter du texte et des liens au pied de page

Pour plus d'informations sur l'ajout d'images, de liens et de texte à un modèle de page Galerie, voir Personnalisation du contenu des pages Expérience du portail.

Note : Les couleurs du site varient en fonction du thème sélectionné.

Configuration d'un flux Twitter

Le site peut être configuré avec un Twitter intégré à la page. Pour connaître les types d'embeds disponibles, consultez le site Web Twitter. Un exemple aime chronologie est montré ci-dessous.

Pour ajouter un flux Twitter, procédez comme suit :

  1. Créez un widget sur le site Twitter. Si vous avez besoin d'aide pour créer un widget Twitter, consultez la documentation de développement de Twitter.
  2. Cliquez sur le lien Twitter Settings sur la page.
  3. Copiez le code HTML du widget Twitter et collez-le dans les paramètres du flux Twitter.
  4. Cliquez sur Save.

Configuration de l'inscription à la messagerie

Le site peut être configuré de manière à ce que les visiteurs du site puissent fournir leur adresse e-mail.

Pour ajouter une inscription à un e-mail, procédez comme suit :

  1. Cliquez sur le lien Email Sign-up Settings sur la page.
  2. Configurez les propriétés de l'inscription à la messagerie selon les besoins.
    • Ne pas afficher le formulaire de piste de courrier électronique - Ne pas afficher le formulaire de courrier électronique
    • Utiliser le formulaire de messagerie fourni avec le modèle - Sélectionnez cette option pour utiliser le formulaire de messagerie par défaut fourni avec le modèle
    • Utiliser du code HTML personnalisé fourni par votre logiciel d'automatisation marketing - Sélectionnez cette option pour utiliser votre formulaire HTML personnalisé pour la capture d'e-mails
  3. Cliquez sur Save.
  4. Une fois les paramètres de messagerie configurés, retournez à la page et cliquez sur le lien Message d'inscription par e-mail et définissez le texte qui apparaît au-dessus du champ d'inscription à l'e-mail.

Toutes les données capturées par le formulaire d'e-mail peuvent être exportées à l'aide de l'option Exporter des pistes de galerie dans le module Audience. Les pistes capturées à l'aide de l'option HTML personnalisée sont affichées dans votre logiciel d'automatisation marketing.

Ajout de vidéos

Cliquez sur le lien +Add Videos pour ajouter des collections de vidéos sur le site. Pour plus d'informations sur la création de collections, voir Ajout de vidéos à une expérience de portail.

La première collection de la liste des collections sera affichée sur la page d'accueil post-événement. Pour afficher toutes les collections, cliquez sur l'une des vidéos de la page d'accueil ou configurez l'objet Lien d'archivage vidéo afin que les spectateurs puissent cliquer dessus pour afficher toutes les collections.
