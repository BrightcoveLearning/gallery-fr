Personnalisation des paramètres du portail d'événements en direct

{{ page.description }}

Lorsqu'une expérience Live Event Portal est créée, plusieurs paramètres spécifiques au modèle peuvent être configurés. Pour configurer les paramètres du modèle Live Event Portal, modifiez l'expérience et cliquez sur Paramètres du > modèle APPARENCE ET COMPORTEMENT dans la navigation de gauche.

Le modèle Live Event Portal comporte différents paramètres qui peuvent être configurés pour chaque état de la page d'accueil.

Cliquez sur un onglet pour configurer les paramètres de chaque état. Les paramètres sont documentés sur la page appropriée pour l'état de la page d'accueil. Pour plus d'informations, reportez-vous à l'une des rubriques suivantes :

  • Création de l'état pré-événement pour une expérience d'événement portail
  • Création de l'état de l'événement en direct pour une expérience d'événement portail
  • Création de l'état post-événement pour une expérience événementielle sur le portail
