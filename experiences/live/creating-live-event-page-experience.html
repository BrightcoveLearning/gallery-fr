Création d'une expérience en direct dans la page

{{ page.description }}

Gallery fournit un modèle Live Event In-Page qui peut être utilisé pour diffuser des événements en direct dans le cadre d'une expérience en page. Brightcove propose de nombreuses options de diffusion de flux en direct. Consultez la section Livrer des flux en direct pour obtenir une liste de toutes les options. Pour obtenir un ensemble détaillé d'étapes sur l'utilisation du module Live pour diffuser un événement en direct, voir :

  • Étape par étape : Utilisation du module Live avec Telestream Wirecast
  • Étape par étape : Utilisation du module Live avec Open Broadcaster Software (OBS)
Note : Live Event In-Page Experiences est automatiquement lu dans les navigateurs de bureau même si le lecteur n'est pas configuré pour la lecture automatique. C'est par conception. Gallery forcera le joueur à jouer une fois le compte à rebours terminé afin que les utilisateurs assis sur la page en attente du début de l'événement puissent commencer à le regarder sans avoir à cliquer à nouveau. Les expériences In-Page Live Event ne seront pas jouées automatiquement sur les appareils iOS ou Android.

Pour créer une expérience en page d'événement en direct, créez une nouvelle expérience dans la page et sélectionnez le modèle Événement Live In-Page .

Le modèle Live Event In-Page comporte certains paramètres et éléments de contenu qui ne sont pas présents sur les autres modèles d'expérience en page.

  1. Afficher maintenant aux visionneuses
  2. État de l'événement en direct
  3. Partage social
  4. Espace réservé d'image
  5. Compte à rebours des événements
  6. Espace réservé de texte
  7. Rappel du calendrier
  8. Espace réservé Vidéos

Afficher maintenant aux visionneuses

Le paramètre Afficher maintenant aux visionneuses permet de configurer l'état d'événement en direct affiché aux visionneuses. Vous, l'éditeur, êtes responsable de modifier l'état de l'événement en direct. Gallery ne bascule jamais automatiquement l'état de l'événement en direct.

Cliquez sur Modifier et sélectionnez l'état à afficher. Après avoir modifié ce paramètre, l'expérience n'a PAS besoin d'être republiée.

Note : Il peut s'écouler quelques minutes avant que le changement d'état n'apparaisse sur le site de production.

États de l'événement en direct

Les états d'expérience d'une expérience dans la page d'un événement en direct sont différents des autres types d'expérience en page. L'apparence de chacun de ces états peut être personnalisée en cliquant sur les liens en haut de la page.

Événements en direct Les expériences en page ont les états suivants :

  • PRÉ-ÉVÉNEMENT - Comment l'expérience en direct apparaîtra avant le début de l'événement en direct
  • LIVE - Comment l'expérience apparaîtra alors que l'événement en direct est en cours
  • POST-ÉVÉNEMENT - Comment l'expérience apparaîtra une fois l'événement en direct terminé

Pour modifier les différents états d'expérience, cliquez sur un état, puis ajoutez des composants à l'expérience selon vos besoins. Le menu LAYOUT fournit d'autres options de personnalisation pour chaque état.

Partage social

Les options de partage social fournissent des liens pour partager l'événement et les vidéos vers vos comptes de médias sociaux.

Pour configurer les paramètres de partage de réseaux sociaux, cliquez sur le lien Configurer le partage ou cliquez sur PARAMÈTRES > Social dans la navigation de gauche. Les paramètres sociaux sont les suivants :

  • Partage d'événements
  • Partage vidéo

Partage d'événements

Les paramètres de partage d'événements permettent au spectateur de partager la page de l'événement sur les comptes de médias sociaux. Procédez comme suit pour configurer les détails de l'événement :

  1. Pour ajouter une image d'événement, cliquez sur Ajouter une image, puis :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  2. Entrez un titre et une description de l'événement.
  3. Choisissez la façon dont les options de partage d'événements sont affichées, Partager du texte ou afficher les icônes sociales.
  4. Sélectionnez les options de partage disponibles.
  5. Entrez l' URL de la page où l'expérience sera incorporée.
  6. Cliquez sur Générer. Cela générera les balises meta qui peuvent être ajoutées à la <head> section de votre page. Grâce à cette fonctionnalité, les réseaux sociaux seront en mesure d’afficher l’image, le titre et la description de l’événement lorsque des visiteurs partagent votre expérience.

Partage vidéo

Les paramètres de partage vidéo permettent au spectateur de partager les vidéos sur les comptes de médias sociaux. Procédez comme suit pour configurer le partage vidéo :

  1. Choisissez la façon dont les options de partage vidéo sont affichées, Partager du texte ou afficher les icônes sociales.
  2. Sélectionnez les options de partage disponibles.
  3. Cliquez sur Save pour enregistrer les modifications apportées.

Espace réservé d'image

Les états PRE-EVENT et POST-EVENT permettent d'afficher une image de héros.

Pour ajouter une image de héros :

  1. Clique le Ajouter une image relier.
  2. Entrez un nom pour l'image (seuls les caractères ASCII sont autorisés).
  3. Pour spécifier une image, vous pouvez :
    • Faites glisser une image depuis le système de fichiers, puis déposez-la sur la zone de dépôt
    • Cliquez sur Parcourir et sélectionnez une image dans le système de fichiers
    • Entrez l'URL d'un fichier image distant
  4. Définissez la taille de l'image.
    • Rogner l'image : recadrez l'image au centre de manière à ce qu'elle remplisse l'espace 16:9, mais une partie du haut, du bas ou de la gauche/droite sera perdue en fonction du rapport d'aspect de l'image.
    • Contenir l'image dans les limites des composants - Contient l'image dans les limites de l'espace 16:9, mais un espace vide au-dessus ou inférieur ou à gauche/droite de l'image apparaîtra en fonction du rapport d'aspect
    • Étendre l'image : ne tient pas compte du rapport hauteur/largeur de l'image et l'oblige à remplir complètement l'espace 16:9, ce qui déforme l'image
    • Taille de l'image d'origine - Utilisez la taille de l'image d'origine, aucun redimensionnement n'est effectué ; ne répond pas
  5. Sélectionnez les états où l'image doit être affichée.
  6. Cliquez sur Enregistrer pour ajouter l'image à l'expérience.

Compte à rebours des événements

Le compte à rebours affiche le nombre de mois, de jours, d'heures, de minutes et de secondes avant le début de l'événement en direct. Un exemple de minuterie de compte à rebours est illustré ci-dessous.

Note : Le style du compte à rebours peut être conçu en cliquant sur STYLE dans la barre de navigation de gauche.

Pour ajouter un compte à rebours, procédez comme suit :

  1. Cliquez sur le lien Configurer le compte à rebours des événements sur la page. La page Paramètres de l'événement s'ouvre.
  2. Cochez Afficher le compte à rebours des événements.
  3. Définissez la date de début, l' heure de début et le fuseau horaire de l'événement.
  4. Saisissez un message de transition. Ce message s'affiche lorsque le compte à rebours atteint zéro mais avant le début du flux en direct.
  5. Cliquez sur Save.

Espace réservé de texte

Les états PRE-EVENT et POST-EVENT permettent d'afficher du texte statique.

Pour ajouter du texte statique :

  1. Clique le Ajouter du texte relier.
  2. Entrez le texte à ajouter à l'expérience. La barre d'outils peut être utilisée pour formater le texte.
  3. Cliquez sur Enregistrer pour ajouter le texte à l'expérience.

Rappel du calendrier

Un rappel de calendrier peut être configuré afin que les utilisateurs puissent cliquer sur un lien pour télécharger un rappel de calendrier. Un exemple de rappel de calendrier est illustré ci-dessous.

Cliquez sur le rappel pour afficher les options de calendrier.

Note : Le style du rappel du calendrier peut être conçu en cliquant sur STYLE dans la barre de navigation de gauche.

Pour ajouter un rappel de calendrier, procédez comme suit :

  1. Cliquez sur le lien Configurer le rappel du calendrier sur la page. La page Paramètres de l'événement s'ouvre.
  2. Cochez Afficher le rappel « Ajouter au calendrier ».
  3. Définissez la date de fin, l' heure de fin et le titre de l'événement. La description de l'événement et l' emplacement de l'événement sont facultatifs.
    Note : L'interface apparaîtra légèrement différente si l'option Afficher le compte à rebours des événements est sélectionnée.
  4. Cliquez sur Save.

Espace réservé Vidéos

Différentes vidéos peuvent être affichées pendant les états PRE-EVENT, LIVE et POST-EVENT. Les vidéos Video Cloud et YouTube peuvent être ajoutées à l'expérience.

Note : Pour que des vidéos YouTube puissent être ajoutées à une expérience, vous devez configurer les paramètres YouTube.

Pour sélectionner les vidéos à afficher :

  1. Cliquez sur le lien Sélectionner une vidéo ou cliquez sur VIDÉOS dans la navigation de gauche. La boîte de dialogue Sélectionner une vidéo s'affiche.
  2. Sélectionnez l'état pour lequel sélectionner la vidéo :
    • Avant l’événement
    • En direct
    • Après l’événement
  3. Sélectionnez la sourcevidéo :
    • Video Cloud - Le contenu vidéo Video Cloud sera utilisé
    • YouTube - Le contenu vidéo YouTube sera utilisé
  4. Recherchez une vidéo, cliquez dessus pour la sélectionner, puis cliquez sur Sélectionner. La vidéo sera ajoutée à la liste des vidéos.
  5. Clique sur le Habitent ou Après l'occassion rubriques ou cliquez sur Sélectionnez des vidéos pour sélectionner des vidéos supplémentaires.

Les interactions et les composants associés peuvent être ajoutés aux vidéos pré-événement et post-événement et affichés au fur et à mesure de la lecture de la vidéo. Les interactions incluent des liens et des cartes et sont affichées dans le lecteur lors de la lecture d'une vidéo. Les composants complémentaires incluent du HTML, des images et des objets texte et sont affichés en dehors du lecteur.

Note : Les interactions et les composants associés ne sont pas disponibles pour la vidéo en direct, sauf s'il s'agit d'une ressource VOD.

Les vidéos qui ont été configurées avec l'interactivité affichent une icône dans la colonne Interactivité indiquant le type d'interaction. Pour plus d'informations sur l'ajout et la configuration d'interactions et de composants complémentaires, voir :

  • Ajout d'interactions à une expérience dans la page
  • Configuration des composants compagnons pour une expérience dans la page

Sélection de la vidéo Live

Lorsque l'état Live est sélectionné, un ID vidéo Video Cloud ou un ID de référence pour la ressource vidéo en direct doit être saisi. La source de la vidéo en direct sera toujours Video Cloud.

La sélection de l'option ID de référence vous permet de modifier l'ID de référence de la vidéo en direct dans le module Media sans avoir à modifier l'ID vidéo sur l'expérience. Par exemple, vous pouvez choisir de désigner la vidéo en direct actuelle avec un ID de référence live-event. Lors de la création de l'expérience en direct, sélectionnez ID de référence et entrez live-event.

Maintenant, chaque fois qu'il y a un nouvel événement en direct, utilisez le module Media pour définir l'ID de référence de cette vidéo sur live-event.

En utilisant l'ID de référence, l'expérience en direct n'aura pas besoin d'être modifiée et republiée au démarrage d'un nouvel événement. Lorsqu'un nouvel événement est démarré, utilisez le module Media pour modifier l'ID de référence vidéo de la ressource vidéo en direct.

Recherche de vidéos

Le champ de recherche peut être utilisé pour rechercher des vidéos. Lors de la recherche de vidéos Video Cloud, Gallery recherche le nom de la vidéo, les descriptions courtes et longues, les balises et les champs personnalisés. Lors de la recherche de vidéos YouTube, Gallery utilise l'API de recherche YouTube.

Cliquez sur Autres options de recherche pour mieux contrôler les champs recherchés et pour inclure et exclure du texte. Notez que ces options sont uniquement disponibles lorsqu'une source Video Cloud est utilisée.
