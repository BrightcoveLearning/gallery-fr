Création de l'état pré-événement pour une expérience d'événement portail

{{ page.description }}

Le modèle Live Event Portal fournit des modèles de page de détail avant, en direct, post-événement, d'archivage vidéo et de page de détails vidéo. Les modèles de page pré-événement, live et port-événement peuvent être personnalisés afin que le site ait une apparence différente avant, pendant et après un événement en direct. L'état avant l'événement est ce que le site affichera avant le début de l'événement en direct.

Pour créer l'état de la page d'accueil pré-événement pour un site, créez une nouvelle expérience de portail à l'aide du modèle Portail d'événements en direct. Dans l'éditeur de site Gallery, cliquez sur PAGES dans la navigation de gauche. Vérifiez que l'état Pre-Event est bien sélectionné.

L'état de la page d'accueil pré-événement fournit les zones de contenu suivantes.

  • A - Bannière : permet d'ajouter des appels à l'action, y compris du HTML personnalisé, des images et des publicités. Pour plus d'informations, reportez-vous à la section Configuration des appels à l'action sur une expérience de portail.
  • B - Logo et nom de la maison - Permet de renvoyer le visualiseur à la page d'accueil. Vous disposez de deux paramètres substituables qui vous permettent par exemple d'afficher une image suivie de texte. Les images seront réduites à une hauteur de 55px et une largeur maximale de 495px.
  • C - Lien vers l'archive vidéo - Permet d'afficher toutes les collections de vidéos créées pour le site. Notez que la première collection de la liste sera considérée comme une « collection vedette » et sera affichée dans le corps de la page de pré-événement (zone H ci-dessous).
  • D - Paramètres Twitter - Permet d'utiliser un widget de flux Twitter. Voir Configuration d'un flux Twitter.
  • E - Image d'arrière-plan, texte et liens - Permet d'ajouter une image d'arrière-plan, un texte de sous-titre et des liens
  • F - Paramètres du compte à rebours - Fournit un compte à rebours jusqu'au début de l'événement. Voir Configuration d'un compte à rebours.
  • G - Message pré-événement - Message texte statique
  • H - Ajouter des vidéos - Permet d'ajouter des collections de vidéos au site. Voir Ajout de vidéos pour plus d'informations.
  • I - Video Grid Box - Permet d'ajouter des appels à l'action, y compris du HTML ou des images personnalisées. Pour plus d'informations, reportez-vous à la section Configuration des appels à l'action sur une expérience de portail.
  • J - Objets de pied de page - Permet d'ajouter du texte et des liens au pied de page

Pour plus d'informations sur l'ajout d'images, de liens et de texte statique aux modèles de page de portail, consultez Personnalisation du contenu des pages Expérience du portail.

Note : Les couleurs du site varient en fonction du thème sélectionné.

Configuration d'un flux Twitter

Le site peut être configuré avec un Twitter intégré à la page. Pour connaître les types d'embeds disponibles, consultez le site Web Twitter. Un exemple aime chronologie est montré ci-dessous.

Pour ajouter un flux Twitter, procédez comme suit :

  1. Créez un widget sur le site Twitter. Si vous avez besoin d'aide pour créer un widget Twitter, consultez la documentation de développement de Twitter.
  2. Cliquez sur le lien Twitter Settings sur la page.
  3. Copiez le code HTML du widget Twitter et collez-le dans le Fil Twitter Les paramètres.
  4. Cliquez sur Save.

Configuration d'un compte à rebours

Le site peut accueillir un compte à rebours qui affiche le nombre de mois, jours, heures et minutes avant le début de l'événement en direct. Un exemple de minuterie de compte à rebours est illustré ci-dessous.

Lorsque le compte à rebours atteint zéro, le texte Coming Soon s'affiche à sa place. Pour ajouter un compte à rebours, procédez ainsi :

  1. Cliquez sur le lien Countdown Settings sur la page.
  2. Localisez la section Event Countdown et saisissez la date et l'heure de l'événement en direct.
  3. Cliquez sur Save.
  4. Cliquez sur le texte au-dessus de la minuterie de compte à rebours pour le personnaliser.
Note : Vous, l'éditeur, êtes responsable de modifier l' état de la page d'accueil. Gallery ne changera jamais automatiquement l'état de la page d'accueil.

Configuration d'un rappel de calendrier

Le site peut être configuré de façon à ce que les utilisateurs puissent télécharger un rappel de calendrier. Un exemple de rappel de calendrier est illustré ci-dessous.

Cliquez sur le lien du calendrier pour télécharger un fichier .ics qui est un format global pour les fichiers de calendrier utilisés par des programmes tels que Google Agenda, Apple Calendar et Microsoft Outlook. Notez que le rappel de calendrier ne sera pas affiché à moins que les paramètres du calendrier ne soient configurés.

Pour configurer les paramètres du fichier .ics de rappel de calendrier, procédez comme suit :

  1. Clique sur le Paramètres du compte à rebours relier.
  2. Configurez les détails du fichier .ics du rappel de calendrier.
  3. Cliquez sur Save.

Ajout de vidéos

Cliquez sur le lien +Add Videos pour ajouter des collections de vidéos sur le site. Pour plus d'informations sur la création de collections, voir Ajout de vidéos à une expérience de portail.

La première collection de la liste des collections sera affichée sur la page d'accueil pré-événement. Pour afficher toutes les collections, cliquez sur l'une des vidéos de la page d'accueil ou configurez l'objet Lien d'archivage vidéo afin que les spectateurs puissent cliquer dessus pour afficher toutes les collections.
