Ajout de questions-réponses Pigeonhole à une expérience sur la page d'un événement en direct

{{ page.description }}

Pigeonhole peut être utilisé pour ajouter facilement une interactivité basée sur des événements (questions et réponses en direct, sondages, enquêtes, etc.) à une expérience sur la page d'événements en direct. Généralement, un nouveau Pigeonhole est créé pour l'événement, puis des sessions sont créées pour les événements en direct. Une fois la session créée, le code d'intégration Pigeonhole est collé dans le modèle Live Event In-Page. Le code d'intégration entraînera l'affichage du pod Q&A Pigeonhole à côté du joueur de l'événement en direct.

Création d'un événement et d'une session Pigeonhole

Vous devez créer un nouveau Pigeonhole pour votre événement en direct, puis ajouter des sessions pour chaque vidéo qui aura Q&R activé.

  1. Connectez-vous à votre compte Pigeonhole.
  2. Cliquez sur + Ajouter Pigeonhole et cliquez sur le compte à utiliser.
  3. Entrez un nom d'événement et un code d'accès.
  4. Saisissez les dates de début et de fin de l'événement. Ces dates contrôlent combien de temps le Pigeonhole sera actif.
  5. Sélectionnez une option de fuseau horaire et de marque personnalisée (le cas échéant).
  6. Cliquez sur Continue.
  7. Maintenant que l'événement est créé, le point Agenda sera sélectionné dans la navigation de gauche. Cliquez sur Ajouter > une session Créer un nouveau.
  8. Sélectionnez un type de session de questions-réponses.
  9. Entrez un nom de session.
  10. Entrer Début et Finir dates et heures.
  11. Configurez les paramètres Q&R selon les besoins.
  12. Cliquez sur Ajouter des questions et réponses. Confirmez que la session a été créée.

Génération du code d'intégration Pigeonhole

Pour que le Pigeonhole s'affiche dans l'expérience, le code suivant sera ajouté à l'aide d'un composant personnalisé sur l'expérience. Le format du code sera :

Pour obtenir les valeurs du code secret de l' événement et de l'ID de session, procédez comme suit.

  1. Dans Pigeonhole, cliquez sur Agenda dans la navigation de gauche.
  2. Sur le Séances onglet, cliquez sur la flèche déroulante à côté de l'événement. L' ID de session s'affiche.
  3. Pour obtenir le code secret de l'événement, recherchez le nom de l'événement dans l'en-tête. Le code d'accès de l'événement s'affiche sous le nom de l'événement.

Le code ci-dessus avec votre code d'accès d'événement et ID de session sera utilisé dans votre expérience.

Ajout d'un pigeonhole à une expérience

Brightcove recommande d'ajouter le Pigeonhole au côté droit de votre expérience Live Event In-Page. Actuellement, le plugin Pigeonhole cible spécifiquement le côté droit.

Note : Si vous devez ajouter le Pigeonhole à gauche de l'expérience, contactez votre responsable de compte.

L'ajout d'un pigeonhole à une expérience Live Event In-Page se fait à l'aide d'un composant personnalisé. Pour ajouter un pigeonhole, procédez comme suit.

  1. Ouvrez le module Gallery.
  2. Cliquez sur le Live Event In-Page Experience pour modifier.
  3. Cliquez sur l'icône plus () sur le côté droit de l'expérience.
  4. Cliquez sur Personnaliser dans le sélecteur de composants.
  5. Entrez un nom pour le composant HTML.
  6. Collez le code HTML Pigeonhole de la section précédente dans le Code HTML personnalisé champ.
  7. Sélectionnez le moment où le code HTML doit être affiché.
  8. Cliquez sur Save. Le code HTML sera rendu sur l'expérience.
  9. Dans la navigation de gauche, cliquez sur PARAMÈTRES > avancés.
  10. Collez le code suivant dans le champ URL JavaScript. 
    https://static.pigeonhole.at/brightcove-plugin/phl-brightcove-gallery-inpage.js
  11. Cliquez sur Save.
  12. Publiez les modifications apportées à l'expérience.

Lorsque vous prévisualisez l'expérience, le pigeonhole doit s'afficher.
