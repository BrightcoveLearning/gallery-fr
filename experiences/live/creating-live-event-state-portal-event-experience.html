Création de l'état de l'événement en direct pour une expérience d'événement portail

{{ page.description }}

Le modèle Live Event Portal fournit des modèles de page de pages détaillées avant, événement en direct, post-événement, archivage vidéo et vidéo. Ces modèles de page sont personnalisables pour que le site ait une apparence différente avant, pendant et après l'événement. L'état de l'événement en direct est ce que le site affichera pendant l'événement en direct.

Pour créer l'état de la page d'accueil d'un événement en direct pour un site, créez une nouvelle expérience de portail à l'aide du modèle Portail d'événements en direct. Dans l'éditeur de site Gallery, cliquez sur PAGES dans la navigation gauche, puis cliquez sur Page d'accueil : Live.

L'état de la page d'accueil de l'événement en direct fournit les zones de contenu suivantes.

  • A - Logo et nom de la maison - Permet de renvoyer le visualiseur à la page d'accueil. Vous disposez de deux paramètres substituables qui vous permettent par exemple d'afficher une image suivie de texte.
  • B - Lien vers l'archive vidéo - Permet d'afficher toutes les collections de vidéos créées pour le site.
  • C - Paramètres Twitter - Permet d'utiliser un widget de flux Twitter. Voir Configuration d'un flux Twitter.
  • D - Image d'arrière-plan - Permet d'ajouter une image d'arrière-plan
  • E - Paramètres vidéo en direct - Permet de configurer l'ID vidéo en direct. Consultez Configuration de la vidéo en direct.
  • F - Nom de l'événement en direct - Message texte statique
  • G - Indicateur « Happening Now » - Cacher/afficher l'indicateur « Happening now »
  • H - Paramètres de chat - Zone facultative pour intégrer un widget de chat tiers (par exemple Chatroll). Consultez Configuration d'un widget de discussion.
  • I - Video Grid Box - Permet d'ajouter des appels à l'action, y compris du HTML ou des images personnalisées. Pour plus d'informations, reportez-vous à la section Configuration des appels à l'action sur une expérience de portail.
  • J - Objets de pied de page - Utilisé pour ajouter du texte et des liens au pied de page

Pour plus d'informations sur l'ajout d'images, de liens et de texte à un modèle de page de portail, consultez Personnalisation du contenu des pages Expérience du portail.

Note : Les couleurs du site varient en fonction du thème sélectionné.

Configuration d'un flux Twitter

Le site peut être configuré avec un Twitter intégré à la page. Pour connaître les types d'embeds disponibles, consultez le site Web Twitter. Un exemple aime chronologie est montré ci-dessous.

Pour ajouter un flux Twitter, procédez comme suit :

  1. Créez un widget sur le site Twitter. Si vous avez besoin d'aide pour créer un widget Twitter, consultez la documentation de développement de Twitter.
  2. Cliquez sur le lien Twitter Settings sur la page.
  3. Copiez le code HTML du widget Twitter et collez-le dans le Fil Twitter Les paramètres.
  4. Cliquez sur Save.

Configuration de la vidéo en direct

Cliquez sur le lien Configurer Live Video pour configurer les paramètres vidéo en direct avec l' ID vidéo ou l' ID de référence de la ressource vidéo en direct. Lors de l'utilisation de l'ID vidéo, l'expérience devra être mise à jour avec le nouvel ID vidéo chaque fois qu'un nouvel événement en direct est créé. L'utilisation d'un ID de référence offre plus de flexibilité. Lorsqu'un nouvel événement en direct est créé, il suffit d'attribuer l'ID de référence désigné à la nouvelle vidéo dans le module Media et aucune modification n'est requise pour l'expérience.

Localisation de l'ID vidéo

L'ID vidéo peut être localisé en utilisant l'une des méthodes suivantes :

  1. L'ID vidéo apparaît sous le titre de la vidéo dans le module Media.
  2. L'ID vidéo apparaît sous le nom de l'événement dans le module Live.

Localisation de l'ID de référence

L'ID de référence peut être localisé à l'aide du module Media.

Configuration d'un widget de chat

Le site peut être configuré pour afficher des widgets de discussion tiers sur une page. Par exemple, à l'aide de Chatroll, vous pouvez incorporer du code HTML dans une page pour ajouter des fonctionnalités de chat en ligne à un site Galerie. Un exemple de chat est montré ci-dessous.

Pour ajouter un widget de chat tiers, procédez comme suit :

  1. Cliquez sur le lien Chat Settings sur la page.
  2. Copiez le code HTML du widget de chat et collez-le dans les paramètres d'intégration du chat.
  3. Cliquez sur Save.
