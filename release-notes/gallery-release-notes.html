    Mises à jour du produit Brightcove Gallery™

    Cette rubrique décrit les mises à jour apportées à l'application Brightcove Gallery.

    Des notes de mise à jour sont également disponibles pour :

    • Notes de mise à jour du modèle Galerie

    Paramètre de lecture automatique de la vidéo

    4 sept. 2019

    Un nouveau paramètre de lecture automatique de la vidéo a été ajouté et peut être utilisé pour contrôler le comportement de lecture des vidéos sur les pages détaillées. Pour plus d'informations, consultez Personnalisation des paramètres de formulaire Player et Lead for a Portal Experience.

    Navigation dans la galerie repensée

    27 sept. 2018

    Le menu de navigation lors de la modification des Expériences de portail a été repensé pour être plus intuitif et cohérent avec l'environnement d'édition In-Page Experience. Pour plus d'informations, consultez Utilisation de l'éditeur de site pour personnaliser une expérience de portail.

    Option de recherche ajoutée à la page d'accueil de la galerie

    1 août 2018

    Une zone de recherche a été ajoutée à la page d'accueil de la Galerie permettant de rechercher dans la liste des expériences. Lors de la recherche, les propriétés suivantes seront recherchées :

    • Titre
    • Description
    • ID
    • Modèle
    • URL

    Ajout de vidéos à une expérience en page

    18 juil. 2018

    Lorsque vous ajoutez des vidéos à une expérience en page, il existe désormais une option permettant d'utiliser un ID de référence de liste de lecture. En utilisant un ID de référence de playlist, les vidéos de l'expérience peuvent facilement être modifiées en réaffectant l'ID de référence à une autre playlist à l'aide du module Media. L'expérience n'a pas besoin d'être modifiée et republiée lorsque vous devez modifier les vidéos qui apparaissent dans l'expérience. Pour plus d'informations, voir Utilisation d'un ID de référence de liste de lecture.

    Modifications apportées aux formulaires de prospecte

    15 mai 2018

    Gallery utilise désormais le module Audience pour créer et gérer des formulaires de prospect. Le public recueillera les données de prospect et fournit une option d'exportation de données. Pour plus d'informations, consultez Utilisation d'un formulaire de prospect avec une expérience de portail.

    Modèle d'événement en direct dans la page

    14 mai 2018

    Gallery fournit désormais un modèle Live Event In-Page qui peut être utilisé pour diffuser des événements en direct dans le cadre d'une expérience en page. Pour plus d'informations, voir Création d'une expérience en page d'événement en direct.

    Mise à jour de l'interface utilisateur des paramètres du site

    13 avr. 2018

    L'interface utilisateur Paramètres du site a été mise à jour pour supprimer les paramètres vidéo de la boîte de dialogue Personnalisée vers une nouvelle boîte de dialogue Paramètres vidéo . Pour plus d'informations, voir Personnalisation des paramètres vidéo pour une expérience de portail.

    Paramètre de modèle de sélection

    30 mars 2018

    Un nouveau paramètre de comportement vidéo a été ajouté au modèle Marquee. Pour plus d'informations, voir Personnalisation des paramètres de modèle de sélection.

    Option d'en-tête et pied de page personnalisés

    30 mars 2018

    Il est désormais possible d'inclure l'en-tête et le pied de page par défaut lors de l'utilisation d'un en-tête et d'un pied de page personnalisés. Pour plus d'informations, voir Configuration du code HTML d'en-tête/pied de page personnalisé.

    Composants 3Play Experience In-Page

    1 févr. 2018

    Les composants de transcription interactifs 3Play Media peuvent désormais être ajoutés à une expérience en page. Cela permet aux spectateurs de voir une transcription vidéo au fur et à mesure qu'une vidéo est lue. Pour plus d'informations, voir Ajout de composants à une expérience en page.

    Composants de chat d'expérience dans la page

    17 janv. 2018

    Les composants de chat peuvent désormais être ajoutés à une expérience en page. Pour plus d'informations, voir Ajout de composants à une expérience en page.

    Page Contrôle d'accès personnalisé

    28 nov. 2017

    La page de contrôle d'accès peut désormais être stylisée de sorte qu'elle soit plus étroitement marquée avec le reste du site. Pour plus d'informations, voir Styliser la page de contrôle d'accès.

    404 Pages Stylisées

    6 nov. 2017

    404 pages utiliseront désormais le style configuré avec le site (en-têtes et pieds de page) afin qu'ils soient plus étroitement marqués avec le reste du site.

    Modèle d'événement en direct - Options d'image d'arrière-plan

    6 nov. 2017

    Le modèle Événement en direct fournit désormais des options de traitement des images d'arrière-plan. Pour plus d'informations, voir Personnalisation des paramètres du portail d'événements en direct.

    Métadonnées collection

    6 nov. 2017

    Pour tous les modèles (sauf Marquee), vous pouvez désormais ajouter des descriptions de collection et cela servira de description des métadonnées pour la page de catégorie (pour les navigateurs partageant la collection). Pour plus d'informations, voir Ajout de vidéos Video Cloud à une expérience de portail.

    Clé API n'est plus requise

    29 sept. 2017

    La clé API n'est plus requise et a été supprimée de la page Paramètres.

    Plusieurs expériences dans la page sur la même page

    15 août 2017

    Gallery prend désormais en charge plusieurs instances de la même expérience dans la page sur la même page. Cela vous permet de créer une expérience, puis de passer dynamiquement dans une playlist ou une liste de vidéos à afficher. Pour plus d'informations, voir Utilisation des remplacements vidéo dynamiques dans une expérience en page.

    L' expérience d'événement prend désormais en charge l'utilisation de l'ID

    9 août 2017

    Lors de la configuration de l'état de l'événement en direct pour une expérience d'événement, la ressource vidéo en direct peut désormais être spécifiée à l'aide d'un ID de référence. L'utilisation d'un ID de référence offre plus de flexibilité car l'ID de référence peut être attribué à la ressource vidéo en direct sans avoir à modifier l'expérience. Pour plus d'informations, voir Création de l'état d'événement en direct pour une expérience événementielle.

    Paramètre de lecteur d'expérience supplémentaire dans la page

    27 juil. 2017

    Un nouveau paramètre de lecteur a été ajouté pour permettre à une expérience en page de rester dans l'état de lecture lorsqu'une vidéo est terminée. Cela peut être utilisé pour afficher l'écran de fin du lecteur. Pour plus d'informations, voir Configuration des paramètres d'une expérience dans la page.

    Remplacer la vidéo dynamique dans une expérience en page

    27 juil. 2017

    Il est désormais possible de remplacer dynamiquement les vidéos associées à une expérience en page en modifiant le code d'intégration. Pour plus d'informations, voir Utilisation des remplacements vidéo dynamiques dans une expérience en page.

    Formation sur l'expérience en page maintenant disponible

    21 juil. 2017

    Une formation à la demande pour la création d'expériences en page est désormais disponible. Pour plus d'informations, voir Formation à la demande : Création d'expériences dans la page à l'aide de Gallery.

    Support vidéo YouTube pour les expériences en page

    11 juil. 2017

    Il est désormais possible d'utiliser des vidéos YouTube dans une expérience en page. Pour plus d'informations, voir Ajout de vidéos à une expérience en page.

    Utiliser la description du site dans les résultats du moteur de recherche

    10 juil. 2017

    Gallery propose désormais une option permettant d'utiliser la description du site dans les résultats des moteurs de recherche. Cela utilisera la description du site dans les balises de description, y compris les balises méta Facebook et Twitter. Pour plus d'informations, voir Configuration des paramètres d'optimisation du moteur de recherche pour une expérience de portail.

    Modèle de catalogue

    28 juin 2017

    Gallery fournit désormais un modèle de catalogue. Pour plus d'informations, consultez Généralités sur les modèles de portail Gallery.

    Expériences dans la page

    31 mai 2017

    Gallery offre désormais la possibilité de créer des expériences dans la page. Pour plus d’informations, consultez Présentation : Création, modification et publication d'expériences dans la page.
